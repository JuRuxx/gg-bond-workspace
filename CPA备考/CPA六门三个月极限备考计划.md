## 三、各科详细策略

### 📗 会计 — 最重，先打

```
目标：理解会计准则逻辑，不是死记分录
方法：张敬富/张志凤的基础课（1.5倍速）
重点：长期股权投资、合并报表、收入、金融工具、所得税
策略：
  - 6月：跟着基础课听完，每章做课后题
  - 7月：做近5年真题的主观题部分
  - 8月：综合题专项突破
放弃：不纠结冷门准则（政府补助、租赁特殊条款）
```

### 📘 审计 — 绕，但套路固定

```
目标：建立审计逻辑框架
方法：金鑫松/刘圣妮，多读多做题找语感
重点：风险评估与应对、审计程序、审计报告、集团审计
策略：
  - 6月：过一遍框架，理解"审计是干啥的"
  - 7月：做简答题，背高频考点
  - 8月：综合题+选择题保持手热
技巧：多选题选不出来时，"宁少勿多"
```

### 📙 财管 — 计算量最大

```
目标：计算速度 > 准确率
方法：郑晓博/闫华红，公式必须背到条件反射
重点：财务报表分析、资本成本、项目投资决策、期权定价
策略：
  - 6月：公式推导一遍，理解内在逻辑
  - 7月：每天做计算题，计时，练速度
  - 8月：模拟卷掐表做，练到"看到题就条件反射"
救命：财管做不完是正常的，选你会做的先做！
```

### 📕 税法 — 碎，但选择题占比高

```
目标：选择题拿高分
方法：叶青/刘颖
重点：增值税、企业所得税、个人所得税、消费税
策略：
  - 6月：各税种过一遍，理解征税逻辑
  - 7月：疯狂刷选择题
  - 8月：综合题练一练，背税收优惠
技巧：税法选择题占50分，选择题拿35+就过了一半
```

### 📒 经济法 — 纯背诵

```
目标：背高频法条+理解案例
方法：郭守杰/黄洁洵
重点：公司法、证券法、破产法、票据法、合同法
策略：
  - 6月：快速过一遍，理解法条逻辑
  - 7月：背法条关键词，做法条分析题
  - 8月：突击背诵（考前两周是黄金期）
技巧：法条不用逐字背，关键词+逻辑说清楚就行
```

### 📓 战略 — 投入产出比最高

```
目标：最简单的，别丢分
方法：吕鹏/杭建平
重点：战略分析（PEST/五力/价值链）、战略选择、风险管理
策略：
  - 6月：过一遍，做思维导图
  - 7月：刷选择题+背模型框架
  - 8月：综合题练模板（SWOT+战略选择+实施）
技巧：综合题用"模型+案例"结构，拿分稳
```

## 六、必须遵守的铁律

### ✅ 要做
1. **会计和财管每天都要碰** — 这两门断了三天，捡起来要花一周
2. **错题本必须有** — 不用美观，但要能快速回顾
3. **周末必须做一套完整模拟** — 掐表，模拟考场节奏
4. **考前两周停所有新课** — 只复习、刷题、背法条

### ❌ 不要做
1. **不要追求完美笔记** — 时间不是花在好看上的
2. **不要钻牛角尖** — 某道题卡了15分钟还想不通？跳过
3. **不要跟别人比进度** — 你只有450小时，按照自己的节奏
4. **不要熬夜超过12点半** — 第二天工作会崩溃，得不偿失
