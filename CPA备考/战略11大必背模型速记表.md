# 第1类 · 分析类模型

---

## ① PEST分析 — 宏观环境四扇窗

**英文速记：PEST = Politics + Economy + Society + Technology**

### 🔟 秒记住
``` 
政经社技，四大环境，宏观分析，它排第一
```

| 字母 | 维度 | 考点关键词 | 例子 |
|:----:|:----:|-----------|------|
| **P** | 政治与法律 | 政策法规、税收、环保、劳动法、政府补贴 | 制造业环保新规→设备升级 |
| **E** | 经济 | GDP增速、利率、汇率、通胀、失业率 | 喇叭行业竞争加剧→利润压缩 |
| **S** | 社会文化 | 人口结构、消费习惯、价值观、教育水平 | 员工对公平感敏感→管理要调整 |
| **T** | 技术 | 新技术、自动化、研发投入、专利 | 设备越来越智能→维护更复杂 |

### 💀 出题陷阱
- 问"经济因素" → 别把**政治**选进去！P和E最容易混
- 技术因素只写"新技术改变行业"，**别写政策鼓励**

## ② 波特五力模型 — 行业竞争五个敌人

**一句话：供应商、买家、新来的、替代品、老同行**

### 🔟 秒记住
```
供买新替同，五力来围攻
供方会抬价，买方拼命压
新来抢蛋糕，替代要你命
同行最激烈，五力合围杀
```

| # | 力量 | 核心考点 | 强弱判断 |
|:-:|:----|:---------|:---------|
| 1 | **现有企业间竞争** | 行业集中度、退出壁垒、产品差异化 | 同行多+壁垒高=竞争激烈 |
| 2 | **潜在进入者威胁** | 规模经济、品牌忠诚、资本需求、政府政策 | 进入壁垒越高→威胁越低 |
| 3 | **替代品威胁** | 性价比、转换成本、买方替代倾向 | 替代越便宜越好→威胁越大 |
| 4 | **供应商议价能力** | 集中度、替代品、前向一体化威胁 | 供应商越少→话语权越大 |
| 5 | **买方议价能力** | 购买批量、转换成本、后向一体化威胁 | 买家越集中→砍价越狠 |

### 💀 出题陷阱
- 官方教材里其实是 **五力 = 5种力量**（不是6个！**没有"政府"** ❌ 政府是PEST里的）
- "供应商议价能力强" ↔ "替代品多" 是反着来的，别记反

## ③ SWOT分析 — 内部 + 外部，四个格子

**S=优势 / W=劣势 / O=机会 / T=威胁**

### 🔟 秒记住
```
优势劣势看自己，机会威胁看外面
SO策略：优势打机会（进攻型）
WO策略：借机会补劣势（改进型）
ST策略：用优势抗威胁（防御型）
WT策略：避开劣势躲威胁（退缩型）
```

### 💡 答题模板
```
S（内部优势）：[公司]具有……（技术/品牌/资源）
W（内部劣势）：但存在……（不足/短板）
O（外部机会）：行业正在……（增长/政策利好）
T（外部威胁）：同时面临……（竞争/替代风险）

→ SO策略：利用[优势]抓住[机会] → [具体做法]
→ WT策略：避开[劣势]减少[威胁] → [具体做法]
```

### 💀 出题陷阱
- S/W必须是**内部**的（如技术、人才、资金）
- O/T必须是**外部**的（如市场、政策、竞争者）
- 考试综合题第一问很大概率就是SWOT 🔥

## ④ 价值链分析 — 拆开每环看谁最赚钱

**基本活动5步 + 支持活动4类**

### 🔟 秒记住
```
基本活动：进→产→出→卖→服务
支持活动：管、人、术、采
价值链条一拆开，哪里赚钱哪里亏
```

### 框架图
```
┌──────────────────────────────────────────────────┐
│                  支 持 活 动                       │
│  ┌────────────────────────────────────────────┐  │
│  │ 企业基础设施（财务/法务/规划）             │  │
│  ├────────────────────────────────────────────┤  │
│  │ 人力资源管理                               │  │
│  ├────────────────────────────────────────────┤  │
│  │ 技术开发（研发/工艺/流程）                 │  │
│  ├────────────────────────────────────────────┤  │
│  │ 采购管理（原材料/设备/服务采购）           │  │
│  └────────────────────────────────────────────┘  │
├──────────────────────────────────────────────────┤
│                  基 本 活 动                       │
│  ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐  │
│  │ 内部 │→│ 生产 │→│ 外部 │→│ 市场 │→│ 服务 │  │
│  │ 后勤 │ │ 运营 │ │ 后勤 │ │ 营销 │ │      │  │
│  └──────┘ └──────┘ └──────┘ └──────┘ └──────┘  │
└──────────────────────────────────────────────────┘
```

### 💀 出题陷阱
- 基本活动5个，**服务**是第5个，不要漏掉
- 支持活动里的"技术开发"≠研发部，是所有与技术相关的活动

## ⑤ 波士顿矩阵 — 四个格子管业务

**明星 / 金牛 / 问题 / 瘦狗**

### 🔟 秒记住
```
明星烧钱要投资，金牛稳定能收钱
问题孩子要培养，瘦狗赶紧处理掉
```

```
            市场增长率
         高 ←──────→ 低
      ┌────────────┬────────────┐
   高 │ ⭐ 明星     │ 🐄 金牛    │
      │ 高增长+高份额 │ 低增长+高份额 │
      │ → 加大投资   │ → 收割利润   │
      ├────────────┼────────────┤
   低 │ ❓ 问题     │ 🐕 瘦狗    │
      │ 高增长+低份额 │ 低增长+低份额 │
      │ → 培育或放弃 │ → 撤退/剥离  │
      └────────────┴────────────┘
```

### 💀 出题陷阱
- 判方向：明星→金牛（自然过渡），问题→明星（需要培养）
- 考法变化：近年常在大题里结合案例考业务组合调整

# 第2类 · 选择类模型

---

## ⑥ 三种基本竞争战略 — 只能选一条

**成本领先 / 差异化 / 集中战略**

### 🔟 秒记住
```
三条路走一条，不可兼得
你要便宜→成本领先
你要特色→差异化
你要聚焦→集中战略
```

| 策略 | 核心 | 适用条件 | 风险 |
|:----:|:----|:---------|:----:|
| **成本领先** | 成本最低，价格最便宜 | 产品同质化、价格敏感型客户 | 被技术颠覆、对手更低价 |
| **差异化** | 独一无二，客户愿意多付钱 | 客户有独特需求、对手搞不定 | 模仿者出现、特色不再值钱 |
| **集中战略** | 专注一个细分市场 | 细分市场有特点、大厂看不上 | 细分市场萎缩、大厂进来了 |

### 💀 出题陷阱
- 集中战略 = 集中成本领先 **或** 集中差异化（二选一）
- "卡在中间"（stuck in the middle）= 哪个都没选对 → 考试坑点

---

## ⑦ 战略类型 — 三个层次各管各的

**公司层 / 业务层 / 职能层**

### 🔟 秒记住
```
公司定方向，业务定打法，职能定干活
一层管一层，别越级
```

| 层次 | 谁定 | 管什么 | 包括 |
|:----:|:----|:-------|:----|
| **公司层战略** | CEO/董事会 | 做什么业务、往哪走 | 一体化/密集型/多元化/收缩/稳定 |
| **业务层战略** | 事业部/部门 | 怎么打赢竞争对手 | 成本领先/差异化/集中 |
| **职能层战略** | 各职能部门 | 日常怎么干 | 营销/财务/研发/生产/人力 |

### 💀 出题陷阱
- 综合题经常让你"分别分析该公司三个层面的战略" → 三个层面都要写，缺一个扣分

### 公司层战略速记表

```
发展战略（扩张）
├── 一体化战略          ← 买上下游或同行
│   ├── 横向一体化      ← 收购同行来抢地盘
│   └── 纵向一体化      ← 收上下游
│       ├── 前向一体化  ← 收渠道/零售商
│       └── 后向一体化  ← 收供应商
├── 密集型战略          ← 现有业务深挖
│   ├── 市场渗透        ← 现有市场卖更多
│   ├── 市场开发        ← 新市场卖现有产品
│   └── 产品开发        ← 现有市场推新产品
└── 多元化战略          ← 新业务新市场
    ├── 相关多元化      ← 跟主业有关
    └── 不相关多元化    ← 跟主业无关（跨界）

收缩战略（瘦身）：扭转/撤退/清算
稳定战略（不动）：按兵不动
```

## ⑧ 发展战略途径 — 三种方式

**内部开发 / 并购 / 战略联盟**

| 途径 | 一句话 | 优点 | 缺点 |
|:----:|:------|:----|:----|
| **内部开发** | 自己慢慢干 | 可控、知根知底 | 慢、资源投入大 |
| **并购** | 花钱买现成 | 快、直接拿市场 | 贵、整合难、文化冲突 |
| **战略联盟** | 合作不结婚 | 灵活、风险共担、资源共享 | 难管控、利益冲突 |

### 💀 出题陷阱
- 考"适合哪种途径"时，关键看**时间紧迫程度**和**资源是否充足**

## ⑨ 组织结构 — 五大常见类型

| 类型 | 一句话 | 适用场景 |
|:----:|:------|:---------|
| **创业型** | 老板一人说了算 | 小公司 |
| **职能制** | 按部门分（生产部/财务部/人力部） | 中型企业，产品单一 |
| **事业部制** | 按产品/地区分独立事业部 | 大型企业，多产品线 |
| **矩阵制** | 职能线 + 项目线双管 | 项目型公司（如建筑/咨询） |
| **网络结构** | 核心团队+外包 | 轻资产、灵活型 |

### 💀 出题陷阱
- 矩阵制"双线汇报"是高频考点 → 优点：资源利用灵活；缺点：容易扯皮

## ⑩ 平衡计分卡 BSC — 四个维度看绩效

**财务 / 客户 / 内部流程 / 学习与成长**

### 🔟 秒记住
```
财务看赚钱，客户看满意
内部看效率，成长看未来
四个维度一张卡，业绩不再只看钱
```

| 维度 | 考什么 | 指标举例 |
|:----:|:------|:---------|
| **财务** | 股东怎么看我们 | 投资回报率、利润率、收入增长率 |
| **客户** | 客户怎么看我们 | 客户满意度、市场份额、客户留存率 |
| **内部流程** | 我们哪里做得最好 | 生产效率、质量合格率、交付时间 |
| **学习与成长** | 我们能不能持续进步 | 员工培训、专利数、员工满意度 |

### 💀 出题陷阱
- 每个维度通常设**3-4个指标**
- 2026新趋势：把BSC跟**战略地图**一起考，四个维度是层层递进的

## ⑪ 战略变革三阶段 — 库尔特·勒温

**解冻 → 变革 → 再冻结**

### 🔟 秒记住
```
先融冰（让大家有危机感）
再变（推动改变）
后冻住（让新状态变成习惯）
```

| 阶段 | 干啥 | 常见方法 |
|:----:|:----|:---------|
| **① 解冻** | 让大家意识到必须变 | 宣告危机、打破惯性、制造紧迫感 |
| **② 变革** | 实施新方案 | 新流程、新架构、新制度 |
| **③ 再冻结** | 稳固新状态 | 制度固化、文化建设、奖惩配套 |

### 💀 出题陷阱
- 战略变革的**阻力来源**也是常考点：
  - 个人层面：不安全感、习惯、经济顾虑
  - 组织层面：结构惯性、资源限制、既有利益

# 📌 考试实战心法

## 综合题三段论（30分大题，核心不能丢）

```
第1步：识别模型 → "这道题考的是五力模型/价值链"
第2步：默写框架 → 五力的5个力量/价值活动的9个环节
第3步：套案例 → "根据案例，[公司]在[某力量]方面表现是[……]"
```

## 绝对不要做的事

❌ 不写模型名称直接写分析 → 没分
❌ 只背模型不结合案例 → 扣分
❌ 回答太多废话 → 改卷看关键点，写多不给你加感情分

## 各题建议时间

| 题型 | 分值 | 时间 |
|:----|:---:|:----:|
| 单选26题 | 26分 | 15-18min |
| 多选16题 | 24分 | 15-18min |
| 简答4题 | ~25分 | 50-60min |
| 综合1题 | 30分 | 40-45min |
| 检查 | — | 5-10min |
