# 009 多元化优缺点与动因

## 一句话说清
为什么要多元化？因为**鸡蛋不能放一个篮子、资源有富余、新领域利润更高**。但同时也有风险：分散资源、不熟悉新领域、整合困难。

## 内涵

### 采用多元化的三大原因（动因）
1. **现有产品/市场增长空间受限** → 现有产品难再创新，现有市场无法满足目标
2. **企业现金富余** → 满足现有生产后还有剩余资金
3. **新领域利润更高** → 多元化进入的领域比原有领域有更高的回报

### 多元化战略的**优点**（7条）
1. **分散风险** → 鸡蛋放进不同的篮子
2. **更容易融资** → 规模扩大，更易从资本市场获得融资
3. **获得新增长点** → 新领域利润更高
4. **充分利用闲置资源** → 有形、无形、人力资源得到利用
5. **运用盈余资金** → 多余资金有去处
6. **获得财务利益** → 盈利领域和亏损领域合并纳税
7. **利用原有声誉** → 原有领域的良好声誉和形象助益新领域

### 多元化战略的**风险**（5条）
1. **资源被分散** → 新领域瓜分原有领域的资源和管理层注意力
2. **风险不降反升** → 市场风险存在时，多元经营反而加大风险
3. **进入不熟悉领域** → 伴随经营风险
4. **退出风险** → 发现投资错误却退出困难
5. **内部整合风险** → 新领域影响各部门经营，引发整合困难

> **速记：优点——分险融、增用财声（分散风险、容易融资、新增长点、利用资源、运用资金、财务利益、利用声誉）**
> **速记：风险——分资、升险、不熟、难退、难整合**

## 🌍 生活实例（小米）
- **动因·资金富余**：小米手机业务成熟后积累了大量现金→用这些钱投资生态链和造车
- **动因·新领域利润更高**：智能手机市场饱和、利润率低→造车（SU7）目标高端利润空间更大
- **优点·分散风险**：小米不止靠手机赚钱→IoT+互联网服务+造车→多元收入结构
- **优点·利用资源**：MIUI系统、小米品牌形象→复用到了电视、笔记本、智能家居
- **风险·分散资源**：造车投入巨大（三年100亿）→ 手机研发和生态链投资资源被挤压
- **风险·不熟悉领域**：汽车制造工艺和手机完全不同→小米需要从零学习

## 🔗 考点
- ⭐⭐ 中频！选择题直接考优缺点的罗列，主观题偶尔结合案例分析
- 注意优点和风险的"数量"：7个优点、5个风险——逐条记忆
- 考点趋势：近年将多元化优缺点的纯记忆题变少，更多是结合案例让分析
- 秒杀：看到"分散风险、充分利用资源"→优点的标志；看到"分散资源、整合困难"→风险的标志
- 🧠 记口诀：优点7条记"分险融、增用财声"；风险5条记"分资、升险、不熟、难退、难整合"
