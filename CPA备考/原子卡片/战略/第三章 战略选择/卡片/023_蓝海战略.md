# 蓝海战略

## 一句话说清
别在**红海**跟人杀到血肉模糊（存量拼价格），去创造一块**没人跟你抢的新市场**（用创新满足新需求），让竞争变得无关紧要。

## 内涵

### 红海 vs 蓝海（表3-12）

| 比较维度 | "红海"战略 | "蓝海"战略 |
|:--------|:----------|:----------|
| **核心逻辑** | 在现有市场空间竞争 | 创造无竞争的新市场空间 |
| **竞争方式** | 打败竞争对手 | 让竞争变得无关紧要 |
| **市场定位** | 利用现有需求 | 创造和获取新需求 |
| **价值导向** | 在价值和成本之间权衡取舍 | 打破价值-成本权衡 |
| **企业活动** | 差异化或低成本选其一 | 同时追求差异化和低成本 |
| **战略重点** | 适应现有产业结构 | 塑造新的产业结构 |

> **秒记**：红海→杀价抢存量；蓝海→创新做增量

### 战略基石：价值创造
蓝海不关注打败对手，而是让客户和企业的价值都出现飞跃——**价值创造**而非价值分配。核心是同时追求**差异化和低成本**（打破波特说的一山不能容二虎的假设）。

### 6项战略原则

| 原则 | 降低的风险 | 核心内容 |
|:---|:---------|:---------|
| **① 重建市场边界** | 搜寻的风险 | 用6条路径跳出竞争，开创蓝海 |
| **② 注重全局而非数字** | 规划的风险 | 画战略布局图，把战略定位可视化 |
| **③ 超越现有需求** | 规模的风险 | 聚焦"非顾客"的共同点，而非细分差异点 |
| **④ 遵循合理的战略顺序** | 商业模式风险 | 买方效用→价格→成本→接受（四步验证） |
| **⑤ 克服关键组织障碍** | 组织的风险 | 认知/资源/动力/组织政治四大障碍 |
| **⑥ 将战略执行建成战略的一部分** | 管理的风险 | 执行遵循"公平过程" |

### 重建市场边界的6条路径

| 路径 | 核心思路 | 关键概念 |
|:---|:--------|:--------|
| **路径一：审视他择产业** | 跳出本行业，看顾客还有什么替代选择 | 他择品 > 替代品（功能和形式都不同的替代方案） |
| **路径二：跨越战略群体** | 搞清真正决定顾客选择的是什么因素 | 突破狭窄的行业群体思维 |
| **路径三：重新界定产业的买方群体** | 买方不只是付款人，还有使用者和影响者 | 关注"买方链条" |
| **路径四：放眼互补性产品或服务** | 分析顾客使用产品前中后的全部需求 | 互补品机会 |
| **路径五：重设客户的功能性或情感性诉求** | 挑战现有功能/情感定位，发现新空间 | 功能vs情感导向 |
| **路径六：跨越时间** | 预测外部趋势，提前布局技术和政策变化 | 趋势洞察 |

## 🌍 生活实例（小米）
- **小米生态链 = 典型的蓝海路径**：
  - **路径一（他择产业）**：小米不是做传统家电，而是做"智能家居系统"，用户可手机统一操控所有设备——这是对传统家电的**他择**
  - **路径三（重新界定买方）**：小米手环/智能家居不仅卖给购买者，更是让**家庭成员共同使用**和**影响彼此购买决策**
  - **路径四（互补性产品）**：手机+手环+电视+灯+空调+门锁——顾客买前中后都有场景联动，形成互补产品体系
  - **路径五（重设情感诉求）**：小米把"高科技"从冷冰冰变成"酷且亲民"——极客玩家的情感诉求
- **小米汽车SU7**：不是简单做电动车，而是做"人车家全生态"——汽车成为手机和家居的延伸，开创了全新的用户场景体验
