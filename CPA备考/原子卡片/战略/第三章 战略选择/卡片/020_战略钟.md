# 战略钟

## 一句话说清
鲍曼在波特三战略基础上画了一张**价格—价值坐标系**，把企业可能的竞争战略按"你卖多少钱、顾客觉得值多少"分成8条路，其中5条能走通，3条是死路。

## 内涵

### 坐标系逻辑
- **横坐标**：产品的价格（从低到高）
- **纵坐标**：顾客认可的产品价值（从低到高）
- 将企业可能的竞争战略在平面上用8种路径表现

### 8种路径详解（图3-6）

```
                           价值 ↑
                               │
                      ┌──── 途径5 ───┐
                      │ 高值高价      │
                      │（集中差异）   │
                      └──────┼───────┘
                             │
         途径4 ──────────────┤
       （高值/差异化）        │
                             │
                    ┌───途径3──┐
                    │  混合战略 │
                    └─────┬───┘
                          │
         途径2 ───────────┤
       （低价/成本领先）    │
                          │
         途径1 ───────────┤
       （低价低值/集中成本） │
                          │
          └──────────┬────┴────────→ 价格
                    低             高
                    ▲ 失败战略区域 ▲
               途径6/7/8（提价不提值等）
```

| 路径 | 类型 | 所属战略 | 说明 |
|:----|:----|:--------|:-----|
| **途径1** | 低价低值 | **集中成本领先** | 价格下降，价值也降。面向**价格敏感**、低收入群体。如廉价超市、基本款产品 |
| **途径2** | 低价 | **成本领先** | 降价但保持质量与服务不变。如规模效应降本后让利给消费者 |
| **途径3** | 混合战略 | **成本领先+差异化** | 既降价又提价值，同时获得两种优势，竞争力最强。理论来源：高质量→高份额→规模经济→低成本 |
| **途径4** | 高值 | **差异化** | 价格不变但价值提高。如升级产品功能/服务而不涨价 |
| **途径5** | 高值高价 | **集中差异化** | 价格和价值同时增加。面向**高端消费群体**，如高档购物中心、高端酒店 |
| **途径6** | 提价不提值 | ❌失败战略 | 涨价但价值不变，顾客觉得不值 |
| **途径7** | 提价降值 | ❌失败战略（险） | 涨价还降品质，最危险（除非有垄断权） |
| **途径8** | 降价降值 | ❌失败战略 | 价格不变但降低质量或服务 |

### 有效战略 vs 失败战略
- **有效（5种）**：途径1、2、3、4、5
- **失败（3种）**：途径6、7、8

### 混合战略（途径3）为何可能成功？
理论上，企业可以同时获得成本领先和差异化优势，原因：
1. **高质量→高份额→规模经济→低成本**
2. 高质量产品**累积经验降成本速度更快**（经验曲线）
3. **提高生产效率**同时降低平均成本

## 🌍 生活实例（小米）
- **途径1（低价低值）**：小米早期的入门款红米手机，价格极低、配置基础，面向低收入/学生群体
- **途径2（低价）**：小米生态链产品（如充电宝、插线板），比同类品质品牌便宜但质量不差
- **途径3（混合）**：小米手机旗舰系列——比苹果便宜很多，但配置和体验接近甚至更好，同时实现了"低价+高价值"
- **途径5（高值高价）**：小米大师系列电视/MIX系列手机——定位高端，价格高但配置也对得起高端
- **途径6~8（失败）**：小米早期争议——一些产品初期定价过高但体验未达预期，就接近途径6的风险区

## 🔗 考点
- ⭐⭐ 中频！主要考：8条路径的分类（哪些有效、哪些失败）+途径3混合战略的合理性解释
- 高频陷阱：区分途径1（集中成本领先）和途径2（成本领先）
- 🧠 记框架：5条有效路径(1~5)+3条死路(6~8)+途径3混合战略理论
