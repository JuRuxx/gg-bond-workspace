# 人力资源战略

## 一句话说清
人力资源战略就是在**对的时间、对的地点、有对的人**——招进来、用起来、激励好、留得住，让人成为企业的核心竞争力。

## 内涵

### 人力资源的战略作用
HR的核心是**取得、开发、管理和激发**企业关键资源。好的HR策略必须：①清晰一致地引导员工为组织目标服务；②有灵活性，能应对外部变化；③在约束条件下找到最优解。

### 人力资源战略的八大目标（选择/简答要点）
1. 精准识别短/中/长期所需的人才类型
2. 通过培训、发展、教育激发潜力
3. 提高早期表现优秀员工的占比
4. 招聘有潜力的年轻新就业者
5. 防止竞争对手挖走人才
6. 招聘有经验的人才并帮其适应企业文化
7. 激励高绩效者并培养忠诚度
8. 创造人才成长的文化，建金牌团队

### 关键机制
| 机制 | 核心要点 |
|:----|:--------|
| **人力资源计划** | 消除人才供需缺口，考虑6方面：招聘/培训/再发展/生产力/冗余/保持 |
| **招聘方式** | 内部招聘（熟悉文化但难有新血液）vs 外部招聘（新鲜思路但成本高） |
| **继任计划** | 发现高潜人才→备选管理层→持续领导力→客观评估（不能只丢给HR） |
| **激励与奖励** | 稳定保障+物质福利+精神激励，三管齐下 |
| **绩效评估** | 基于效果、目标实现、资源利用三维度计量 |
| **员工培训** | 培训需求分析（组织/人员/任务三个层次）、培训方案设计、培训效果评估 |

## 🌍 生活实例（小米）
小米的HR实践典型体现了互联网科技公司的特点：
- **招聘方式**：早期大量**内部招聘**和"人拉人"（小米早期一半员工是创始人从金山/微软/Google拉来的），快速建立核心技术团队。
- **激励与奖励**：小米2018年上市前推行**股权激励**，核心员工持有期权，形成"合伙人"心态。2021年奖励技术骨干近1.2亿股，锁住高端人才。
- **培训与发展**：小米有"小米大学"内训体系，研发人员每年参加技术开放日和黑客马拉松。造车团队大量从外部**招聘有经验人才**（前宝马、奔驰工程师）并帮其融入"小米文化"——这是八大目标里第6条的鲜活案例。
- **继任计划**：2023年雷军宣布"小米内部创业"战略，让各业务线总经理有更大自主权，实质是在培养下一个阶段的管理梯队。

## 🔗 考点
- ⭐⭐ 选择题考点：八大目标（题干中的考点清单）、继任计划的4个基本要求、培训需求分析的三个层次（组织/人员/任务）、培训方法选择的原则
- 🧠 记框架：8大目标+5关键机制(计划/招聘/继任/激励/培训)

## 🏗 讲义位置
第三章·第三节·人力资源战略部分（八大目标、继任计划、员工培训）

## 📚 章节联系
人力资源战略是**波特价值链**中的辅助活动——人才管理支撑所有基本活动（生产、营销、服务）。激励与绩效评估机制影响**公司战略**中战略控制的执行效果。继任计划在高管断层风险上的应对，也呼应**风险管理的应对策略**。
