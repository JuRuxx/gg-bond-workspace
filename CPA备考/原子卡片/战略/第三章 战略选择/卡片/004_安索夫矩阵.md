# 004 安索夫矩阵（产品—市场战略组合矩阵）

## 一句话说清
安索夫用**"现有/新产品"×"现有/新市场"**画出一个2×2矩阵，产生4种战略：市场渗透、市场开发、产品开发、多元化。前三者合称**密集型战略**。

## 内涵

### 表3-3 "产品—市场战略组合"矩阵（安索夫矩阵）

|  | 现有产品 | 新产品 |
|:---|:---|:---|
| **现有市场** | **①市场渗透**<br>→ 老产品老市场<br>→ 卖更多、抢份额<br>→ "坚守阵地" | **③产品开发**<br>→ 新产品老市场<br>→ 给老客户新东西<br>→ 研发/改良/延寿 |
| **新市场** | **②市场开发**<br>→ 老产品新市场<br>→ 换个地方卖<br>→ 开新区、找新客户 | **④多元化**<br>→ 新产品新市场<br>→ 进入全新领域<br>→ 属于发展战略但不属密集型 |

### 核心要点
- **密集型战略** = 市场渗透 + 市场开发 + 产品开发（安索夫矩阵的前三格）
- **多元化**虽然也在矩阵中占一个格子，但它本身是另一大类战略（不符合"密集"的定义）
- 判断口诀：用"产品×市场"两维定位，锁定格子即可

```
              市场（旧）                 市场（新）
                  │                         │
                  │                         │
 产品（旧） ─────┼──── 市场渗透 ──────── 市场开发
                  │                         │
                  │                         │
 产品（新） ─────┼──── 产品开发 ──────── 多元化
                  │                         │
```

## 🌍 生活实例（小米）
- **市场渗透**：小米6系列持续降价促销，通过电商大促（618、双11）提高已有市场的销量
- **市场开发**：小米手机从中国→印度→东南亚→欧洲→拉美 → 现有手机产品进新市场
- **产品开发**：小米从红米Note系列→小米数字系列→小米MIX系列 → 给已有市场的老用户推新机型
- **多元化**：小米从手机→智能电视→路由器→IoT设备→造汽车（SU7） → 新产品全新市场

## 🔗 考点
- ⭐⭐⭐ 高频！选择题必考——给案例描述判断属于矩阵哪一格
- 主观题也可能出：案例材料给了企业行为，让判断是哪种密集型战略
- 秒杀：锁定"产品变不变"+"市场变不变"两个维度，对号入座
- 坑点：多元化虽然也在矩阵第4格，但**不属于密集型战略**！
- 🧠 记框架：2×2矩阵4格+哪三格是密集型
