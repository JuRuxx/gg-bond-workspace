# 014 内部发展战略

## 一句话说清
内部发展（新建/内生增长）就是"靠自己从零干起"——不靠收购，完全利用内部资源自建业务、研发产品，慢慢长大。风险低但速度慢。

## 内涵

### 定义
内部发展战略（又称内生增长战略）是指企业在不收购其他企业的情况下，依靠**自身的资源**（利润、规模、活动）来实现发展壮大。这在需要高科技设计或制造方式的企业中是主要的战略方式。

### 内部发展的动因（9条）
1. 慢慢发展→更了解自身产品及市场
2. 没有合适的并购对象（想买但买不到）
3. 能保持统一的企业文化和经营管理，减轻混乱
4. 为管理者提供晋升通道
5. 无需支付高额的并购溢价
6. 没有并购中隐藏的或无法预测的风险
7. **只有内部发展才能实现真正的企业创新**
8. 循序渐进，易获得财务支持，成本随时间分摊
9. **风险较低**（不用承担业主以前的错误决策后果）

### 内部发展的缺点（5条）
1. 逐步扩大→竞争加剧
2. 无法接触外部知名企业的管理和技术→长期落后风险
3. 发展缓慢→难获得规模效应或经验曲线效应
4. 市场发展快时→内部发展太慢，丧失机会
5. 进入新市场的进入壁垒高

### 内部发展的应用条件（3条）
1. 产业处于**不均衡**状态，结构性障碍尚未建立（新兴产业特征）
2. 产业内现有企业的行为性障碍容易被制约（报复成本＞收益）
3. 企业有能力克服进入障碍，或进入后收益＞克服障碍的成本

## 🌍 生活实例（小米）
- **澎湃OS自研**：小米从零开始自研操作系统澎湃OS → 全部利用内部的技术人员、研发资源，不走收购路线 → 内部发展的典型
- **小米汽车部分自研**：自动驾驶算法、电驱系统、压铸技术等核心技术自研 → 而非直接收购一家整车厂
- **小米工业设计团队**：小米设计能力是内部慢慢培养积累的，不是通过收购设计公司获得的
- **对比说明**：小米选择内部发展造车（部分自研）而非直接收购整车厂 → 原因：没有合适的收购对象（动因#2）+ 保持小米文化和创新（动因#3/#7）

## 🔗 考点
- ⭐⭐ 中频！选择题考内部发展的动因/缺点辨析，尤其注意与并购对比
- 主观题少单独考，但常作为"并购vs内部发展"的对比分析
- 秒杀：看到"自研/自建/从零开始" → 内部发展；看到"收购/买下" → 并购
- 注意：内部发展**不等于**稳定战略——内部发展是发展战略的途径，是"进攻但不靠收购"
- 🧠 记准用词：靠自己从零干→自研自建、风险低但慢，vs并购的反面
