# 015 企业战略联盟

## 一句话说清
战略联盟就是"合伙干大事不求拥有"——两个或多个企业合作但不合并，各自独立又共享资源。它是介于企业和市场之间的"中间组织"，被誉为"20世纪20年代以来最重要的组织创新"。

## 内涵

### 基本特征（三大视角）
| 视角 | 特征 |
|:----|:-----|
| **经济组织形式** | 介于**企业和市场之间**的中间组织——联盟内交易既非企业内交易，也非市场交易 |
| **企业关系** | 资源共享、优势互补、彼此信任、相互独立 → **平等合作关系**（相互往来平等、合作长期、利益互补、组织开放） |
| **企业行为** | **战略性合作关系** → 技术优势互补、加快创新、降低风险 |

### 战略联盟形成的动因（6大动因）
1. **促进技术创新** → 共担创新成本和风险
2. **避免经营风险** → 信息共享，把握不确定环境
3. **避免或减少竞争** → 竞争变合作，减少竞争费用
4. **实现资源互补** → 各取所长，共享优势资源
5. **开拓新的市场** → 快速实现经营范围多样化及地区扩张
6. **降低协调成本** → 相对于并购无需企业整合

### 战略联盟的主要类型

| 类型 | 细分 | 说明 |
|:----|:----|:-----|
| **①股权式联盟** | **合资企业** | 共担风险、共享收益，股权各约50%（强调平等独立） |
|  | **相互持股投资** | 各方持有对方少量股份，双向持股，人员资产不变 |
| **②契约式联盟**<br>（功能性协议） | **技术交流协议** | 互相交流技术知识资料 |
|  | **合作研究开发协议** | 共享科研成果、共同使用设施，共同开发新产品 |
|  | **生产营销协议** | 共同生产、共同销售，各保独立性 |
|  | **产业协调协议** | 建立全面协作与分工的产业联盟体系 |

### 表3-9 股权式联盟 vs 契约式联盟

| 比较维度 | 股权式联盟 | 契约式联盟 |
|:--------|:----------|:----------|
| **组织形式** | 实体组织（合资企业、相互持股） | 非实体组织（功能性协议） |
| **联盟关系** | 更紧密，利益捆绑 | 较松散，相对独立 |
| **灵活性** | 灵活性差，退出成本高 | 灵活性强，进退自如 |
| **控制力** | 股权比例决定控制力 | 按协议约定，控制难度大 |
| **长期性** | 长期稳定 | 可短可长 |
| **保密性** | 较好（利益共同体） | 较差（易泄露核心信息） |

> **秒记：股权式——紧密稳定但笨重；契约式——灵活自由但不牢**

## 🌍 生活实例（小米）
- **契约式联盟-合作研发**：小米与**徕卡**签订影像技术合作协议 → 不是收购，而是技术+品牌合作，共同开发小米手机影像系统
- **契约式联盟-生产营销**：小米与**高通**长期合作 → 优先获得骁龙芯片供应，联合营销（"骁龙+小米"品牌绑定）
- **股权式联盟-相互持股**：小米入股**TCL**（约0.48%股权）→ 少量持股但不控制，加强供应链合作关系
- **合资企业**：小米在印度与当地企业成立合资公司 → 共同开拓印度市场

## 🔗 考点
- ⭐⭐⭐ 高频！选择题必考——股权式vs契约式区别（表3-9）
- 六大动因 → 主观题考给出案例说明建立联盟的动机
- 三个基本特征（中间组织、平等合作、战略合作）→ 客观题判断
- 秒杀："合资/持股"→股权式；"合作协议"→契约式
- 🧠 记框架：股权式vs契约式对比+6大动因+3个基本特征
