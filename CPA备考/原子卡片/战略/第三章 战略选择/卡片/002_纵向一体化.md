# 002 纵向一体化

## 一句话说清
纵向一体化就是企业沿着产业链"上扒下踩"——向上控制供应商（后向），向下控制分销商/零售商（前向），核心是**降低交易成本、保证质量、稳定供需**。

## 内涵

### 什么是纵向一体化？
沿着产业价值链向前或向后延伸和拓展原有业务，实质是**扩大单一业务的经营范围**。

### 表3-1 纵向一体化战略的优缺点

| 方面 | 优点 | 缺点 |
|:----|:----|:----|
| **纵向一体化整体** | ①节约与上、下游企业的交易成本<br>②保证产品质量、稳定供需关系 | ①进入新领域带来不熟悉的风险<br>②投资金额大、资产专用性强，增加退出成本 |
| **前向一体化**（控制下游） | ①降低销售渠道费用<br>②贴近终端市场，更快响应需求变化 | — |
| **后向一体化**（控制上游） | ①降低采购成本<br>②保证原材料供应稳定、质量可控 | — |

> **速记口诀：前向卡渠道，后向卡原料，纵向一体化省交易成本**

### 表3-2 前向/后向一体化适用条件

| 战略类型 | 主要适用条件 |
|:--------|:------------|
| **前向一体化** | ①现有销售商成本高、不可靠或不满足企业需要<br>②企业所在产业增长潜力较大<br>③企业具备所需资金和人力资源<br>④销售环节利润率较高 |
| **后向一体化** | ①供应商数量少而需求者众多（供应商议价能力强）<br>②企业所在产业增长潜力较大<br>③企业具备所需资金和人力资源<br>④供应环节利润率较高<br>⑤产品价格稳定至关重要 |

> **考试口诀：前向——销售商不给力、增长潜力大、销售利润高；后向——供应商太强势、增长潜力大、供应利润高**

### 图3-1/3-2 纵向一体化示意图
```
          后向一体化 ←─────────────── 前向一体化
                                  ───────────────→
    ┌────────────┐        ┌────────────┐        ┌────────────┐
    │  供应商    │  ←──── │  生产企业  │  ────→  │  分销/零售  │
    │  （上游）   │  后向  │  （核心）  │  前向   │  （下游）   │
    └────────────┘        └────────────┘        └────────────┘
```

## 🌍 生活实例（小米）
- **后向一体化**：小米投资松果电子（自研澎湃芯片）→ 控制上游芯片供应链，降低对外部供应商依赖
- **前向一体化**：小米之家直营店 → 自己掌握销售渠道，减少对京东、天猫等平台的通道费依赖
- 小米生态链本质上也有纵向整合的影子：通过投资上游零部件企业，保证米家产品的质量和成本

## 🔗 考点
- ⭐⭐⭐ 高频！常结合波特五力模型考适用条件（供应商议价强→后向，销售商不给力→前向）
- 主观题易考：给案例判断是前向还是后向，并说明理由
- 秒杀：控制下游渠道→前向；控制上游原材料→后向
- 🧠 记框架：前向vs后向区分+适用条件对比+优缺点表
