# 集中化战略

## 一句话说清
企业**资源有限，打不起全市场大仗**，就聚焦一个细分市场，要么在这个小池子里**做到成本最低**（集中成本领先），要么**做得最特别**（集中差异）。

## 内涵

### 核心逻辑
集中化战略不追求在全行业竞争，而是针对**特定购买群体、产品细分或地域市场**，在目标细分市场内实行成本领先或差异化。分为两类：

```
                 集中化战略
                   │
        ┌──────────┴──────────┐
        ▼                     ▼
  集中成本领先             集中差异
  （在细分市场内拼低价）  （在细分市场内做独特）
```

### 适用企业
- 一般为**中小企业**——资源和能力有限，无法在全市场实施成本领先或差异化
- 也可以是大企业在某一**战略重点市场**的局部策略

### 优势
1. 能够**抵御产业5种竞争力**的威胁（在细分市场中做到极致）
2. **避免正面冲突**——不在大范围内与行业巨头硬碰硬

### 实施条件
1. 购买者群体之间在**需求上存在差异**
2. 目标市场在**市场容量、成长速度、获利能力**方面具有相对吸引力
3. 目标市场上**没有其他竞争者**采用类似竞争策略
4. 企业**资源和能力有限**，难以在全市场实施成本领先或差异化，只能针对细分市场

### 风险
1. **目标市场狭小**→需求有限，难以匹配较大生产规模
2. **购买者需求差异变小**→细分市场边界模糊，原有战略不再适用
3. **竞争对手进入与竞争**→大企业也可能盯上这个细分市场

## 🌍 生活实例（小米）
- **小米早期策略**：2011年小米成立时，国内手机市场被苹果（高端）和"中华酷联"（运营商渠道）瓜分。小米没有正面硬刚，而是**聚焦"线上极客发烧友"这一细分市场**——这群人懂参数、爱刷机、接受线上抢购模式，且对价格敏感——这正是**集中成本领先+集中差异化的混合体**
- **小米生态链中的"细分战场"**：小米手环早期聚焦"长续航+低成本智能手环"细分市场，避开Apple Watch的高端路线
- **小米海外新兴市场**：在印度市场聚焦"性价比智能手机"细分群体，快速抢占份额

## 🔗 考点
- ⭐⭐ 中频！常考判断属于集中成本领先 vs 集中差异
- 秒杀：看到"只做某一类客户/某个区域/某个产品线"→集中化；看到"大企业全面布局"→不是集中化
- 🧠 记准用词：在细分市场内做到成本最低(集中成本领先)或最特别(集中差异)
