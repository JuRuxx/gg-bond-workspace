# 016 企业战略联盟管控

## 一句话说清
联盟签了约不代表完事——还得管！两大手段：**协议约束**（白纸黑字）+ **信任维系**（软性润滑）。

## 内涵

### 管控手段一：订立协议
1. **严格界定联盟目标** → 明确各自要什么
2. **周密设计联盟结构** → 防止内部竞争
3. **准确评估投入资产** → 专利/技术等无形资产要估值
4. **规定违约责任和解散条款** → 出了事有章可循

### 管控手段二：建立合作信任
- 信任是**最难模仿、最稀缺的资源**
- 一方面减少监督成本
- 另一方面弥补正式协议的不足

### 为什么需要管控？
| 对比方式 | 优势 | 管控挑战 |
|:---|:---|:---|
| 并购 | 完全控制 | 整合难 |
| 内部发展 | 完全掌控 | 速度慢 |
| 战略联盟 | 灵活、低成本 | 关系松散，易失控 |

## 🌍 生活实例（小米）
- 小米与徕卡合作：双方明确目标（小米提升影像、徕卡扩大影响力）+ 协议约束
- 小米生态链：通过持股+协议双重管控

## 🔗 考点
- ⭐ 低频，了解即可，偶有选择题
- 两个手段：协议+信任，缺一不可
- 🧠 懂就行：协议(硬约束)+信任(软润滑)两手抓，缺一不可
