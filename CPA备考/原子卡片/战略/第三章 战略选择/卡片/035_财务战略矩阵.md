# 财务战略矩阵

## 一句话说清
财务战略矩阵就是通过"**赚钱还是亏钱**（回报率vs资本成本）"和"**有钱还是缺钱**（增长vs可持续增长）"两个维度，把业务分成四类，每类给一套对症的财务策略。

## 内涵

### 矩阵结构（图3-8⭐️⭐️⭐️）
```
                  投资资本回报率 - 资本成本
                       ↑ 创造价值
              第二象限  │  第一象限
           增值型现金剩余│ 增值型现金短缺
         ───────────────┼──────────────→ 增长差=销售增长率-可持续增长率
          增长差<0      │    增长差>0
          现金剩余      │    现金短缺
              第三象限  │  第四象限
           减损型现金剩余│ 减损型现金短缺
                       ↓ 减损价值
```

**纵轴** = 投资资本回报率 - 资本成本（>0创造价值，<0减损价值）
**横轴** = 销售增长率 - 可持续增长率（>0现金短缺，<0现金剩余）

### 四象限财务战略（表3-26⭐️⭐️⭐️）
| 象限 | 业务类型 | 特征 | 财务战略 |
|:----|:--------|:----|:--------|
| **第一象限** | **增值型现金短缺** | 赚钱但钱不够花（增长快、现金流紧张） | ①提高可持续增长率（提高经营效率、改变财务政策）②增发股份③减少/停止股利④并购"现金牛" |
| **第二象限** | **增值型现金剩余** | 赚钱且钱花不完（增长慢、现金富余） | ①加速增长（内部投资、收购相关业务）②增加股利支付③回购股份 |
| **第三象限** | **减损型现金剩余** | 亏钱但还有闲钱（盈利能力差、不缺现金流） | ①提高投资资本回报率（提高利润率/周转率）②降低资本成本③出售业务单元 |
| **第四象限** | **减损型现金短缺** | 既亏钱又缺钱 | ①彻底重组②出售业务③退出市场（清算） |

**速记口诀**：
- 第一象限缺钱→提高增长快筹钱
- 第二象限钱多→加速投资多分红
- 第三象限赚钱少→提高回报或卖掉
- 第四象限又亏又缺→重组卖掉或退出

### 关键公式
企业市场增加值 =（投资资本回报率-资本成本）×投资资本÷（资本成本-增长率）

影响价值创造的核心因素只有三个：**投资资本回报率 ↑、资本成本 ↓、增长率**（回报率>资本成本时，增长率越高越好；回报率<资本成本时，增长率越高越糟糕）。

## 🌍 生活实例（小米）
- **2014-2016年→第一象限（增值型现金短缺）**：小米手机高速增长，创造价值（投资回报率高），但现金严重短缺——高速扩张需要大量铺货、建供应链、搞研发。小米的策略：①提高经营效率（优化供应链，2015-2016年控制库存）②增发股份（2018年IPO）③零股利（所有利润再投资）。
- **2020-2023年→第二象限（增值型现金剩余）**：手机市场饱和，增长放缓，但IoT业务利润丰厚，现金充裕。小米的策略：①加速增长（投资造车——内部投资+收购DeepMotion自动驾驶公司）②增加股利支付（港股投资者要求分红，2022年起逐年提高）。
- **不同业务分属不同象限**：小米手机可看作第二象限（增值但有闲钱），而造车业务处于第一象限（花钱创造价值）。**整体矩阵要对每个业务单元分别分析。**

## 📚 章节联系
财务战略矩阵的"增长率差"概念源自**可持续增长率**（第二章财务管理基础），"投资资本回报率-资本成本"直接等于**EVA（经济增加值）**——与第六章**业绩评价**中的经济增加值考核方法衔接。矩阵第一象限"增发股份"的选项，又与**筹资策略**（表3-20股权融资优缺点）形成联动。同时，经营风险与财务风险的搭配决定了企业处于矩阵哪个位置。
