# 市场营销—市场细分

## 一句话说清
市场细分就是按照**地理、人口、心理、行为**四个维度把大海一样的消费者分成一个个小"鱼塘"，好让企业挑自己最擅长的那塘鱼来钓。

## 内涵
市场细分是确定目标市场的第一步，其核心是**找对变量、划清边界**。消费者市场有四大细分变量（口诀：**地人心行**）：

| 变量 | 举例 |
|:----|:-----|
| **①地理细分** | 南方vs北方、一线vs三线、气候带 |
| **②人口细分** | 年龄、性别、收入、职业、教育程度 |
| **③心理细分** | 生活方式、个性、社会阶层、价值观 |
| **④行为细分** | 使用时机、追求利益、使用频率、忠诚度 |

产业市场（面向企业客户）还有两个特有变量：**最终用户**和**顾客规模**。比如卖钢材给汽车厂vs家电厂，定价和营销策略完全不同。

细分之后就是目标市场选择——区别**无差异**（一个策略打天下，成本低但风险大）、**差异**（分群定制，成本高但风险分散）、**集中**（只打一个细分市场，专业但脆弱）。注意"多数谬误"——大家一窝蜂选最大子市场，结果反而竞争最激烈。

## 🌍 生活实例（小米）
小米早期用**心理细分+行为细分**锁定"追求高性价比的数码发烧友"这一细分群体（即"米粉"），而不是按地域或年龄做泛泛划分。这一批人热衷刷机、追求参数、为发烧而生，小米为他们定制MIUI系统、搞论坛互动、做跑分宣传——完美命中目标细分。后来小米做大后开始**按收入水平分层**，推出红米（低端走量）和MIX系列（高端探索），这是在不同细分市场做差异营销。

## 🔗 考点
- ⭐⭐ 常见选择题考点：四大细分变量对应的例子，"半数谬误"概念，产业市场特有两个变量
- 🧠 记口诀：'地人心行'=地理/人口/心理/行为四大细分变量
