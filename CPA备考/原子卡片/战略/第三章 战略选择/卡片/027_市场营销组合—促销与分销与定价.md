# 市场营销组合—促销与分销与定价

## 一句话说清
产品出来后，**促销**让人想买、**分销**让人买得到、**定价**让人买得起——3P联动决定了产品能不能从货架走进消费者的购物车。

## 内涵

### 促销策略（Promotion）
促销是营销中最具**控制权**的环节，目的是让顾客**关注→感兴趣→想拥有→下单**。四大工具：
1. **广告促销**：电视/网络/户外广告→树形象
2. **营业推广**：折扣券/试用品/赠品→刺激立即购买
3. **公关宣传**：新闻/公益活动→塑造公众好感
4. **人员推销**：销售代表直接沟通→搞定大客户/复杂产品

### 分销策略（Place）
解决"从工厂到用户手上"的问题：
- **直接分销**：厂家→消费者（如小米线上商城），省掉中间商
- **间接分销**：厂家→经销商→零售商→消费者
- **分销密度**：独家分销（专营）→选择分销（几家）→密集分销（到处有卖）

评估分销结构看**三性**：经济性（利润最大）、控制性（渠道听不听话）、适应性（能不能调整）。

### 定价策略（Price）
定价策略分两大类：

**① 差别定价法**（同产品不同价）：
| 方式 | 举例 |
|:----|:-----|
| 顾客细分 | 学生票/成人票/老人票 |
| 地点细分 | 火车上中下铺、不同区域座位 |
| 版本细分 | 精装书vs简装书、软件免费版vs专业版 |
| 时间细分 | 酒店淡旺季、机票早晚不同价 |
| 动态定价 | 机票/网约车随需求变化 |

**② 上市定价法**（新品定价）：
- **渗透定价**：低价冲量→抢市场份额（适合价格敏感市场）
- **撇脂定价**：高价先捞→后降价（适合技术领先产品，如iPhone首发）

## 🌍 生活实例（小米）
- **促销**：小米发布会是招牌广告+公关的组合拳——雷军亲自"人员推销"；线上限时抢购+米粉节是营业推广套路。
- **分销**：小米早期只走**线上直销**（直接分销），省掉中间商差价，支撑极致性价比。2016年后遭遇增长瓶颈，开始大举铺设**小米之家**线下店（密集分销），实现"线上下单+线下体验"的全渠道布局。
- **定价**：Redmi用**渗透定价**（799元走量），小米数字系列用**性价比定价**，MIX Fold用**撇脂定价**（万元起步）——同一公司、不同定位、不同定价。

## 📚 章节联系
定价中的"渗透定价"对应**竞争战略**的成本领先逻辑，"撇脂定价"对应差异化/技术领先逻辑。分销渠道类型也影响**采购战略**中的货源选择（中间商越大，议价能力越强）。促销与**品牌策略**直接挂钩——打广告就是建品牌认知。
