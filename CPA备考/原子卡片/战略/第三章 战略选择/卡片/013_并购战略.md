# 013 并购战略

## 一句话说清
并购（外部发展）就是"拿钱买成果"——通过收购其他企业来快速进入新领域、获得协同效应、减少竞争。但并购成功率不高，常因决策不当、整合不好、出价过高而失败。

## 内涵

### 表3-8 并购的类型

| 分类标准 | 类型 | 说明 |
|:--------|:----|:-----|
| **按并购双方所属产业** | **横向并购** | 同一行业、同类产品企业之间（如两啤酒企业合并） |
|  | **纵向并购** | 同一产业链上下游企业之间（如制造企业收购原料供应商） |
|  | **多元化并购** | 不同行业、无关联企业之间 |
| **按被并购方态度** | **友善并购** | 双方协商达成一致 |
|  | **敌意并购** | 被并购方反对，收购方强行收购 |
| **按并购方身份** | **产业资本并购** | 非金融企业作为并购方（以实业经营为目的） |
|  | **金融资本并购** | 投资银行、基金等金融机构并购（以资本增值为目的） |
| **按收购资金来源** | **杠杆收购** | 大多数收购资金来自借贷（高比例负债） |
|  | **非杠杆收购** | 以自有资金为主收购 |

> **速记：按产业→横/纵/多；按态度→友/敌；按身份→产/金；按资金→杠/非杠**

### 并购的动机（3大原因）
1. **避开进入壁垒，迅速进入** → 尤其适合需要审批的领域（如"买壳上市"）
2. **获得协同效应** → 1+1＞2（聚焦效应、转移互补、耦合激荡）
3. **克服负外部性，减少竞争** → 两败俱伤的竞争通过并购变成一家

### 并购失败的原因（4大原因）
1. **决策不当** → 没做充足分析 → 波特"吸引力测试"（进入成本测试+相得益彰测试）
2. **并购后整合不好** → 企业文化、组织结构、制度不同导致矛盾
3. **支付过高费用** → 未准确评估被并购方价值（参考方法：市盈率法、股票现价、净资产价值、现金流折现等）
4. **跨国并购的政治风险** → 需评估政治风险、多国化经营、当地化策略

## 🌍 生活实例（小米）
- **横向并购**：小米收购**美图手机**品牌权 → 同行业手机企业之间的并购，获得美图女性用户群和影像技术
- **多元化并购**：小米收购**紫米科技**剩余股权（全资控股） → 紫米做充电宝/电源适配器，和手机不完全相同
- **友善并购**：小米与TCL战略入股 → 双方协商一致，互相持股
- **杠杆收购**：小米至今较少采用高杠杆并购，多为自有资金
- **并购失败风险**：小米收购美图后，美图手机品牌影响力未达预期，整合效果一般

## 🔗 考点
- ⭐⭐⭐ 高频！四个分类标准（横向/纵向/多元化、友善/敌意、产业/金融、杠杆/非杠杆）→ 选择题必考
- 三大动机和四大失败原因 → 主观题常考结合案例
- 注意：并购战略中的"横向并购"和"横向一体化"基本等同；"纵向并购"和"纵向一体化"对应
- 秒杀：看产业关系→分横/纵/多；看协商与否→分友/敌
- 🧠 记框架：4个分类维度(产业/态度/身份/资金)+3动机+4失败原因
