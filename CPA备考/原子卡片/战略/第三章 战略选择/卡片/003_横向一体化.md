# 003 横向一体化

## 一句话说清
横向一体化就是**收购/合并同行竞争对手**，目的是实现规模经济、减少竞争压力、增强自身实力。

## 内涵

### 定义
企业收购、兼并或联合竞争企业的战略。图示如下（图3-1）：

```
                  ┌──────────────────────────────┐
                  │      供应商（上游）             │
                  └────────────┬─────────────────┘
                               │
                       后向一体化 ↑
                               │
                  ┌────────────┴─────────────────┐
  横向一体化 ←──→ │     生产者（核心企业）         │ ←──→ 横向一体化
    （收购同行）   │                               │      （收购同行）
                  └────────────┬─────────────────┘
                               │
                       前向一体化 ↓
                               │
                  ┌────────────┴─────────────────┐
                  │       分销商/零售商（下游）     │
                  └──────────────────────────────┘
```

### 主要目的
①实现**规模经济**（扩大生产至最优规模点）
②**减少竞争**压力（合并后竞争者数量减少）
③增强自身实力，获取竞争优势

### 适用条件（5条）
1. 产业内竞争激烈，企业数量较多
2. 该产业**规模经济较为显著**
3. 未违反反垄断法规，能在局部地区获得一定垄断地位
4. 产业处于成长期，增长潜力较大
5. 企业已具备横向一体化所需的资金、人力资源

## 🌍 生活实例（小米）
- 小米在手机市场的发展：初期通过收购/整合**摩托罗拉**的部分团队和专利 → 增强手机研发实力
- 小米生态链的横向整合：投资多家智能硬件公司（华米、紫米、云米等）→ 虽然不全是收购，但生态链模式本质是"横向联合"抢占不同细分领域的智能硬件市场
- 在电视领域：小米收购**TCL**的部分股权 → 增强电视供应链和制造能力，与海信、创维等竞争

## 🔗 考点
- ⭐⭐ 中频！判断是否属于横向一体化，注意与纵向一体化区分
- 常考选择题：给并购案例，问是横向还是纵向还是多元化
- 秒杀：同行之间并购→横向；上下游之间→纵向；不同行业→多元化
- 🧠 记框架：横向vs纵向区分+5条适用条件
