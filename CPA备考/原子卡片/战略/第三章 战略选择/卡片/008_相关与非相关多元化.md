# 008 相关与非相关多元化

## 一句话说清
多元化 = **新产品 + 新市场**，跳出现有领域。相关多元化是在原有优势上"同心扩散"，非相关多元化是进入全新的无关领域。

## 内涵

### 图3-3 两种多元化新产业与原有产业的关系

```
【相关多元化】（同心多元化）——利用原有核心技术/市场/渠道
       ┌─────────────────┐
       │     原有产业     │  （核心技术/市场优势）
       └──┬───┬───┬───┬──┘
          │   │   │   │
          ▼   ▼   ▼   ▼
       ┌──┬──┬──┬──┬──┬──┐
       │新│新│新│新│新│新│ （在生产、技术、市场等方面相关）
       └──┴──┴──┴──┴──┴──┘

【非相关多元化】（离心多元化）——进入完全无关的全新领域
       ┌─────────────────┐  ┌─────────────────┐
       │     原有产业     │  │     全新产业     │
       │ （A领域）        │  │ （B领域，完全无关）│
       └─────────────────┘  └─────────────────┘
```

### 相关多元化（同心多元化）
- 定义：以现有产业为基础，进入在生产、技术、市场等方面**相关**的新领域
- 优势：利用原有产品的知识、生产能力、销售渠道 → 获得**融合优势**（1+1＞2）
- 适用：现有市场吸引力下降，但企业在现有领域有良好信誉和优势

### 非相关多元化（离心多元化）
- 定义：进入与原有领域**毫不相关**的新产业
- 目标：**分散风险、平衡现金流、获得新的利润增长点**
- 适用：现有市场吸引力下降，且企业在现有领域**没有**良好信誉和优势
- 不追求利用原有产品技术和销售渠道的共性

## 🌍 生活实例（小米）
- **相关多元化**：小米从手机→智能电视/路由器/手环/笔记本 → 共享MIUI系统、小米账号、销售渠道、品牌口碑 → 都是智能硬件，技术和渠道高度相关
- **相关多元化**：小米生态链企业模式 → 投资紫米（充电宝）、华米（手环）→ 都是围绕"智能硬件+互联网服务"展开
- **非相关多元化**：小米造汽车（SU7）→ 手机和汽车在制造工艺、供应链上差别巨大 → 但也不是完全无关（共享品牌、智能生态概念），偏向相关多元化
- 严格非相关多元化的例子在小米较少，小米整体偏向相关多元化

## 🔗 考点
- ⭐⭐⭐ 高频！给案例判断是相关还是非相关
- 主观题常考：分析某企业多元化类型并说明依据
- 秒杀：看新旧产业是否有技术/市场/渠道的共用——有就是相关，没有就是非相关
- 坑点：相关≠相同，有"交集"就算相关
- 🧠 记准用词：看新旧产业有无技术/市场/渠道共用→有则相关，无则非相关
