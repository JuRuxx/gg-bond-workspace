# 001 总体战略三大类型（发展/稳定/收缩）

## 一句话说清
总体战略分三大类：**发展战略**（进攻，做大做强）、**稳定战略**（维持，守住阵地）、**收缩战略**（撤退，断臂求生）。

## 内涵

### 三大类型速览
```
                     ┌─────────────────────────┐
                     │     总体战略（公司层）     │
                     └───────────┬─────────────┘
                                 │
         ┌───────────────────────┼───────────────────────┐
         ▼                       ▼                       ▼
 ┌───────────────┐     ┌───────────────┐     ┌───────────────┐
 │   发展战略    │     │   稳定战略    │     │   收缩战略    │
 │  （进攻）     │     │  （维持）     │     │  （撤退）     │
 ├───────────────┤     ├───────────────┤     ├───────────────┤
 │ ①一体化战略   │     │ 保持现有      │     │ ①紧缩与集中  │
 │ ②密集型战略   │     │ 经营状况不变   │     │ ②转向战略    │
 │ ③多元化战略   │     │              │     │ ③放弃战略    │
 └───────────────┘     └───────────────┘     └───────────────┘
```

### 发展战略（⭐重点）
细分为3类：**一体化、密集型、多元化**

### 稳定战略
- 内外部环境变化不大时采用
- 不改变使命目标，集中资源维持

### 收缩战略
- 内外部环境都不利时采用
- 细分为3种方式：紧缩与集中、转向、放弃

### 三大战略的实施途径
| 类型 | 途径 | 说明 |
|:---|:---|:---|
| 发展战略 | 外部发展（并购）、内部发展（新建）、战略联盟 | 三种途径均可 |
| 稳定战略 | 内部维持 | 不扩张也不收缩 |
| 收缩战略 | 紧缩/转向/放弃 | 减负求生 |

## 🌍 生活实例（小米）
- **发展**：小米从手机→IoT→汽车，一体化+多元化
- **稳定**：小米手机在国内市场份额稳定时，保持基本盘
- **收缩**：小米砍掉一些不赚钱的生态链产品

## 🔗 考点
- ⭐⭐⭐ 高频！给案例判断属于三大总体战略中的哪一种
- 秒杀：看到扩张/并购/新产品→发展战略；看到保持/不变→稳定战略；看到退出/裁员/出售→收缩战略
