# 021 变革受抵制的原因和克服策略

## 抵制原因
- ①**生理变化**：不适应新环境
- ②**环境变化**：办公地点/环境变化
- ③**心理变化**：未知恐惧、习惯被打破

## 克服策略
| 策略 | 说明 |
|:---|:---|
| ①对话沟通 | 讲清为什么变 |
| ②参与管理 | 让大家参与决策 |
| ③鼓励支持 | 培训+心理辅导 |
| ④加速变革 | 快刀斩乱麻 |
| ⑤谈判/奖励 | 给好处交换支持 |
| ⑥操纵/收买 | 非常手段 |
| ⑦强制 | 最后手段 |

## 🔗 考点
- ⭐⭐⭐ 主观题常考！给案例→分析谁抵制、为什么、怎么克服
- 🧠 记框架：抵制三原因（生理/环境/心理）+克服七策略（对话沟通→参与管理→鼓励支持→加速变革→谈判奖励→操纵收买→强制）

## 🌍 生活实例
公司上ERP系统：老员工说"学不会"→开培训班（鼓励支持）
