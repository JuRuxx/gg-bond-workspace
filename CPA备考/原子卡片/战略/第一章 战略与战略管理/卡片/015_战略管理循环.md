# 015 战略管理循环过程（图1-3）

```
     ┌─────────────┐
     │  战略分析    │←我们在哪
     └──────┬──────┘
            │
     ┌──────▼──────┐
     │  战略选择    │←我们去哪
     └──────┬──────┘
            │
     ┌──────▼──────┐
     │  战略实施    │←怎么去
     └──────┬──────┘
            │
        ┌───┴───┐
        │反馈调整│←不断循环
        └───────┘
```

## 🌍 生活实例
就像导航：查路况（分析）→选路线（选择）→开车走（实施）→堵车了换路（反馈调整）
