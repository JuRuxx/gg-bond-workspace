# 四章故事汇

> 猪猪侠编的CPA《战略》四章连续剧 🐷
> 
> 主人公：雷小米（小米公司CEO，人见人爱花见花开）
> 配角：阿米（雷小米的CPO）、米婶（CFO）、金老师（外部顾问）
> 
> **一个故事串一章考点，四章共一部连续剧！**

## 第一章 · 雷小米的"三问"

### 🔑 考点覆盖
- 传统vs现代概念 | 使命/目标 | 战略三层次 | 战略管理三要素（分析→选择→实施）| 变革类型 | 变革抵制与克服

---

雷小米坐在小米科技园13楼的办公室里，窗外的北京天空湛蓝得像MIUI的默认壁纸。

他面前摆着一行字——"让每个人都能享受科技的乐趣"。

这是他十年前在银谷大厦写下的。现在小米已经做到了全球前三的手机品牌，但他总觉得哪里不太对劲。

"阿米，"雷小米叫住从门口路过的CPO，"我有个问题。"

阿米折回来："啥问题？"

"咱们公司到底在做什么？"

阿米愣了一下："做手机啊。"

"不对，"雷小米摇头，"手机只是产品。我是说，咱们的*使命*是什么？"

"……这不就写墙上了吗？"

"那你跟我说说，使命和目标的区别。"

阿米站住想了想："使命≈方向，目标≈里程表？"

"对！"雷小米一拍桌子，"使命是你为什么出发，目标是你要走到哪里。咱们使命是'让人享受科技乐趣'，目标是'今年全球出货量前二'。"

雷小米在白板上画了一个大大的三叉树：

```
         ┌── 总体战略（公司层）—— 现在要不要造车？
         │
  战略 ──┼── 业务单位战略（竞争层）—— 手机怎么赢华为？
         │
         └── 职能战略（运营层）—— 研发怎么招人？
```

"这就是战略的三个层次，"雷小米用马克笔敲了敲板子，"**看主语秒杀**——'集团'是总体，'业务线'是竞争，'部门'是职能。"

阿米竖起大拇指："那咱们下一步怎么走？"

"按**战略管理三要素**走：分析→选择→实施。"雷小米在板上画了个箭头循环，"先搞清楚环境，再选方向，最后落地执行。"

正说着，销售副总裁老王推门进来，脸色不太好看："雷总，印度市场出事了……我们被罚款了5.5亿。"

雷小米沉默了三秒。

"这不是危机，这是变革的契机。"他站起身，"我有三种方案：第一，交罚款接着干——这是**渐变式变革**；第二，退出印度市场——这是**革命性变革**；第三，把印度团队本地化，让印度人管印度市场——这是**产品和服务变革**。"

阿米插嘴："但销售团队估计会抵制——你看内部群已经炸了。"

"变革被抵制的原因有四个：**结构惯性、有限的变革点、习惯、对失去权力的恐惧**。"雷小米掰着手指头说，"克服的办法也很简单——教育和沟通、参与、促进和支持——先把销售VP拉进来做决策小组。"

老王眼睛亮了："所以把我拉进决策小组，我就不会抵制了？"

"你终于明白了。"雷小米笑着拍了拍他的肩膀。

> **考点秒杀**
> 
> 使命=方向，目标=里程表，看主语秒杀三层次
> 变革4类型+抵制4原因+克服6方法=第一章通关

## 第二章 · 知己知彼，雷小米的"五面镜子"

### 🔑 考点覆盖
- PEST分析 | 五力模型 | 产品生命周期 | 资源与能力 | 核心能力三问 | 价值链 | 波士顿矩阵 | SWOT

---

距离做出"造车"决定的那个深夜已经过去了一个月。

雷小米把阿米、米婶（CFO）和金老师（外部战略顾问）叫到了会议室。会议桌中间放着一面镜子——不，是一块巨大的白板，上面画着一个五角星。

"今天咱们做**战略分析**，"雷小米在白板上写下四个字母，"先看宏观环境——PEST。"

他一边写一边说：

```
P = Political（政策）：新能源补贴、双积分政策——利好！
E = Economic（经济）：消费升级，但2024年经济放缓——双刃剑
S = Social（社会）：Z世代接受度85%，90后愿意为智能车买单——利好！
T = Technological（技术）：电池技术成熟、AI座舱——机会！
```

"所以宏观上，造这事儿能搞。"米婶点头，"但产业环境呢？"

雷小米画了一个"田"字格，中间标着小米：

```
          潜在进入者（蔚小理+华为≈最大威胁）
              ↓ 进入壁垒：资金、资质、技术
供应商（宁德时代）→ 小米 → 购买者（年轻人≠商务人士）
              ↑ 替代品威胁：公共交通≈低威胁
          产业内竞争（极度激烈：比亚迪≈每个价位都有产品）
```

"**五力模型**一拉，每个力都是中到高威胁——除了替代品。汽车这产业不是人待的。"雷小米感叹。

阿米举手："那咱们什么时候入场比较合适？"

金老师扶了扶眼镜："看**产品生命周期**就知道了。新能源汽车现在在**成长期**——增速快、竞争者涌入、技术还在迭代。成长期入场的策略是——砸品牌、建渠道、差异化。"

"好，宏观和产业都分析了。再看看自己。"雷小米翻了一页白板，"**内部环境分析**，先从资源开始。"

```
有形资源：工厂、设备、产线 ✅
无形资源：小米品牌、MIUI用户6亿 ⭐⭐⭐ 最强武器
人力资源：雷军+全明星创始团队 ✅
```

"资源有了，能不能创造竞争优势？要看四条标准——**稀缺性、难以模仿性、组织性、价值性**。"雷小米说，"小米的品牌辨识度，华为想copy都copy不走。"

"那**核心能力**是什么？"阿米问。

"用**三问法**：①它能让我们进入多个市场吗？——可以！手机→IoT→汽车都行。②顾客能明显感受到价值吗？——必须的！性价比是刻在骨子里的。③竞争对手难模仿吗？——超级难，因为这是十年的积累。"

雷小米在白板上画出**价值链**：

```
基本活动：研发（MIUI+自动驾驶）→ 供应链（全球采购）→ 生产（比亚迪代工）→ 
          营销（雷式发布会）→ 服务（小米之家售后）
支持活动：采购（宁德电池≈议价能力）→ 技术（徕卡联合研发）→ 
          人力（挖了吉利整个团队）→ 基础设施（雷布斯本人）
```

"**价值链**告诉我们——挣钱的重点在基本活动的营销和生产，支持活动的技术和人力是护城河。"

最后，雷小米画了四个圈——**波士顿矩阵**：

```
明星⭐（手机）：高增长+高份额 → 继续投资
金牛🐮（IoT）：低增长+高份额 → 挤奶拿钱
问号❓（汽车）：高增长+低份额 → 赌一把
瘦狗🐕（旧生态链）：低增长+低份额 → 收缩
```

"看明白了吗？手机≈金牛出钱，汽车≈问号烧钱，IoT≈明星中间站。"

米婶恍然大悟："所以咱们用手机赚的钱砸汽车？"

"Bingo！"雷小米转向白板最后一个方框——**SWOT分析**：

```
S（优势）：品牌、生态链、MIUI、雷军本人
W（劣势）：没有造车经验、起步晚、缺核心技术
O（机会）：新能源政策、Z世代、智能化趋势
T（威胁）：华为下场、比亚迪碾压、价格战

SO战略：用品牌优势+政策红利推高端车型（小米SU7）
WO战略：通过人才引进弥补技术短板（挖团队）
ST战略：用生态链构建差异化（人车家全生态）
WT战略：回避红海市场，聚焦差异化产品（轿跑路线）
```

金老师鼓掌："这四个象限一出，**战略方向已经定了**。"

"第二章的核心，就六个字——知己知彼，然后做选择。"雷小米总结道。

> **考点秒杀**
> 
> PEST看外部，五力看产业，生命周期看时机
> 价值链看哪赚钱，波士顿看钱怎么花
> 三问法辩核心能力，SWOT综合出方向

## 第三章 · 兵分三路

### 🔑 考点覆盖
- 总体战略（发展/稳定/收缩）| 一体化 | 密集型（安索夫矩阵）| 多元化 | 并购/联盟 | 成本领先/差异化/集中化 | 蓝海战略 | 职能战略（营销/研发/财务）| 战略钟 | 国际化经营

---

战略分析做完后，雷小米在内部发了三封邮件，直接决定了小米未来三年的走向。

### 第一封邮件：**总体战略——公司的"三叉戟"**

"从今天起，小米的**总体战略**有三条线——"

**第一条：发展！**
"手机业务已经到天花板了？那就买。我们**纵向一体化**——投资芯片设计公司（后向一体化）、建设小米之家直营渠道（前向一体化）。"

然后雷小米打开一张四象限图——**安索夫矩阵**：

```
           现有产品         新产品
            ↓               ↓
现有市场  ⚡市场渗透 ⚡      ⚡产品开发 ⚡
          （砍华为份额）    （小米汽车）
            
新市场    ⚡市场开发 ⚡      ⚡多元化 ⚡
          （进拉美非洲）    （造车≈相关多元化）
```

"手机老用户升级→**市场渗透**；进拉美→**市场开发**；造车→**相关多元化**（同一个人车家战略）。"雷小米敲着桌子，"每一格都要赢。"

**第二条：稳定！**
"IoT业务在高位运行，不用猛进也不用退——**稳定战略**，继续赚现金流。"

**第三条：收缩！**
"旧生态链里那些做充电宝、手环的老产品线——**收缩战略**。卖掉一些，停掉一些，让资源集中到汽车上。"

说到发展途径，雷小米又画了一个二元对比表：

```
并购（买买买）     vs    战略联盟（合作共赢）
——快速拿资源     ——低成本、灵活
——整合风险大     ——信任问题
——砸钱就行       ——需要人缘

小米的选择：缺技术就并购（买自动驾驶团队）
           要生态就用联盟（徕卡合作）
           要稳就内部发展（MIUI自研）
```

### 第二封邮件：**业务单位战略——怎么打赢？**

雷小米在内部大会上展示了三张幻灯片：

**第一张：成本领先**
"手机业务——**成本领先战略**。我们要把每一分钱花在刀刃上：线上直销省渠道费、黑灯工厂省人工费、供应链管理降采购费。五个字——**总成本最低**。"

**第二张：差异化**
"汽车业务——**差异化战略**。SU7必须不一样：人车家互联、生态链整合、雷军站台。消费者要的是'这车只有小米能做'的体验。"

**第三张：集中化**
"针对高端发烧友，推出MIX专线——**集中化战略**。就服务那一小撮人，死磕品质。"

阿米插嘴："那咱们在**战略钟**上站哪个位置？"

雷小米在白板上画了一个"钟面"：

```
      差  值  低  价          溢  价
        12           3
    高值中价 ⬆️     ⬆️ 高值高价
        9           6
    低值低价    低值中价/高价（❌ 不选）
```

"小米手机站**高值中价**——使命驱动的性价比。汽车站**高值高价**——高端定位，不能便宜了。"

然后雷小米翻了一个新的PPT——**蓝海战略**。

"手机市场是红海，红得发紫。但汽车市场，我们可以创造蓝海。"

他在白板上写下**蓝海六原则**：
```
重建市场边界（手机用户→汽车用户）
注重全局而非数字（看用户价值，不看成本）
超越现有需求（造还没人想到的功能）
遵循合理战略顺序（品牌→功能→定价→成本）
克服组织障碍（跨部门协作，打破部门墙）
把执行纳入战略（不只说要做，直接开做）
```

### 第三封邮件：**职能战略——落地！**

营销团队收到一条微信："3月28日开发布会，卖了5万台。你们搞定PR。"

研发团队的压力最大——MIUI Auto要在明年上线。

雷小米定的研发战略很明确："不做全栈自研，**自主+合作**——核心的电池管理自研，非核心的找第三方。"

财务上，米婶画了一个**财务战略矩阵**：

```
         高ROIC（回报高）          低ROIC（回报低）
          ↑                        ↑
高增长  ⚡ 创造价值现金剩余 ⚡     ⚡ 创造价值现金短缺 ⚡
       （手机=多给股东分红）     （汽车=借钱融资）
低增长  ⚡ 减损价值现金剩余 ⚡     ⚡ 减损价值现金短缺 ⚡
       （IoT=回购股票）          （淘汰产品=剥离）
```

"手机在左上象限——**创造价值+现金剩余**，应该加大分红。汽车在右上象限——**创造价值+现金短缺**，应该融资。一出一进，正好平衡。"

关于**经营风险与财务风险搭配**，米婶拍板了一句话：

"汽车项目风险高，所以财务上要保守——少借钱。手机业务风险低，可以多借钱扩张。"

### 最终章 · 出海！

印度被罚之后，雷小米没灰心。他把目光投向了欧洲。

"国际化不是你想去就能去——得先明白**为什么要出海**。"

他在白板上写了三种理论：

```
国际生产要素组合理论：小米的技术+中国的制造+欧洲的品牌溢价=利润
发展中国家国际化动因（4个）：① 寻求市场 → ② 寻求效率 → 
③ 寻求资源 → ④ 寻找现成资产（欧洲的品牌！）
```

"所以欧洲不能慢慢来——直接**并购**。买下诺基亚的一些专利，买下一个小众欧洲品牌——**跨国并购+合资**双管齐下。"

至于国际化战略类型，雷小米把全球四个象限画了出来：

```
              低全球协调             高全球协调
               ↓                       ↓
高本地响应  ⚡ 多国本土化 ⚡     ⚡ 跨国战略 ⚡
           （欧洲版本手机）    （MIUI全球统一+本地化）
低本地响应  ⚡ 国际战略 ⚡     ⚡ 全球化 ⚡
           （只出口，不改）    （全球一个版本）
```

"小米走的是**跨国战略**——全球统一的MIUI系统+本地化的功能。既有规模效应，又接地气。"

最后，阿米问了一句："在印度我们该怎么做——小米已经在那儿被针对了。"

雷小米沉默了一会儿，在白板上写下了**本土企业四种战略选择**：

```
           全球竞争压力高             全球竞争压力低
             ↓                         ↓
本土优势高  ⚡ 躲闪者 ⚡          ⚡ 防御者 ⚡
          （华为≈被排除）       （中国奶茶≈不出海）
本土优势低  ⚡ 抗衡者 ⚡          ⚡ 扩张者 ⚡
          （小米≈全球硬刚）       （小米进非洲≈蓝海）
```

"在印度，我们是**抗衡者**——全球竞争压力大，但印度是我们的基本盘。不能退，要硬刚——法律战、公关战、本地化。"

> **考点秒杀**
> 
> 总战略三叉戟：发展/稳定/收缩
> 安索夫四格：渗透/开发/产品/多元
> 成本领先怎么省，差异化怎么不同，蓝海怎么创造
> 财务矩阵四象限，国际化四类型，本土四选择——记住"四四"大法！

## 第四章 · 怎么干？——从一张组织图开始的战争

### 🔑 考点覆盖
- 组织结构（集/分权、高长/扁平、8种结构）| 结构追随战略 | 企业文化4种 | 文化适应性矩阵 | 战略控制（BSC/预算/失效）| 利益相关者 | 公司治理（董事/委员会/两大问题/监管）| 信息技术与价值链

---

清晨六点，雷小米在小米科技园跑步。

他的手机响了——是联合创始人林斌的电话。

"出事了。汽车事业部跟手机事业部因为芯片分配打起来了。两边都说是你的重点，互不相让。"

雷小米停下脚步。他知道，这不是偶然——这是**组织结构**不匹配战略的结果。

### —— 第一节：组织设计 ——

回到办公室，雷小米画了一张小米十年的组织进化图：

```
2010：创业型（雷军直管30人）← 一个人说了算
2012：职能型（按研发/营销/生产分）← 增长需要分工
2015：事业部制（手机/IoT/生态链）← 多元化战略
2018：M型企业（多事业部并行）← 集团化
2020：SBU（生态链企业各立山头）← 200+企业各自为战
2024：矩阵制（产品线×职能线交叉）← 汽车+手机要协同
```

"从小米的历史可以看到——**结构永远追随战略**。每一次战略转型，都倒逼组织重构。"雷小米感叹。

"那现在的问题是什么？"阿米问。

"小米现在是高长型，层级太多。汽车一个审批要走7层——**扁平化**才能快。但扁平意味着管理幅度大——一个VP管20个产品经理，信息可能会失真。"

他画了一个对比表：

```
高长型（现在的小米）：层级多、决策慢、控制严
扁平型（理想的小米）：层级少、决策快、授权大
```

"所以我们要往扁平化走。但先明确一件事——**集权还是分权**？"

"汽车业务：刚起步需要集中资源—**集权**。手机业务：稳定运行，放权给事业部—**分权**。"

阿米举手："那组织结构用哪种？"

雷小米翻了一页，把**8种横向结构**画成一个表：

| 结构 | 适用场景 | 小米案例 |
|:----|:--------|:--------|
| 创业型 | 初创期 | 2010年雷军+7人 | 
| 职能型 | 单一产品 | 2012年手机 |
| 事业部制 | 多产品线 | 手机/IoT/汽车三部 |
| M型 | 超大规模 | 集团多事业部 |
| SBU | 生态型 | 小米生态链200+企业 |
| 矩阵制 | 高科技多项目 | 汽车+手机协同 |
| H型 | 控股型 | 投资生态链企业 |
| 国际化 | 全球化 | 印度/欧洲/东南亚 |

"矩阵制最适合现在——汽车和手机之间要有双线汇报，项目负责人听产品线经理指挥，但也要向业务线经理汇报。"

"那协调机制呢？"阿米问。

"六种，从简到繁——①直接监督、②流程化、③标准化、④成果考核、⑤跨部门会议、⑥矩阵制。我们现在停在⑤，要迈到⑥。"

### —— 第二节：企业文化 ——

组织定了，但"人"的问题没解决。

雷小米发现，手机事业部的人私下里把汽车事业部称为"那群败家子"。汽车事业部的人回称"那群老古董"。

"这是**文化冲突**，"雷小米揉了揉太阳穴，"查尔斯·汉迪四种文化类型都占全了——"

```
权力导向型：雷军自己拍板造车，谁都不敢说不
角色导向型：手机业务按规章办事，流程180天
任务导向型：汽车团队像打了鸡血，不要流程要结果
人员导向型：MIUI工程师自由度高="参与感"文化
```

本质问题：手机和汽车的文化天生矛盾。

"咱们做个**文化适应性矩阵**。"雷小米画了一个2×2表：

```
              战略变化大                   战略变化小
               ↓                            ↓
文化适应力强  ⚡ 加强协作 ⚡          ⚡ 以企业为主 ⚡
             （手机+汽车联合团队 = 紧密协作）
文化适应力弱  ⚡ 重新制定战略 ⚡     ⚡ 根据文化管理 ⚡
             （汽车要独立一套体系 = 独立管理）
```

"汽车文化适应力弱，战略变化大——最好的方法是**重新制定战略**：不要硬融，让汽车团队自成一体。"

### —— 第三节：战略控制 ——

方案做出来了。但这还不够——得有人盯着执行。

"阿米，你来做小米的**首席控制官**。"

"我的职责是？"

"确保战略不跑偏。"雷小米在白板上写了三件事：

**第一件事：分清战略控制 vs 预算控制**

```
战略控制 vs 预算控制（表4-16）：
时间：长期 vs 短期
方法：定性+定量 vs 纯定量
重点：方向对不对 vs 钱花没花
纠正：调整战略 vs 调预算
```

"举个例子——我宣布要造车，这是**战略控制**；每季度看研发花了多少钱，这是**预算控制**。两个都要看，不能只看钱。"

**第二件事：平衡计分卡（BSC）**

雷小米画了一个平衡计分卡的四角形：

```
财务角度：汽车业务3年盈亏平衡 ✅ 目标：12月售5万台 ✅
客户角度：NPS>75分 ✅ 目标：首批用户复购率>60% ✅
内部流程：生产良率>95% ✅ 目标：交付周期<45天 ✅
学习成长：工程师留存率>85% ✅ 目标：自动驾驶专利数100+ ✅
```

"四个维度，每个季度打一次分——全面评估，不看表面。这叫**平衡计分卡**，不是只看财务数字。"

**第三件事：认识战略失效**

"最后，我的警告——**战略失效有三个阶段**："
```
① 早期失效（刚发布时叫好不叫座≈SU7预售太火爆反而不像真的）
② 偶然失效（中期出现质量问题≈电池自燃事件）
③ 晚期失效（执行到最后没结果≈造车亏100亿）
```

"早期的，别慌——熬过去。晚期的，别硬撑——及时止损。中间的，全力以赴解决。"

阿米听完默默点头："所以你叫我做控制官，是因为……"

"因为我需要有人在我上头的时候，拉我下来看看预算表。"

### —— 第四节：权力与利益相关者 ——

小米要造车，消息一传出，各大利益相关者坐不住了。

雷小米在会议室里画了一张**利益相关者地图**：

```
内部利益相关者：
├── 🌟 股东（散户）："造车太烧钱，跌我的股价！"
├── 🌟 经理人（高管层）："造车才有未来！"
└── 🌟 员工："加班太多！股权激励什么时候兑现？"

外部利益相关者：
├── 🌟 债权人（银行）："小米现金流够不够？"
├── 🌟 供应商（宁德时代）："大单！今年业绩稳了！"
├── 🌟 政府："新能源指标给小米，拉动GDP！"
└── 🌟 社会公众（米粉）："SU7太帅了什么时候交付？！"
```

"每个人都想要不一样的东西——这就是**利益相关者的矛盾**。"

阿米问："那股东和经理人的矛盾怎么解决？"

雷小米写了四个模型：

```
鲍莫尔的"销售最大化"：经理人喜欢做大销售额（为了面子+奖金）
马里斯的"增长最大化"：经理人要增长（为了晋升）
威廉森的"管理权限"：经理人追求高薪+安全（不是股东利益）
彭罗斯的"最佳投资策略"：经理人谨慎投资（害怕失败）
```

"所以经理人≈保守、求稳、不想冒险。但造车明显是个冒险的事情……这个问题怎么解决？"

"**权力和职权的区别**来了——"雷小米敲了敲桌子：

```
权力：我有影响力，但不一定有位置（雷军在员工心中的号召力）
职权：我有位置，但不一定有影响力（新来的CMO说话没人听）
```

"在小米，我是创始人，既有权力又有职权。所以我宣布造车没人敢反对——但这不代表我永远是对的。权力的来源有五种：**法定权、强制权、奖赏权、专家权、参照权**。我靠的是专家权和参照权——你们信我，因为我在手机这事儿上从来没输过。"

"那冲突了怎么办？"阿米问。

雷小米画了一个坐标图——**对待矛盾与冲突的行为模式**：

```
       合作性高         合作性低
坚定性高  ⚡ 对抗 ⚡        ⚡ 协作 ⚡
        （我要赢你输）     （双赢方案）
坚定性低  ⚡ 折中 ⚡        ⚡ 规避 ⚡
        （一人让一步）     （以后再说）
                   和解（认输）
```

"手机和汽车抢芯片——最好的方案是**协作**：成立联合采购组，统一谈判，降本增效。如果有人死不同意——先**折中**，再慢慢**协作**。"

"造车这个变革属于哪一种？"

雷小米画了**战略变革四象限**：

```
            自主              强势
参与度广  ⚡ 协调型 ⚡     ⚡ 计划型 ⚡
         （全员投票决定）   （高层定方案）
参与度窄  ⚡ 接受型 ⚡     ⚡ 迫使型 ⚡
         （开会通知结果）  （不干就滚蛋）
```

"小米造车是**计划型**——高层定了方向，再层层下达。这不是独裁，是效率。变革不是请客吃饭。"

### —— 第五节：公司治理 ——

造车太烧钱了。小米需要融资。

但每一次融资，都意味着股权稀释。

"我们面临**两大公司治理问题**——"

雷小米在黑板上写下：

```
第一类：代理型公司治理问题（经理人vs股东）
——雷小米是不是乱花钱？造车该不该投？
——解法：董事会监督+股权激励

第二类：剥夺型公司治理问题（大股东vs小股东）
——雷小米+林斌≈控制投票权，小股东说话不算？
——解法：完善独立董事制度+信息披露
```

"小米有AB股制度——我的投票权是普通股东的10倍。这是保护创始人决策权的手段，但也带来了**剥夺型问题**。"

他开始介绍董事会结构，画了一张**三类董事**的图：

```
执行董事（全职干活的）：
├── 雷小米（CEO）
├── 林斌（总裁）
└── 刘德（组织部部长）

非执行董事（不干活，只决策）：
├── 李嘉诚基金代表
└── 高瓴资本代表

独立董事（纯局外人，监督）：
├── 某大学会计教授（审计委员会主席）
├── 某律所合伙人（薪酬委员会主席）
└── 某前车企CEO（战略委员会）
```

"**四个专门委员会**各司其职——"

```
① 审计委员会（财技监督）→ 确保小米不造假
② 薪酬委员会（发钱规则）→ 雷小米的年薪谁来定？
③ 提名委员会（选人标准）→ 新董事谁来当？
④ 战略委员会（未来方向）→ 要不要再投100亿造车？
```

"外部还有五层**政府监管体系**——"

```
① 法律监管（公司法+证券法≈港股规则）
② 行政监管（证监会≈罚款+警告）
③ 行业自律（手机协会等）
④ 社会监管（媒体+舆论+米粉监督）
⑤ 审计监管（四大会计所")
```

### —— 第六节：信息技术 ——

"最后一件事。"雷小米调出一张图，"**信息技术**——这是小米的秘密武器。"

"**信息技术对组织的影响**——"他列举了四个维度：

```
① 组织扁平化：飞书=TMD把所有层级打平，VP直接和一线沟通
② 新型组织（虚拟组织）：MIUI全球开发者社区=不在一个办公室也一起干活
③ BPR（流程再造）：SU7预订→生产→交付全线上，砍掉30%中间环节
④ 打破组织边界：IoT平台=Million级设备互联，小米+200家企业=1个网络
```

"最后——**IT对企业价值链的全面渗透**——"

雷小米打开一张表：

| 价值链环节 | 信息技术加持 |
|:---------|:-----------|
| 内部物流 | 智能仓储机器人（效率+300%） |

"信息技术不是IT部门的活——它是整个战略的骨架。"

阿米听完雷小米的讲解，默默说了一句："所以你有没有想过——这些考的都是什么？"

雷小米一笑："CPA《战略》第四章——战略实施。我刚才讲的每一句话，都是考点。"
