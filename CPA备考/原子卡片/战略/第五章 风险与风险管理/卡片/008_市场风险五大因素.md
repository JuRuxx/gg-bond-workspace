# 051_008 市场风险五大因素

## 一句话说清

价格/供需/金融/竞争/信用五大市场因素变化风险

## 内涵

**定义**：市场风险是指由于市场环境变化导致企业损失的风险。市场风险需要考虑5大因素（表5-4）：

| 序号 | 因素 | 具体内容 |
|:---:|:----|:--------|
| **1** | **产品或服务的价格及供需变化** | 价格波动、供需失衡导致收入不确定性 |
| **2** | **物资供应的充足性、稳定性和价格变化** | 原材料、能源、配件等的供应风险和价格波动 |
| **3** | **税收政策和利率、汇率、股票价格指数的变化** | 政策变动、利率汇率波动、股市震荡带来的财务影响 |
| | — 税收风险 | 税收政策变化使企业税后利润发生变化 |
| | — 利率风险 | 利率提高或降低产生预期之外损失 |
| | — 汇率风险（货币风险） | 汇率变动导致货币价值变动 |
| | — 股票价格风险 | 股价波动影响企业融资和投资者 |
| **4** | **潜在进入者、竞争者与替代品的竞争** | 市场竞争格局变化、新玩家入场、替代品威胁 |
| **5** | **主要客户、供应商的信用风险** | 客户违约、供应商断供造成的损失风险 |

**⚠️ 易混提示**：流动性风险属于财务风险，不是市场风险！选择题常考辨析。

## 🌍 实例（小米）

小米面临的市场风险全谱系：
- **价格及供需变化**：中国智能手机市场从增量转为存量竞争，2022年手机出货量下降导致小米手机收入承压
- **物资供应变化**：2021年全球芯片短缺，小米被高通芯片供应不足困扰，部分机型被迫延迟发货
- **金融因素**：①汇率风险——小米印度业务赚取卢比，人民币升值导致汇兑损失；②利率——融资成本随利率上升而增加
- **竞争格局**：红米品牌打价格战面对荣耀、realme等竞品的激烈竞争；华为退场后小米未充分承接高端市场份额，OPPO/vivo持续挤压
- **信用风险**：小米生态链企业销量若下滑，可能出现货款回收困难
