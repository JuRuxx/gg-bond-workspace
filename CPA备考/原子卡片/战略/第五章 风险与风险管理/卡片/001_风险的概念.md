# 051_001 风险的概念

## 一句话说清

影响企业实现战略目标的不确定性（含正负两面）

## 内涵

**定义**（2001年国际内部审计师协会）：风险是指那些影响企业实现其战略目标的不确定性，此不确定性既可能给企业带来机会，也可能给企业带来损失。

**四大理解要点：**
1. **与企业战略相关**：风险与企业战略目标紧密关联，战略目标不同，面临的风险也不同
2. **风险是一系列可能结果**：不是单一的最可能结果，而是包含众多不确定性的"范围"
3. **客观性与主观性并存**：风险本身客观存在，但特定情况可由人的主观判断选择
4. **风险与机遇并存**：负面的叫威胁，正面的叫机会，企业应平衡风险与机遇

**知识拓展**——风险观念的演变：
- 前现代社会：迷信、罪恶 → 责备、被迫接受 → 补偿、惩罚
- 现代社会：可预测、可度量的负面因素 → 避免、保护 → 财务赔偿
- 现代市场经济：可管理可操纵的机会 → 接受专业建议 → 系统性改善

## 🌍 实例（小米）

小米进军高端市场战略面临的风险正体现了"风险与机遇并存"的理念：
- **机遇方面**：冲击高端市场可提升品牌溢价和毛利率（小米11 Ultra等机型曾获市场认可）
- **威胁方面**：若高端化战略不成功，可能丢失原有性价比用户基本盘，导致市场份额下滑
- 雷军在2022年提出"高端化是小米成长的必由之路"，正是对风险机遇二重性的认知——明知有风险，但仍要"平衡风险可能导致的相反结果所带来的机遇"

## 🔗 考点（🧠颗粒度等级：⭐）

- **考点频率**：⭐ 低频
- **出题方式**：选择题（概念辨析）
- **记忆要求**：理解性记忆，不需逐字背诵，但要能辨析核心要点
- **考试技巧**：风险概念的核心是"不确定性"+"双向性"（机会+损失），选择题题干描述"风险只有负面影响"的选项必然是错的
