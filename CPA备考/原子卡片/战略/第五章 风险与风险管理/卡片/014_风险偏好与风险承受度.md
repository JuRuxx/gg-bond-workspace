# 051_014 风险偏好与风险承受度

## 一句话说清

企业愿承担什么风险+承担多少的量化边界

## 内涵

**两个核心概念**：
- **风险偏好**：企业希望承受的风险范围，包括公司希望**承担什么风险**和**承担多少风险**（定性+定量的表述）
- **风险承受度**：企业风险偏好的边界，可将其作为企业采取行动的**预警指标**

**风险承受度的划分**：
```
       风险偏好范围
    ◄──────────────────►
    │                    │
    │   □□□□□□□□□□□□□□□   │
    │   ↑风险承受度上限  │   ← 超过此线需要立即采取措施（预警线/红线）
    │                    │
    │   ◇◇◇◇◇◇◇◇◇◇   │
    │   ↑风险承受度下限  │   ← 低于此线表示过于保守（安全线/底线）
    │                    │
    └────────────────────┘
     ← 保守区 → ← 风险区 →
```

**⚠️ 名师点拨**：
- 风险偏好的意义在于研究**成本与收益的关系**
- 基于风险管理概念的二重性，为了实现企业价值最大化目标，必须确立好风险承受度
- 在**机遇和风险中寻求平衡点**——这是风险管理的核心

**可设置多个承受度指标**以显示不同警示级别（如：安全线、预警线、红线）

## 🌍 实例（小米）

小米的风险偏好与风险承受度：
- **风险偏好**：雷军以"敢为天下先"著称，小米的风险偏好属于**较高水平**——敢于进入竞争激烈的手机行业，敢于投资生态链数百家企业，敢于押上全部声誉造车
- **量化边界**：雷军公开表示"小米硬件综合净利润率不超过5%"，这既是商业承诺，也是一种风险承受度的上限——宁可牺牲硬件利润也不愿超过这个界限
- **预警机制**：小米造车业务设立了明确的资金投入上限——首期100亿美元、初期投入100亿元人民币，超过这个额度需要重新评估
- **平衡点**：小米手机以低毛利获取用户，通过互联网服务（广告、游戏）变现，这就是在"赚取利润vs维持口碑"之间寻找风险与收益的平衡点

## 🔗 考点（🧠颗粒度等级：⭐）

- **考点频率**：⭐ 低频
- **出题方式**：选择（概念辨析，风险偏好vs风险承受度）
- **记忆要求**：懂就行——能区分风险偏好（意愿范围）和风险承受度（量化边界）即可
- **考试技巧**：选择题可能考辨析——"企业愿意承担风险的种类和数量"→风险偏好；"风险偏好的量化边界/预警指标"→风险承受度。不复杂，理解即可
