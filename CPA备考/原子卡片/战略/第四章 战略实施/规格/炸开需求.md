# 第四章 炸开需求文档

## 任务概述

将第四章《战略实施》讲义炸开为原子卡片，按"炸两遍"流程执行（先骨架后血肉）。

## 目标

- 每节拆成若干张原子卡片，每卡6字段齐全
- 所有卡片共用宿主案例：小米
- 产出链接地图（考点频率+编排+跨章+实例+核心逻辑图）
- 验收标准：0张空白卡，每卡300~800字，格式统一

## 产出物

```
原子卡片/战略/第四章 战略实施/
├── 卡片/       ← 独立原子卡片文件（编号_概念名.md）
├── 地图/
│   └── 链接地图.md  ← 4维度链接图
└── 规格/
    └── 炸开规格.md  ← 填充规范
```
