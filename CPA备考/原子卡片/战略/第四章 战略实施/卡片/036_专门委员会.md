# 036 专门委员会

## 一句话说清
董事会下设四大委员会各司其职：审计看财务，薪酬管发钱，提名挑人才，战略看长远。

## 内涵
- **4个专门委员会（表4-25）**：
  1. **审计委员会**——监督财务报告、内控与风险管理、内部审计，审查重大关联交易，提议更换外部审计
     - 组成：全部非执行董事，独立董事占多数并任召集人，至少1人为会计专业人士
  2. **薪酬与考核委员会**——制定董事高管薪酬方案、考核履职情况、监督制度执行
     - 组成：全部非执行董事，独立董事占多数并任召集人
  3. **提名委员会**——研究董事/高管选择标准和程序，审查候选人资格，评估董事会构成
     - 组成：全部非执行董事，独立董事占多数并任召集人
  4. **战略委员会**——研究长期发展战略和重大投资决策，对战略调整提建议，跟踪战略执行
     - 组成：至少1名独立董事，可包含执行董事
- **设置意义**：委员会制度是董事会专业化的体现，把专业性强的治理职能交由独立人员把关，避免管理层操纵决策

## 🌍 实例（小米）
小米董事会下设审计委员会（由独立董事王舜德任主席，另两名独立董事李家杰等组成），负责审阅小米每季度的财务报告、监督关联交易（如澎湃芯片研发投入）、审查生态链企业交易是否公允。薪酬委员会（同样独立董事主导）负责制定雷军和林斌的薪酬方案——2018年雷军99亿股权激励就是薪酬委员会审议、独立董事同意的结果，引发了争议但程序合规。战略委员会参与讨论小米造车、小米SU7推出等重大战略决策。

## 🔗 考点
- 考法：选择题考每个委员会的职责匹配和组成要求（如"审计委员会至少几名独立董事"）；简答题写4个委员会的职责
- 频率：⭐⭐（中频）
- 关键词：审计委员会 / 薪酬委员会 / 提名委员会 / 战略委员会 / 独立董事占多数
- 易混提示：审计委员会≠内部审计部门——审计委员会是董事会下设的监督机构，内部审计是公司职能部门，前者监督后者。薪酬委员会考得最多的是"全部由非执行董事组成"
- 🧠 记框架：4委员会职责+组成要求：审计(财务监督)、薪酬(考核)、提名(选人)、战略(长远)
