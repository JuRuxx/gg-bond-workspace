# 027 预算与预算控制类型

## 一句话说清
预算不只"今年花多少钱"，它还能做计划、分资源、设责任、给激励。编制方法有增量/零基，执行方式有定期/滚动/作业/弹性等。

## 内涵
- **预算的9大目的**：强迫计划、交流思想、协调活动、资源分配、提供责任框架、授权、建控制系统、绩效评估、激励员工
- **预算控制**：将实际结果与预算持续对比的过程，聚焦资源有效利用和成本管理
- **主要预算编制/控制类型**：
  1. **增量预算**：以历史为基数，调整增量部分（存在即合理）
  2. **零基预算**：每年从零论证每笔支出（合理才存在）
  3. **定期预算**：以固定期间（如一年）为周期编制，年底重编
  4. **滚动预算**：每过一段时间往后延展一期，保持预算覆盖始终为固定长度
  5. **作业基础预算**：按作业活动驱动来分配资源，强调流程与资源之间的关系
  6. **弹性预算**：基于实际业务量调整预算标准，用于可比性分析
- **关键注意**：预算控制系统需要与其他绩效管理体系（如平衡计分卡）相辅相成，不能只盯成本

## 🌍 实例（小米）
小米手机业务使用**定期+增量**组合预算：每年Q4启动下一年度预算，各产品线基于上年出货量和费用基数按±15%调整。2017年后引入**滚动预算**——季度滚动的销售预测让供应链提前备货，避免2016年那样的缺货危机。互联网服务部门还用**弹性预算**——广告收入预算随MIUI月活用户数动态调整，用户涨了预算跟着弹性放大，而不是死卡年初数字。

## 🔗 考点
- 考法：选择题辨别不同类型预算的定义和特点；主观题可能要求指出案例用了哪种预算方式及其优缺点
- 频率：⭐⭐（中频，比增量/零基单独考的少，主要考分类识别）
- 关键词：定期预算 / 滚动预算 / 弹性预算 / 作业基础预算 / 资源分配
- 易混提示：滚动预算≠零基预算——滚动是时间维度上的刷新，零基是内容维度上的重构；弹性预算≠增量预算——弹性看业务量变动调整，增量看历史基数加码
- 🧠 记框架：6种预算类型(增量/零基/定期/滚动/作业/弹性)各自特征
