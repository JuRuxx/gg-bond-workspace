# 025 平衡计分卡

## 一句话说清
平衡计分卡把战略画成一张表——从财务、顾客、内部流程、创新与学习4个角度拆出可衡量的指标，让你的战略不再"飘在天上"。

## 内涵
- **提出者**：卡普兰、诺顿；**核心理念**：战略要转化为可衡量的具体指标
- **4角度（图4-25/表4-19）**：
  1. **财务角度**（股东怎么看我们）：股东回报率、现金流、主要顾客收益率、利润预期、销售增长率
  2. **顾客角度**（顾客怎么看我们）：交货时间、顾客满意度、市场份额、新客户开发率
  3. **内部流程角度**（我们必须在哪方面卓越）：与顾客相处时间、雇员人均收入、收益率、交货时间、工程进度完成率
  4. **创新与学习角度**（我们能否持续提升）：新产品占比、雇员调查、关键员工保留率、员工能力评估
- **因果关系链**：学习成长→内部流程→顾客满意→财务成功（"学徒内财"口诀）
- **5大平衡**：①财务与非财务 ②长期与短期 ③结果与动因 ④内外部利益相关者 ⑤领先与滞后
- **特点**：有助于战略管理、提高管理效率、强调团队合作、促进员工参与、减少信息量

## 🌍 实例（小米）
小米2020年后用平衡计分卡拆解"手机×AIoT"战略：财务角度考核手机毛利率（目标从8%提到12%），互联网服务收入占比；顾客角度考核MIUI月活用户数、NPS净推荐值、高端机型的用户满意度；内部流程角度考核每款新机从立项到量产的平均周期、供应链周转天数；创新与学习角度考核IoT新品类孵化数、工程师人均专利数、核心研发人员留存率。4个角度的数据每季度在总裁办会上对表，尤其是"用户满意度"和"手机毛利率"之间的因果关系——用户满意了才买高端，高端才拉高毛利。

## 🔗 考点
- 考法：主观题高频——给案例让补全4角度的指标；选择题混搭4角度的指标问属于哪个角度
- 频率：⭐⭐⭐（高频——几乎每年必考，案例题常客）
- 关键词：4角度 / 因果关系链 / 学徒内财 / 五大平衡 / 战略落地工具
- 易混提示：财务角度不是"全部"——传统只用财务指标，平衡计分卡的最大创新是非财务维度（尤其创新与学习）；别和024增量/零基预算混，一个是绩效考核工具，一个是预算编制方法
- 🧠 记框架：4角度(财务/顾客/内部流程/学习成长)+因果关系链"学徒内财"
