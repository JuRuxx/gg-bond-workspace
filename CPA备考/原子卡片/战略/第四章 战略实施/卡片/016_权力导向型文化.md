# 016 权力导向型文化

## 一句话说清
老板说了算的文化——权力集中于一人或少数高层，决策快但依赖个人能力，家族企业常见。

## 内涵
- **别称**：集权式文化、铁腕型家长文化
- **核心特点**：
  1. 权力集中于一人或少数高层，掌权人对下属有绝对控制权
  2. 高层管理者能力决定企业未来，成败系于一人
  3. 采用传统框架结构，决策速度快
  4. 忽视人的价值和一般福利，中层士气低、流失率高
- **优点**：决策速度快，反应迅速；决策正确时助力公司快速成长
- **缺点**：决策质量高度依赖经理人员能力，决策错误会带来灾难；容易专横、滥用权力；中层人员士气低、流失率高
- **适用范围**：家族式企业、初创企业

## 🌍 实例（小米）
小米早期团队规模小时，雷军对产品和战略方向有很强的控制力，重大决策多由核心创始人拍板，是权力导向型的缩影。后来公司壮大后转向更制度化的管理，但雷军在关键决策（如造车）上仍保留高度集权的风格，体现了权力导向文化在大企业中的余韵。
