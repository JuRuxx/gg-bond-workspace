# 022 战略控制与预算控制

## 一句话说清
战略控制管"方向对不对"，预算控制管"钱花没花超"——两者在时间跨度、方法、重点和纠偏时点上截然不同。

## 内涵
- **定义**：战略控制是监督战略实施进程、及时纠正偏差、确保战略目标实现的手段；预算控制是实际结果与预算持续比较的监控过程。
- **4维差异（表4-16）**：
  1. **控制期间**：战略控制几年到十几年以上；预算控制通常一年以下
  2. **控制方法**：战略控制定性与定量并用；预算控制主要采用定量
  3. **控制重点**：战略控制兼顾内部和外部；预算控制重点在内部
  4. **纠正行为时点**：战略控制持续监控、不断纠正；预算控制通常在预算期结束后才纠正
- **秒记口诀**：长期定性内外兼，不断纠正是战略；短期定量重内部，事后纠正属预算
- **两者关系**：预算控制是战略控制的重要工具之一，但战略控制远不止预算——还关注外部环境、竞争优势和长期价值创造

## 🌍 实例（小米）
小米2018年上市后，战略层面将"手机×AIoT"作为核心战略，战略控制持续监控IoT连接数、互联网服务收入占比等指标，随时调整生态链打法；而年度预算控制聚焦各业务线成本费用——手机部门年初定好研发预算和营销费，年底对比实际支出。两者时间跨度不同：IoT生态布局看3~5年，而手机季度促销预算只看当季。

## 🔗 考点
- 考法：选择题给出描述让判断是战略控制还是预算控制；或简答题写出4维差异
- 频率：⭐⭐（中频）
- 关键词：长期短期 / 定性定量 / 内外重点 / 持续纠偏vs事后纠偏
- 易混提示：容易和控制跨度更大的"公司治理"搞混——控制是执行层面的纠偏，治理是制度层面的制衡
- 🧠 记口诀："长定性内外不断纠=战略；短定量内事后纠=预算"
