# 022 通用矩阵（GE矩阵）

## 一句话说清
波士顿矩阵的升级版：不只是看增长率和份额，综合看"产业吸引力"和"竞争地位"。

## 基本原理
- **横轴**：竞争地位（强→弱）
- **纵轴**：产业吸引力（高→低）
- **9宫格**，每个格子对应不同战略：
  - 左上3格（强势产业）→ **增长/发展**
  - 对角线3格 → **维持/选择发展**
  - 右下3格（弱势产业）→ **停止/撤退**

## 指标
| 维度 | 指标举例 |
|:---|:---|
| **产业吸引力** | 增长率、市场规模、获利能力、竞争结构、技术、社会政治 |
| **竞争地位** | 相对市场占有率、产品差异化、生产能力、管理水平 |

## 局限
- 指标权重难定→容易偏差
- 分得太细→大公司没必要

## 波士顿 vs 通用
| 对比 | 波士顿（2×2） | 通用（3×3） |
|:---|:---|:---|
| 轴 | 2个指标 | 多个综合指标 |
| 格子 | 4个 | 9个 |
| 难度 | 简单 | 复杂 |
| 适用 | 产品组合分析 | 更精细的业务分析 |

- ⭐ 低频，了解概念+与波士顿对比即可
- 🧠 懂就行：知道GE矩阵是波士顿升级版(3×3九宫格)+了解基本对比即可
