# 011 产业内的战略群组

## 一句话说清
一个行业里不是所有人互相竞争——只有群组内的人才是近身肉搏的对手

## 定义
同一产业内采用**相同或相似战略**的企业组成的群体

## 战略群组分析的意义（5个）
| 意义 | 说明 |
|:---|:---|
| ①了解竞争状况 | 近处敌人（同组）vs远处敌人（他组） |
| ②了解移动障碍 | 从一个群组跳到另一个群组的障碍 |
| ③了解竞争焦点 | 同组内不同能力导致不同效果 |
| ④预测市场/发现机会 | 群组图能看出空白地带 |
| ⑤了解对手优劣势 | 同组对手的优劣势和战略方向 |

## 🌍 实例
手机行业：
- 高端群组：苹果、华为Mate → 竞争焦点是品牌+技术
- 中端群组：小米、OPPO、vivo → 竞争焦点是性价比+渠道
- 低端群组：红米、realme → 竞争焦点是低价+走量
小米想从"中端"跳到"高端"→遇到**移动障碍**（品牌认知）

## 🔗 考点
- ⭐⭐ 客观题：5点意义要能选出来，尤其"移动障碍"这个点
- 口诀：**竞争→移动→焦点→预测→对手**
- 🧠 记口诀：'竞争移动焦点预测对手'=战略群组5点意义
