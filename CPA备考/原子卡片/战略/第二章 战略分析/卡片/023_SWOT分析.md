# 023 SWOT分析

## 一句话说清
内外结合看自己：内部看优势和劣势，外部看机会和威胁。

## 四象限
```text
                 内部
              S(优势)  W(劣势)
           ┌────────┬────────┐
    O      │   SO    │   WO    │
    (机会)  │ 增长型  │ 扭转型  │
   外      │ (进攻)  │ (改进)  │
   部      ├────────┼────────┤
    T      │   ST    │   WT    │
    (威胁)  │ 多元化  │ 防御型  │
           │ (规避)  │ (收缩)  │
           └────────┴────────┘
```

| 组合 | 含义 | 策略 | 🌍实例 |
|:---|:---|:---|:---|
| **SO** | 优势+机会 | 利用优势抓机会 | 小米有技术+新能源机遇→造车 |
| **WO** | 劣势+机会 | 改进劣势抓机会 | 小米缺高端形象+市场高端化→推MIX系列 |
| **ST** | 优势+威胁 | 利用优势避威胁 | 小米有供应链管理+对手降价→成本优势 |
| **WT** | 劣势+威胁 | 减少劣势避威胁 | 小米海外市场弱+贸易摩擦→收缩海外低效业务 |

## 注意事项
- SWOT不是"列清单"，是要**交叉分析**
- SO是最理想的→优先选
- WT是最危险的→优先回避

## 🔗 考点
- ⭐⭐⭐ 高频！主观题+客观题都爱考
- 给一段材料→判断属于哪个象限→匹配策略
- 口诀：**SO进攻、WO改进、ST规避、WT收缩**
- 🧠 记框架：4象限(SO/WO/ST/WT)+各象限策略逻辑
