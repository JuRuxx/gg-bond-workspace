# 018 价值链：基本活动

## 一句话说清
产品从进公司到出公司到客户手里的整个链条中，直接创造价值的5个环节。

## 五种基本活动
```text
内部后勤 → 生产经营 → 外部后勤 → 市场营销 → 服务
  入库/存货    加工/组装    发货/配送    品牌/渠道    售后/维修
```

| 活动 | 内容 | 🌍实例（小米） |
|:---|:---|:---|
| **①内部后勤** | 原料入库、检验、盘存 | 芯片/Screen入库质检 |
| **②生产经营** | 加工、装配、检测、设备维护 | 手机生产组装 |
| **③外部后勤** | 成品入库、订单处理、配送 | 小米仓配到京东/天猫 |
| **④市场营销** | 广告、定价、渠道、品牌 | 小米发布会、米粉节 |
| **⑤服务** | 安装、维修、培训、零件供应 | 小米之家售后服务 |

## 🔗 考点
- ⭐⭐ 给一个活动→判断属于基本活动中的哪一种
- 记忆：**进→生→出→营→服**（进进出出，营生服务，哈哈）
- 🧠 记口诀：'进生出营服'=内部后勤→生产经营→外部后勤→市场营销→服务
