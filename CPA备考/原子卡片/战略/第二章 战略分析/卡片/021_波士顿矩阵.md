# 021 波士顿矩阵（BCG矩阵）

## 一句话说清
把钱投给最有前途的产品，从"现金牛"拿钱喂"明星"和"问题"，不管"瘦狗"。

## 四象限
```text
           高←市场增长率→低
         ┌────────┬────────┐
   高    │ ⭐明星 │ 💰现金牛 │
  相对   │ 高增长  │ 低增长  │
  市场   │ 高份额  │ 高份额  │
  占有  │ (多投)  │ (收割)  │
  率    ├────────┼────────┤
   低    │ ❓问题  │ 🐶瘦狗  │
         │ 高增长  │ 低增长  │
         │ 低份额  │ 低份额  │
         │ (发展/放弃)│ (收割/放弃)│
         └────────┴────────┘
```

| 象限 | 特征 | 战略 | 🌍实例 |
|:---|:---|:---|:---|
| **⭐明星** | 高增长+高份额 | 发展（多投钱做大） | 小米汽车初期 |
| **💰现金牛** | 低增长+高份额 | 保持/收割（拿利润） | 小米手机 |
| **❓问题** | 高增长+低份额 | 发展（有前途的）或放弃 | 小米某些新业务 |
| **🐶瘦狗** | 低增长+低份额 | 收割或放弃（别浪费钱） | 小米早期一些小产品 |

## 四种战略
- **发展**：投钱→尽早变明星（适用于问题中有潜力的）
- **保持**：维持现状（适用于大现金牛）
- **收割**：赚最后一波钱（适用于走下坡的现金牛/问题/瘦狗）
- **放弃**：剥离（适用于瘦狗和没前途的问题）

## 价值与局限
**价值**：最早的组合分析法、简单明了、帮企业集中资源、可推测对手
**局限**：增长率/占有率难确定、指标太粗、假设市场占有率=回报率、只考虑钱不看人

## 🔗 考点
- ⭐⭐⭐ 高频！给案例→判断属于哪个象限+对应战略
- 秒杀法：增长率高+份额高=明星；增长率低+份额高=现金牛
- 口诀：**明问金牛和瘦狗，发展保持收割弃**
- 🧠 记框架：4象限(明星/问题/现金牛/瘦狗)+特征+对应战略(发展/保持/收割/放弃)
