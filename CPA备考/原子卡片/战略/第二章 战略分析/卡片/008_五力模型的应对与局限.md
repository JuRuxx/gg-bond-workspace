# 008 五力模型：应对策略与局限性

## 一句话说清
五力分析完了怎么办？三种应对方式。但模型本身有6个方面的不足。

## 应对五力竞争的三种战略
- ①**自我定位**：用成本优势或差异优势"隔离"五力
- ②**集中战略**：选细分市场，不和所有人为敌
- ③**战略联盟**：建立长期合作关系

## 五力模型的6大局限
| 局限 | 说明 |
|:---|:---|
| ①静态分析 | 只反映一个时间点的格局 |
| ②不适用非营利 | 对非营利机构无用 |
| ③理想假设 | 假设能分析→就能定战略 |
| ④信息完备假设 | 假设你能了解全部行业信息 |
| ⑤低估合作关系 | 低估与供应/客户的长期合作 |
| ⑥考虑不全面 | 互补品等因素没考虑 |

## 🔗 考点
- ⭐ 低频，了解即可
- 🧠 懂就行：知道3种应对战略+6个局限概述，不背细节
