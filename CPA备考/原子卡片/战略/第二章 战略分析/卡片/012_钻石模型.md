# 012 钻石模型（波特国家竞争优势）

## 一句话说清
一个国家为什么能在某个产业上特别强？四个要素决定。

## 四个要素
```
            ┌─────────────┐
            │ 生产要素    │ ← 资源/人才/基础设施
            └──────┬──────┘
                   │
    ┌──────┬───────┼───────┬──────┐
    │      │       │       │      │
┌───┴───┐ ┌───┴───┐ 需求  │  ┌───┴───┐
│相关与 │ │企业战略│ 条件──┤ │政府  │
│支持性 │ │结构与 │       │ └───────┘
│产业   │ │同业竞争│       │
└───────┘ └───────┘       │ ┌───────┐
                           │ │机会  │
                           │ └───────┘
```

| 要素 | 含义 | 🌍实例（中国新能源车） |
|:---|:---|:---|
| **①生产要素** | 资源、人才、资本、基础设施 | 锂矿、工程师红利 |
| **②需求条件** | 国内市场需求规模和成长性 | 全球最大汽车市场 |
| **③相关与支持性产业** | 上下游产业链配套 | 宁德时代电池、充电桩网络 |
| **④企业战略与同业竞争** | 企业组织方式+国内竞争激烈度 | 比亚迪/蔚小理互相内卷→促创新 |

**两个变量**：政府（政策）、机会（技术突破）

## 🔗 考点
- ⭐⭐ 给一个因素→判断属于钻石模型哪个要素
- 注意区分波特五力和钻石模型的用途不同
- 🧠 记框架：4个要素(生产要素/需求条件/相关与支持产业/企业战略同业竞争)+2个变量(政府/机会)
