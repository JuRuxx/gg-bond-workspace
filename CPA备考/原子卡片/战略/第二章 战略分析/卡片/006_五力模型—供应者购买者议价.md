# 006 五力模型：供应者与购买者讨价还价的能力

## 一句话说清
谁更"离不开"谁，谁就占上风。只有你有、我不要也行→你厉害；你不卖我找别人→我厉害

## 供应者议价能力强的情况（买方因素角度）
| 买方因素 | 解释 |
|:---|:---|
| 集中程度 | 供应商少→他们说了算 |
| 产品差异化 | 只有他家有→他定价 |
| 资产专用性 | 设备只能生产他家零件→被绑定 |
| 纵向一体化 | 供应商自己也能做下游→"我自己开店也行" |
| 信息掌握 | 供应商知道你的底牌→压价更狠 |

## 购买者议价能力强的情况（镜像）
- 买方集中→大客户说了算
- 产品同质→换供应商零成本
- 买方后向整合→"我自己生产也行"

## 🔗 考点
- ⭐⭐ 案例题：给描述判断是供应者还是购买者议价能力强
- 🧠 记框架：供应者vs购买者各5个议价因素(集中度/差异化/专用性/一体化/信息)
