# CPA 刷题工具 — 规格说明书

> 2026-06-10 主人下达的刷题 HTML 单页工具规格
> 注：配图中的框线理解为功能模块的示意范围，具体 UI 由开发者自由发挥

---

## 一、项目概述

纯前端 HTML 单页应用，零依赖，双击 `index.html` 即可在浏览器打开使用。所有数据存储在浏览器 `localStorage` 中。

**目标用户**：CPA 备考者（陈绍禹）
**题库科目**：公司战略与风险管理（先做战略，后续可扩展到其他科目）
**开发语言**：HTML + CSS + JavaScript（纯前端，无后端）

---

## 二、功能模块

### 模块 1：首页（导航面板）

功能入口面板，四个核心入口：

| 入口 | 说明 |
|:-----|:------|
| **章节练习** | 按教材章节顺序刷题，不限时，选完即出解析 |
| **模拟考试** | 随机抽题，限时45分钟，交卷出分 |
| **错题本** | 自动记录做错的题，支持重做和移出 |
| **收藏题** | 手动收藏的题，可单独刷 |

> **配图说明**：配图中，"第一章 走进去" 对应 "章节练习" — 点进去自动进入当前章节的连续刷题模式；"收藏" 入口在答题页面的选项区旁。

---

### 模块 2：章节练习

**流程：**
1. 进入 → 显示章节选择题号列表（1~N 题）
2. 点击任一题号 → 显示题目 + 选项
3. 用户选择选项 → 点击"确认" → 立即显示 **对/错 + 解析**
4. 答错的题**自动**加入错题本
5. 可点击"上一题"/"下一题"切换

**交互细节：**
- 点击选项 -> 该选项高亮 -> 确认答案按钮可用 -> 按确认后显示正误与解析
- 答对的题可点击"收藏"按钮，加入收藏题
- "收藏"按钮位于选项区附近
- 支持"重新开始"按钮，重置当前章节的做题进度

> **配图说明**：配图展示做题状态、（局部错误）显示、题号栏、进度块、点击收藏等功能。选择项点击后高亮，按确认后显示正误与解析。

---

### 模块 3：模拟考试

**流程：**
1. 从当前科目的总题库**随机抽取 30 道题**
2. **倒计时 45 分钟**，时间到自动交卷
3. 全部答完或手动交卷 → 显示成绩报告

**成绩报告：**
- 总分：X/30
- 正确率：X%
- 错题回顾：逐题列出，显示正确答案和你的答案
- 按钮：**重做错题** → 跳转错题本

**交互细节：**
- 答题期间不能查看解析
- 交卷后才显示正误和解析
- 支持中途退出（但退出不计入成绩）

---

### 模块 4：错题本

**功能：**
- 按章节筛选错题
- 显示错题列表（题号 + 题干摘要）
- 点击任一错题 → 进入刷题模式（与章节练习相同）
- 重做正确的题 → 自动从错题本移除
- 支持"清空错题本"（需二次确认）

**配图说明**：配图展示了错题列表、"重做错题"按钮、按章节筛选等功能。

---

### 模块 5：收藏题

**功能：**
- 显示所有手动标记收藏的题
- 支持取消收藏
- 支持按章节筛选
- 点击题目进入刷题模式

---

### 模块 6：答题界面标准交互

| 元素 | 交互 |
|:-----|:------|
| 选项 | 单选点选一个，多选可点多个 |
| 确认答案 | 单选：选择一个选项后 -> 确认 -> 显示正误与解析 |
| 显示解析 | 确认后显示绿色（正确）或红色（错误）标记 + 解析文字 |
| 收藏按钮 | 任何时候可点击切换收藏状态 |
| 题号栏 | 当前章节的全部题号，答过的有颜色标记 |
| 进度条 | 当前章节：已答题数/总题数 |
| 返回 | 左上角返回按钮回到首页 |

> **配图说明**：配图展示了已完成题目的蓝色状态、错误状态（叉号背景）和正在做的题的高亮状态。

## 三、数据模型

### 题目数据结构

```javascript
{
  id: Number,            // 唯一编号
  chapter: String,       // 章节名称，如"第一章"
  section: String,       // 节名称，如"第一节"
  type: String,          // "单选题" 或 "多选题"
  difficulty: String,    // "易" / "中" / "难"
  question: String,      // 题干，支持 HTML
  options: [
    { label: "A", text: "选项内容" },
    { label: "B", text: "选项内容" },
    { label: "C", text: "选项内容" },
    { label: "D", text: "选项内容" }
  ],
  answer: String,        // 单选："A"；多选："AB"（字母拼接）
  analysis: String       // 解析文字，支持 HTML
}
```

### localStorage 数据结构

```javascript
{
  wrongQuestions: [id1, id2, ...],          // 错题本的题号数组
  favoriteQuestions: [id1, id3, ...],       // 收藏题的题号数组
  chapterProgress: {                        // 各章节做题进度
    "第一章": { answered: [1,3,5], wrong: [3] },
    "第二章": { answered: [10,12], wrong: [] }
  },
  examHistory: [                            // 模拟考试历史记录
    { date: "2026-06-10", score: 25, total: 30, wrongIds: [2,7] }
  ]
}
```

## 四、界面风格要求

- 配色：**蓝白基调**，清晰干净，适合长时间刷题
- 布局：左导航 + 右内容区（参考配图布局）
- 字体：系统默认字体，标题稍大
- 响应式：适配桌面浏览器即可，不做移动端优化
- 简洁为主，不炫技

> **配图说明**：配图展示主页、答题页、错题页的布局风格——蓝白主调，左侧导航菜单，右侧工作区。考试记录页展示已完成模拟考历史的得分率统计。

## 五、使用流程示意

```
打开 index.html
        │
        ▼
    ┌─────────┐
    │  首页    │ ← 四个功能入口
    └────┬────┘
         │
    ┌────┴────┐
    │ 选择功能 │
    └────┬────┘
         │
    ┌────┴─────────────────────────────────────┐
    │                                          │
    │  章节练习 → 选章 → 刷题(自动记错题本)     │
    │  模拟考试 → 随机30题 → 45分钟 → 出分     │
    │  错题本   → 选章 → 刷错题 → 做对移除     │
    │  收藏题   → 选章 → 刷收藏题 → 取消收藏   │
    │                                          │
    └──────────────────────────────────────────┘
```
