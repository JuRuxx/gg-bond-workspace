## 🧠 方法：什么是"炸开"学习法？

> 罗永浩做锤子手机时，把整本书拆成一条条知识点贴满墙。  
> 我们把讲义拆成原子卡片，再按四种维度重新链接。

### 流程

```
讲义（整篇的）
   ↓ 🧨 炸开
原子卡片（每张独立讲一个概念）
   ↓ 🔗 重连
考点频率 → 讲义编排 → 跨章链接 → 生活实例（小米）
```

### 每张卡片7字段

```markdown
## 一句话说清     ← 高压缩率，1秒get核心
## 内涵            ← 教材精华提炼，头条化
## 🌍 生活实例     ← 小米案例（宿主案例）
## 🔗 考点         ← 考法+频率+关键词秒杀+🧠颗粒度等级
## 🏗 讲义位置     ← 可回查
## 📚 章节联系     ← 跨卡/跨章链接
```

### 🧠 颗粒度法则（2026-06-12 主人定）

不同内容按不同深度记忆，**不要所有考点一个深度背书**。

| 等级 | 怎么记 | 适用内容 | 例子 |
|:----|:------|:--------|:----|
| **🧠 记准用词** | 逐字记住核心措辞（1~2句话） | 概念定义、一句话能说清的考点 | 使命≈方向vs目标≈里程表 |
| **🧠 记框架** | 记住模型结构+每一格内容 | 模型框架，可能的画图/结构题 | PEST=4字母、波士顿=2×2四象限 |
| **🧠 记口诀** | 多步骤压缩成3~5字口诀 | 流程/步骤类考点 | 关键风险指标6步→'找阈值预盯' |
| **🧠 懂就行** | 只理解本质，不背细节 | 低频考点、历史演变、大量枚举 | 知道变革是渐变积累→革命即可 |

**判断原则**：
- ⭐⭐⭐高频 → 优先选「记准用词」或「记框架」（要下真功夫）
- ⭐⭐中频 → 用「记口诀」或「记框架」压缩记忆
- ⭐低频 → 「懂就行」，不浪费脑缓存
- 背6个编号=浪费大脑缓存。大脑是用来做模式识别的，不是当PDF阅读器的。

### 炸两遍原则

- **一次炸不开的，炸两遍**：先骨架后血肉
- **炸骨架**：创建空壳卡片+空壳链接地图+填充规格
- **炸填肉**：派子Agent并行填充内容
- **验收兜底**：模糊老师最后过一遍
