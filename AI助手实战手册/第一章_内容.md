# 第一章：装好你的AI副手——Claude 桌面客户端 + CC-Switch 极简安装

---

> 本章目标：装好两个软件，然后跟AI对话一次。全程不需要动任何代码，只要会双击鼠标、会打字就行。

---

## 1.1 AI副手能帮你做什么（3分钟）

每天上班，你是不是也经常做这些事——

- **写周报**：对着空白文档憋半天，不知道上周干了啥
- **改格式**：复制了一段文字，字体字号全乱，一个个调
- **翻译**：收到一封英文邮件，看得头皮发麻
- **整理数据**：Excel 里乱七八糟的备注栏，要把所有日期和金额摘出来

这些事，AI 都能干。而且你不需要会写代码，**你只需要会打字**。

AI 就像一个刚入职的实习生——你把活说清楚，它就给你干好。干不好？再跟它说一遍就行。

所以接下来的 30 分钟，我们装好两个软件，然后你就能跟这个"实习生"聊上了。

---

## 1.2 两个工具体系：办公室 + 万能遥控器（3分钟）

只用一句话解释：你需要装 **两个软件**，一个当办公室，一个当接线员。

| 比喻 | 工具名称 | 干什么用 |
|------|----------|----------|
| 你的AI办公室 | **Claude 桌面客户端** | 你在这打字跟AI说话，AI在这里回你 |
| 万能遥控器 | **CC-Switch** | 帮AI联网、选脑子（用什么AI模型由它管） |

为什么不能只装一个？因为 Claude 客户端本身是跟国外的 AI 服务器通信的，在国内联网不太稳定。CC-Switch 的作用就是当"接线师傅"——你装好它，设置一下，它就能帮 Claude 客户端顺利连上国内的 AI 大脑（DeepSeek）。

你不需要知道技术细节，只要记住：**装好 CC-Switch，你的 AI 就能正常工作了。**

## 1.3 安装 Claude 桌面客户端（6分钟）

> ⚠️ **提前说**：安装 Claude 客户端需要一次性的科学上网环境（装完后续用CC-Switch就不需要了）。如果你手头没有，找老板要一下，装完就还给老板。
>
> Claude 桌面客户端就是你之后跟 AI 聊天的地方。先装好，**先不要登录**，等 CC-Switch 也装好再说。

### 1.3.1 Windows 安装

1. 打开你电脑上的浏览器（Chrome、Edge 都可以），在地址栏输入 `claude.ai/download` 然后**按回车**。

2. 页面会自动识别你的系统，显示一个**下载**按钮，**点击**它。

3. 下载完成后，找到文件（一般在"下载"文件夹里，叫 `Claude-Setup.exe`），**双击**它。

4. 安装界面弹出来后，一路**点击**"下一步"或"I Agree"，不用改任何设置。

5. 安装完成后，桌面上会出现 Claude 的图标。**先不要打开它**——我们先装 CC-Switch。

> 💡 **提示**：安装完后如果电脑提示重启，可以关掉所有软件重启一下。重启回来后，桌面的 Claude 图标还在。

### 1.3.2 Mac 安装

1. 打开 Safari 或 Chrome 浏览器，在地址栏输入 `claude.ai/download`，**按回车**。

2. 页面自动识别你的系统，**点击**下载按钮，会下载一个 `.dmg` 文件。

3. 下载完成后，**双击**那个 `.dmg` 文件，会弹出一个窗口。

4. 把 Claude 的图标**拖拽**到旁边的"Applications"文件夹图标上。

5. 首次打开时，Mac 可能会提示"无法验证开发者"。别慌：
   - **点击**屏幕左上角的苹果图标 → **系统设置**
   - **点**"隐私与安全性"
   - 往下翻，看到"仍要打开"按钮，**点击**它

6. 安装完成，桌面上会出现 Claude 图标。**先不要打开**，我们装了 CC-Switch 再说。

## 1.4 安装 CC-Switch（8分钟）

> CC-Switch 就是前面说的"万能遥控器"，它负责帮 Claude 客户端连上 AI 大脑。

### 1.4.1 下载和安装

1. 打开浏览器，输入 `github.com/farion1231/cc-switch/releases`，**按回车**。

2. 找到最新版本，看你的系统下载对应的安装包：
   - **Windows用户**：下载 `.msi` 安装包
   - **Mac用户**：下载 `.dmg` 安装包

3. 下载完成后：
   - **Windows**：**双击** `.msi` 文件，一路点"下一步"安装
   - **Mac**：**双击** `.dmg` 文件，把 CC-Switch **拖** 到 Applications

4. 第一次打开 CC-Switch，Mac 用户可能又遇到"无法验证开发者"提示——跟前面一样去"系统设置→隐私与安全性"里**点**"仍要打开"。

5. 打开后，你会看到一个主界面，顶部有 Claude Desktop、Claude Code 等标签页。

### 1.4.2 注册 DeepSeek + 获取 API Key

> API Key（中文叫"API密钥"），相当于你的 AI 账号密码。AI 不是免费的，但 DeepSeek 给了每个新用户免费额度，够用很久。

1. 打开浏览器，输入 `platform.deepseek.com`，**按回车**。

2. **点击**"注册"，用手机号或邮箱注册一个账号。
   > 💡 **提示**：国内手机号就可以注册，很方便。

3. 注册成功、登录进去后，在左侧菜单找到**API Keys**，**点击**它。

4. **点击**"创建 API Key"按钮，给 Key 取个名字（比如"我的AI助手"），然后**点击**确认。

5. 页面上会显示一串看起来很乱的字母和数字——这串东西很关键。**点击"复制"按钮，把 Key 复制到剪贴板。**

> ⚠️ **非常重要**：这个对话框关掉之后，你就再也看不到这串 Key 了。**复制完后，先粘贴到一个记事本里临时存着**，后面马上要用。

### 1.4.3 在 CC-Switch 中添加 Provider

> Provider（可以理解成"AI大脑的来源"），这里我们要告诉 CC-Switch："DeepSeek 的大脑在哪，怎么连上它。"

1. 回到 CC-Switch 界面，找一下有没有**"+"号**按钮（通常在左侧列表）；**点击**它，选择"添加 Provider"。

2. 在弹出的窗口里，找到 **DeepSeek**（一般在预设列表里），**选中**它。

3. 你会看到一个"API Key"输入框，**粘贴**刚才复制的那串 Key。

4. 在"模型"下拉框里**选择** DeepSeek 对应的模型。

5. 填写好之后，**点击**保存或确认。

### 1.4.4 测试连接

1. 在刚才添加的 DeepSeek Provider 旁边，有一个**"健康检查"按钮**（也可能叫"测试连接"），**点击**它。

2. 等几秒钟，如果出现绿色的"连接成功"或"健康检查通过"，说明一切正常！

3. 如果显示"连接失败"：
   - 检查你复制的 API Key 有没有多复制或漏掉字符
   - 检查你的电脑是不是连着网络
   - 重新复制一次 Key 再试

## 1.5 连接 Claude 客户端与 CC-Switch（3分钟）

现在两个软件都装好了，连接它们只需要一步。

1. 在 CC-Switch 中，找到你刚添加的 DeepSeek Provider。
2. 旁边有一个**"启用"开关**（打开后 CC-Switch 会接管网络代理），**把它打开**（变成绿色/蓝色）。

> 这个开关打开后，Claude 客户端就会自动通过 CC-Switch 去调用 DeepSeek 的 AI 能力。你什么都不用配置。

3. 现在**打开**桌面上 Claude 的图标。

4. Claude 客户端打开后，你会看到聊天界面——一个输入框，跟微信聊天差不多。

---

## 1.6 第一次对话验证（4分钟）

1. 在 Claude 客户端的输入框里，**输入**下面这句话：

> 请帮我把下面这段话改成正式邮件格式：
> "小王你好，周报你写好了吗？我这边还要看看，另外周五的会议你安排了吗？"

2. **按回车键**，等几秒钟。

3. AI 会回复你一段正式邮件格式的文字。**恭喜你，你的 AI 副手上线了！**

> 💡 **提示**：如果 AI 回复有点慢（3-10秒），是正常的，它在"思考"。不是电脑坏了，别急。

如果出现报错信息：
- 检查 CC-Switch 的"系统代理"开关是否已经打开
- 检查 CC-Switch 里的健康检查是否还是绿色状态
- 关掉 Claude 客户端，重新打开再试一次

## 1.7 关于数据安全（2分钟）

你可能会想：我把工作内容打进去，数据安全吗？说实话是这样的——

**只有你发出去的那句话会经过网络**，你电脑上的文件不会跑出去。

数据流向：
```
你打的话 → Claude客户端 → CC-Switch → DeepSeek服务器 → 计算结果原路返回
```

- ✅ 你发给 AI 的那句话，会经过网络（这是肯定的，因为 AI 不在你电脑里）
- ✅ 你电脑上的其他文件，它们不会自动发送出去
- ⚠️ 如果你粘贴了机密数据（比如完整的合同），它会经过网络

**给你的建议**：如果处理敏感内容，只粘贴需要改写或翻译的那几行，不要全文复制。

## 1.8 本章小结 + 常见问题排查（2分钟）

**安装三步走回顾：**
1. 🟢 **装 Claude 桌面客户端** → 你的 AI 聊天室
2. 🟢 **装 CC-Switch** + 配置 DeepSeek → 万能遥控器
3. 🟢 **打开系统代理开关** + 开始对话 → 跑通第一次

**常见问题排查**

| 问题现象 | 解决方案 |
|----------|----------|
| 安装包打不开，说"不是有效的 Win32 程序" | 下载文件可能损坏，重新下载一次；或者电脑是 32 位系统，装不了 |
| Mac 提示"无法验证开发者" | 去"系统设置 → 隐私与安全性 → 仍要打开"，点一下就行 |
| CC-Switch 里添加 Provider 时看不到 DeepSeek | 先检查 CC-Switch 版本是否为最新，或者点"自定义"手动添加 |
| 健康检查显示"连接失败" | 检查网络是否正常、API Key 有没有复制对（注意不要多复制空格） |
| Claude 客户端打开后提示"登录"或"Sign in" | 不用理它——确认 CC-Switch 的代理开关已经打开，然后关掉 Claude 重新打开 |
| AI 回复特别慢或者一直不回复 | 等 10-15 秒，如果还没反应，关掉 Claude 重新打开；检查 CC-Switch 状态是否正常 |
| 对话刚开始正常，突然报错了 | 可能网络临时抖动，关掉 Claude 再打开就行 |
