### 2. WinDbg 实战（notepad.exe）

**安装方式**：`winget install Microsoft.WinDbg`（ARM64 版本）

**实战流程**：
1. 命令行 `windbg notepad` 启动
2. `bp notepad!wWinMain` 设断点 → `g` 运行 → 断点命中 ✅
3. `g` 继续 → notepad 正常弹窗 ✅
4. Ctrl+Break 暂停回来 → `dd @sp L10` 看栈内存 ✅
5. `uf notepad!wWinMain` 反汇编函数 ✅

**ARM64 寄存器注意事项**：
- 栈指针不叫 `rsp`，叫 `sp`
- 程序计数器不叫 `rip`，叫 `pc`
- 帧指针不叫 `rbp`，叫 `fp`
- 返回地址在 `lr`（x30）寄存器

### 3. PE 结构分析工具

- **PE-bear**：ARM64 Win11 界面加载异常，找不到导入表入口 → 弃用
- **CFF Explorer** 替代：装好后直接查导入表和节区 ✅

### 4. CFF Explorer 实操

- **File → Open → notepad.exe**
- **Import Directory**：看到 DLL 列表（USER32、GDI32、api-* 等）
- **Section Headers**：看到 .text / .fothx / .rdata / .data / .rsrc 等
- Number of Sections = 8（含隐藏元数据）
- Machine = 0xAA64（ARM64）
