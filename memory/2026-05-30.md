## 设备保养看板生成器开发

**需求**：主人要做一张图发微信群，用于技术团队对齐设备保养计划

**方案**：CSV输入 → Python脚本(plot_board.py) → PNG输出

**CSV格式**：
```
优先级,项目,内容,负责人,交付日期,进度
P0,三轴点胶机,导轨润滑+丝杆精度校准,张三,2026-06-02,100
```

**字段说明**：
- 优先级：P0(紧急)/P1(重要)/P2(常规)
- 进度：0~100 百分比数字，转进度条显示

**配色体系**（莫兰迪柔和风）：
- 背景：柔和灰白 #F5F6F8
- 表头：深蓝灰 #2C3E50
- P0标签：暗红 #C0392B
- P1标签：暖橙 #E67E22
- P2标签：柔和蓝 #2980B9
- 进度条：>90%绿 / 30-89%蓝 / <30%灰

**脚本路径**：`workspace/喇叭产线自动化/看板生成器/plan_board.py`
**样例数据**：`.../看板生成器/sample.csv`
**用法**：`python3 plan_board.py` 或 `python3 plan_board.py my.csv --title "标题"`

**收获**：
- 用 web_search + Kimi API 搜到了设备信息和配色方案，要多用这个工具
- 四步流程（需求→规格→内容→验收）有效，灵魂团队分工明确
