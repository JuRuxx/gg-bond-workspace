### 3. XOR 加密解密原理

**核心规则**（4条）：
- 0 XOR 0 = 0
- 0 XOR 1 = 1
- 1 XOR 0 = 1
- 1 XOR 1 = 0
- 一句话：**相同→0，不同→1**

**关键特性**：A XOR B XOR B = A（同一个数 XOR 两次回到原数）
所以加解密用同一个操作。
