### 全书核心确立

大野耐一的命题链：
```
前提：日本战后穷、市场小、需求多样化 → 不能抄美国大批量生产
核心：消除浪费是根本
两大支柱：JIT（准时制）+ 自治化（带人字旁的自动化）
实现工具：看板（6条规则）
目标：缩短从客户下单到收款的时间线
```

第一章核心：
- 思想原点：日美劳动生产率1:9 → 不是美国人力气大 → 日本人肯定在浪费 → 消除浪费就能翻10倍
- 丰田喜一郎"三年追上美国"
- 三个反转：前工序推→后工序拉；自动化→自治化；大批量→小批量多品种

第二章核心（六把工具）：
1. **5Why** — 找到根因（装滤网，不换保险丝）
2. **7大浪费** — 过生产是最可怕的"母浪费"
3. **标准作业三要素** — 节拍时间、作业顺序、标准手持
4. **互助运动** — 交接棒、多能工
5. **看板6条规则** — 第6条最狠：减少看板=倒逼持续改进
6. **平准化** — 换模时间从2-3小时降到3分钟，混流生产

第三章核心（管理哲学）：
- 企业自主神经系统：决策权下放，现场自主响应
- 信息也要JIT：信息过量=浪费，只有最终总装线知道完整排程
- 动作分类：浪费→非增值→增值，目标是增值工作达到100%
- 表观效率陷阱：机器轰轰响≠效率高，市场不需要就是浪费
- 乌龟和兔子：稳定的乌龟比时快时停的兔子浪费更少
- 保护旧设备：机器价值看赚钱能力，不看年限
- 减人≠省力化：省0.9人=没省，至少省1人才叫减人
- 忍术管理：管理不是算术，是修炼出来的直觉

第四章核心（谱系）：
- 丰田佐吉（自动织机）：现场观察的极致→5Why的源头；"日本人只是模仿者"的警醒
- 丰田喜一郎：1933年说"学习但不照搬"→准时制思想的起源；三年不出车但坚持
- 100万日元专利费全投研发，敢于为长期目标忍受短期痛苦

第五章核心（福特翻案）：
- 福特本人的真正意图是"建立流动"，不是"大批量"；继承者曲解了他
- 福特在《Today and Tomorrow》中写：效率不是速度和数量；标准应由一线制定；理想状态是没有东西要回收
- 大野引用福特原著来证明TPS与福特本意一致

第六章核心（24术语）：
