## 核心钢钉

**传播是波的物理现象。**

不是隐喻，是机制。传播学所有的问题，都是波的问题。

## 全书结构

### 上半部：用声学重新解释传播学

**目的**：把传播学/新闻学的经典问题，用波的物理学重新翻译，从"描述性理论"升级为"机制层解释"。

**核心概念展开**：

| 波的属性 | 物理含义 | 传播学映射 |
|---------|---------|-----------|
| **波** | 信息和能量的传播方式 | 传播的本质——不是"信息传递"，是模式在介质中的传播 |
| **震动** | 波的来源，什么在动 | 信息源——每个发声的人/组织都是震源 |
| **频率** | 每秒震动次数 | 为什么有些信息能传播、有些不能——频率必须匹配接收者的固有频率 |
| **振幅** | 波的高度/强度 | 信息的能量——投入多少、信号多强 |
| **相位** | 在周期中的位置 | 时机和同步性——同相放大，反相抵消 |
| **波长** | 一个完整周期的长度 | 传播范围和精度的权衡——长波传远但模糊，短波精准但传不远 |

**两大核心机制**：

| 机制 | 物理解释 | 传播学现象 |
|------|---------|-----------|
| **复读机** | 波的叠加——多个震源同频、相邻时间发射，波互相放大 | 为什么"常识"会出现、为什么观点会"火" |
| **回声机** | 受迫振动/共振——你的波命中其他震源的固有频率，引起反应 | 为什么有些信息能引爆、有些无人理睬 |

**传播学经典问题的波物理学翻译**：

| 传播学问题 | 传统解释（描述性） | 波物理学解释（机制层） |
|---|---|---|
| 为什么有些消息传播快？ | "有新闻价值" | 高频率+大振幅+介质匹配 |
| 谣言为什么止于智者？ | "有判断力" | 固有频率不匹配，无法共振 |
| 信息茧房为什么存在？ | "算法推荐" | 频率过滤——只接收匹配固有频率的波 |
| 沉默的螺旋怎么发生？ | "社会压力" | 小振幅波被大振幅波压制（破坏性干涉） |
| 意见领袖为什么有效？ | "两级传播" | 高振幅震源，引起大规模受迫振动 |
| 假新闻为什么能传播？ | "情绪化" | 频率命中恐惧/愤怒的固有频率，共振强烈 |
| 热点为什么会消退？ | "注意力转移" | 波的自然衰减+新震源出现 |

### 下半部：信息爆炸下如何自处

**目的**：既然传播是波的现象，个体如何在这个波场里活得清醒、有效？

**可能的章节方向**（待展开）：

1. **调频** — 选择接收什么频率的波（注意力管理）
2. **做震源** — 如何成为一个有效的信息源（发声策略）
3. **识别叠加** — 如何分辨"真共识"和"假共识"（批判性思维）
4. **利用共振** — 如何让声音传播得更远（影响力策略）
5. **过滤噪声** — 如何在爆炸中保持信号（信息饮食）

## 关键洞察记录

### 2026-06-02 思考轨迹

1. "回声筒"不对味 → 不是反馈，是重复
2. "复读机"更接近 → 说过的话被重复很多次
3. "世界既是回声筒也是复读机" → 不够尖锐
4. "调频你的信号" → 书名定了，但缺乏钢钉
5. "波水一个"启发 → 意识和世界不可分
6. 两本书不在一个维度 → 《计算》是机制层，调频是本体层
7. **突破**：用波的物理学重新解释传播学 → 和《计算》同构但不同领域
8. 复读机 = 波的叠加，回声机 = 受迫振动/共振 → 机制层解释到位
9. 上半部解释机制，下半部给自处方案 → 结构落定
