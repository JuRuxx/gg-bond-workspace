# 五条波动规律（Cross-domain Compression）

> 日期：2026-06-03
> 来源：ChatGPT 反馈 + 猪猪侠整理
> 核心洞察：**不是给传播学理论找声学类比，而是找底层生成规律**

---

## 核心压缩

传播学的13个理论，最后压缩成5条波动规律：

| 波动规律 | 对应传播现象 | 本质问题 |
|:---|:---|:---|
| **共振 (Resonance)** | 议程设置、回声室、群体极化、品牌传播、个人影响力、宗教传播、模因传播 | **传播为什么发生** |
| **滤波 (Filtering)** | 把关人、信息茧房、算法推荐、注意力机制、世界观 | **什么能传播** |
| **干涉 (Interference)** | 沉默的螺旋、舆论冲突、政治对立、品牌竞争 | **不同传播如何相互作用** |
| **衰减 (Attenuation)** | 遗忘曲线、谣言传播、组织沟通 | **为什么传播会失真** |
| **相变 (Phase Transition)** | 创新扩散S曲线、微信/ChatGPT/TikTok爆发 | **为什么突然爆发** |

---

## 1. 共振（Resonance）— 第一主角

**物理机制**：外力频率 = 系统固有频率 → 振幅急剧放大

**传播学解释**：
> 不是声音大就能传播，而是频率匹配才能传播。

**对应现象**：
- 议程设置：媒体调高某议题的"振幅"→公众对该频率共振
- 回声室：同质群体内部频率匹配→信号反复共振放大
- 群体极化：同频信号叠加→振幅越来越大→走向极端
- 品牌传播：品牌频率与目标用户固有频率匹配→引发共振
- 个人影响力：你的频率与某个群体匹配→成为他们的"震源"

**核心金句**：
> 世界是回声筒。你发出什么频率，就引发什么共振。

---

## 2. 滤波（Filtering）— 第二主角

**物理机制**：只允许特定频率范围的波通过，其他被阻挡

**传播学解释**：
很多人以为：
```
世界 → 我
```
实际上：
```
世界
 ↓
滤波器
 ↓
我
```

**对应现象**：
- 把关人：编辑/算法是滤波器，只让特定频率通过
- 信息茧房：用户主动选择→自我建构的滤波器
- 过滤气泡：算法被动建构的滤波器
- 注意力机制：大脑本身就是一个滤波器
- 世界观：**最大的滤波器不是算法，而是你的世界模型**

**核心金句**：
> 你看到的世界，是你的滤波器决定的。

---

## 3. 干涉（Interference）— 第三主角

**物理机制**：
- 建设性干涉：两波同相叠加 → 振幅增大（A + A = 2A）
- 破坏性干涉：两波反相叠加 → 振幅抵消（A + (-A) = 0）

**传播学解释**：
很多争论其实不是说服，而是互相抵消。

**对应现象**：
- 沉默的螺旋：劣势意见与主流声音反相→被抵消（破坏性干涉）
- 舆论冲突：两种对立观点反相→互相削弱
- 政治对立：左右两派反相→社会撕裂
- 品牌竞争：两个品牌同频竞争→互相抵消能量

**核心金句**：
> 建设性干涉让信号更强，破坏性干涉让信号消失。

---

## 4. 衰减（Attenuation）— 被忽视的规律

**物理机制**：波传播过程中能量损失→振幅减小

**传播学解释**：
现实世界：
```
发射的信息 ≠ 接收到的信息
```
而是：
```
接收信息 = 发射信息 - 衰减 - 噪音
```

**对应现象**：
- 遗忘曲线：信息随时间衰减
- 谣言传播：信息在传播链中逐层失真
- 组织沟通：层级越多，信息衰减越严重
- 跨文化传播：信息跨越不同介质（文化）时能量损失

**核心金句**：
> 信息每传播一次，就衰减一次。你以为的"共识"，可能只是衰减后的残影。

---

## 5. 相变（Phase Transition）— 爆发机制

**物理机制**：系统在临界点发生质的变化（水→蒸汽）

**传播学解释**：
在临界点之前：变化很小
超过临界点：突然爆发

**对应现象**：
- 创新扩散S曲线：早期缓慢→起飞期快速增长→饱和期减缓
- 微信爆发：用户数在某个临界点后指数增长
- ChatGPT爆发：AI对话在某个临界点后突然全民使用
- TikTok爆发：短视频在某个临界点后席卷全球

**核心金句**：
> 传播不是线性的，而是相变的。临界点之前看似无动静，临界点之后势不可挡。

## 统一模型（最终压缩）

```
世界 = 场
人 = 振荡器
信息 = 波
认知 = 滤波器
传播 = 共振
```

**展开解释**：
- **人像振荡器**：每个人都在发射和接收信号，有自己的固有频率
- **组织像共振腔**：组织是一个封闭空间，内部信号反复共振放大
- **媒体像放大器**：媒体调高某些频率的振幅，让公众更容易共振
- **算法像滤波器**：算法决定哪些频率能通过，哪些被阻挡
- **舆论像干涉场**：多种声音叠加，形成建设性或破坏性干涉
- **创新扩散像相变**：在临界点之前缓慢积累，临界点之后突然爆发

## 与《计算你的语言》的关系

| | 《计算你的语言》 | 《调频你的信号》 |
|:---|:---|:---|
| **宇宙观** | 计算宇宙观 | 波动宇宙观 |
| **核心框架** | 信息、压缩、预测 | 波动、共振、传播 |
| **底层机制** | 语言是向量空间的计算 | 传播是波的物理现象 |
| **解释领域** | 语言活动（读/想/说/写） | 传播活动（发射/接收/共振/滤波） |

**汇合点**：
> "识智何分，波水一个" — 计算的是结构（水），波动的是过程（波）。真正的统一模型，同时需要两种视角。
