## 扬声器结构快速参考

### 零件的正确空间关系（从外到内）

```
盆架（外壳）
  └─ 振膜（纸盆/锥形，推动空气）
       └─ 防尘帽（盖在音圈上方）
       └─ 音圈（粘在振膜中心，插在华司与T铁之间的磁间隙里）

磁路系统（全程静止，只提供磁场）
  ├─ 华司（上面，导磁上板）
  ├─ 磁钢（中间，永磁体）
  └─ T铁（下面，导磁下板+中心凸柱）

弹波（定心支片）
  └─ 连接音圈和盆架，让音圈只上下动不左右晃
```

**关键理解：**
- 磁路（华司+磁钢+T铁）**全程静止不动**，不是被推动的
- 信号链路：音频电流→音圈受力上下振→带动振膜振→推空气产生声波
- 华司内孔和T铁凸柱之间的环形缝隙叫**磁间隙**，音圈插在这里上下振动
- 磁间隙通常只有0.2-0.5mm，所以音圈对中是良率的关键

### 喇叭工作原理总结（陈绍禹版）

> 扬声器是一种把电信号变成声音的装置。核心由**磁系统**、**振膜**和**机械系统**三部分协同工作。

#### 完整信号链路

```
音频电信号 → 音圈（通电后在磁场中受安培力）→ 音圈上下振动
→ 带动振膜振动 → 振膜推拉空气 → 声波 → 耳朵听到声音
```

#### 三大系统

**① 磁系统（能量来源）**
- 由一块**永磁体（磁钢）** + **华司（导磁上板）** + **T铁（导磁下板+中心柱）** 组成
- 三者配合在环形间隙中形成集中强磁场
- 音圈（铜线圈）插在这个间隙里，通电后在磁场中受力
- ❌ 常见误解：不是两块磁体相对，而是一块永磁体产生磁场，音圈是通电后受力运动，不是磁体

**② 振膜（发声部件）**
- 表面的圆形薄片，常用聚酯膜或纸制成，轻薄且有刚性
- 与音圈紧密相连，音圈动它就动
- 振膜振动推拉空气 → 产生声波

**③ 机械系统（支撑与保护）**
- **盆架**：金属或高强度塑料制成，固定所有零件的骨架
- **折环（衬垫）**：振膜边缘的弹性环，保护振膜并密封背面防止漏音
- **弹波（定心支片）**：波浪形圆片，连接音圈和盆架，确保音圈只上下动不左右晃（防止擦圈的关键零件）

#### 音量和音质
- **音量**：取决于输入信号强度（电压高→振幅大→声音大）
- **音质**：受频率响应、振膜材料、磁路结构设计等多因素影响
- 多单元分频设计：不同尺寸的振膜+线圈分别处理高音、中音、低音

#### 一言以蔽之
> 电生磁，磁驱线圈，线圈推振膜，振膜推空气——这就是扬声器。
