## 一、胶水类型及特性

### 1.1 胶水分类

| 类型 | 特点 | 适用场景 | 注意事项 |
|:----|:-----|:---------|:---------|
| **含溶剂胶** | 含有机溶剂，挥发后固化 | 音圈、振膜粘接 | 需控制挥发速度，注意通风 |
| **AB胶** | A胶+B胶双组分混合固化 | 磁路系统、结构粘接 | 气温升高变粘稠，需冷藏保存 |
| **含甲苯胶** | 固含量4500-20000，甲苯为溶剂 | 通用粘接 | 随温度变化粘度明显，甲苯高温易挥发 |

### 1.2 AB胶特性补充

| 条件 | 表现 | 应对 |
|:----|:----|:----|
| 气温升高 | 胶水变粘稠 | 存放于冰箱（2-8℃） |
| 标准测试温度 | 20℃ | 以此温度下的粘度作为标准参考值 |
| 禁忌 | 不能加任何溶剂 | 严格按A胶:B胶比例混合（如2:1） |

### 1.3 含甲苯胶特性补充

| 条件 | 表现 | 应对 |
|:----|:----|:----|
| 高温环境 | 甲苯挥发加快 → 胶水变稠 | 控制环境温度，缩短胶水存放时间 |
| 低温环境（冬季） | 胶水流动性差，出胶困难 | 可按 9.5:0.5 比例加入少量水改善 |
| 密封方法 | 表面倒一层薄薄甲粉 | 保持胶水浓度和流畅性 |
| 固含量范围 | 4500-20000 | 根据具体型号确认 |

## 二、调胶环境标准

### 2.1 环境参数

| 项目 | 标准值 | 允许偏差 |
|:----|:-------|:---------|
| 温度 | 20℃ | ±3℃ |
| 湿度 | 60%-70% RH | — |
| 通风 | 良好 | 含甲苯胶需有排风装置 |

### 2.2 适用场景

- 调胶操作
- 胶水储存
- 粘度测试
- 工艺实验

> **注意：** 环境偏离标准范围时，胶水粘度会随之变化，影响点胶一致性和粘接质量。

## 三、调胶操作流程

### 3.1 调胶步骤

```
┌─────────────────────────────────────────────┐
│ ① 准备                                           │
│    ▸ 确认环境温湿度符合标准                   │
│    ▸ 取出所需胶水（AB胶提前30分钟从冰箱取出回温） │
│    ▸ 准备调胶容器、搅拌棒、量杯、粘度计         │
└──────────────────────┬──────────────────────────┘
                       ↓
┌──────────────────────┴──────────────────────────┐
│ ② 配比                                             │
│    ▸ 按工艺要求确定胶水配比比例                 │
│    ▸ AB胶：严格按A胶:B胶比例混合（如2:1）      │
│    ▸ 含甲苯胶：确认固含量值范围                 │
└──────────────────────┬──────────────────────────┘
                       ↓
┌──────────────────────┴──────────────────────────┐
│ ③ 混合搅拌                                     │
│    ▸ 匀速搅拌至胶水均匀、无明显色差或颗粒      │
│    ▸ 搅拌时间不宜过长，避免气泡产生                │
└──────────────────────┬──────────────────────────┘
                       ↓
┌──────────────────────┴──────────────────────────┐
│ ④ 粘度检测（存档）                             │
│    ▸ 使用粘度计测量当前胶水粘度值              │
│    ▸ 标准：以不拉丝且满足工艺实验和客户要求为准 │
│    ▸ 记录粘度值存档（见附件：粘度记录表）       │
└──────────────────────┬──────────────────────────┘
                       ↓
┌──────────────────────┴──────────────────────────┐
│ ⑤ 密封                                             │
│    ▸ 调好的胶水表面倒一层薄薄甲粉密封           │
│    ▸ 密封后可保持胶水浓度和出胶流畅性             │
└──────────────────────┬──────────────────────────┘
                       ↓
┌──────────────────────┴──────────────────────────┐
│ ⑥ 标识                                               │
│    ▸ 在调胶容器上标注：胶水型号、调配时间、粘度值   │
│    ▸ 有效期：当天调好当天用，不可隔夜使用         │
└─────────────────────────────────────────────────┘
```

### 3.2 用量计算参考

**例：小型喇叭**
- 单只用胶量：0.3克/个
- 日产量：8,000个/天
- 日需胶量：2.4公斤
- 建议容器：2.5公斤小桶（刚好用完，不留余胶）

> 各型号用胶量请参考《各型号用胶量参数表》（待补充）。

## 四、胶水使用注意事项

### 4.1 含甲苯胶

| 场景 | 注意事项 |
|:----|:---------|
| 高温天气 | 胶水干得快，应适当加快操作节奏或减少单次调胶量 |
| 低温天气（冬季） | 流动性差，可用水少量稀释（9.5:0.5比例，需经工艺确认） |
| 常规操作 | 胶水用完后需及时密封，防止甲苯挥发 |

### 4.2 AB胶

| 场景 | 注意事项 |
|:----|:---------|
| 温度较高 | 从冰箱取出后需回温30分钟再调配 |
| 调配 | 不能加任何溶剂，严格按比例混合 |
| 混胶 | A胶+B胶充分搅拌均匀 |

### 4.3 通用注意事项

| # | 内容 |
|:-:|:-----|
| 1 | 当天调好的胶水当天用完，不留余胶过夜 |
| 2 | 调胶时穿戴防护手套，含甲苯胶需佩戴口罩 |
| 3 | 每次调胶后记录粘度值（存档） |
| 4 | 胶水容器使用后及时密封，防止溶剂挥发 |
| 5 | 发现胶水外观异常（结块、变色、分层）立即停用 |

## 五、开胶操作

### 5.1 开机准备

| 步骤 | 操作 | 说明 |
|:----|:------|:------|
| ① 门领放气 | 逆时针拧门领放气，胶稠时拧到45度 | 打开胶桶密封 |
| ② 开机监控 | 两三分钟查看一次胶量 | 持续约半小时至稳定 |
| ③ 稳定气压 | 正常工作气压 3.5-4.x kg/cm² | — |
| ④ 气压过高处理 | 超过5时去掉针头排一碗胶 | 排胶时间长，隔天查看 |

## 六、胶枪结构与原理

### 6.1 胶枪结构总览

```
     ┌─────┐
气① ─┤     ├─ 气②    ← 上、下气接头，接电磁阀气管
     │ 气缸 │         ← 气推动活塞上下活动
     └──┬──┘
        │
    ┌───┴───┐
    │  枪身  │         ← 胶管进胶
    └───┬───┘
        │
    ┌───┴───┐
    │ 行程   │         ← 拧动调节顶针抬起高度→调节出胶量
    │ 螺母   │
    └───┬───┘
        │
      ┌─┴─┐
      │顶针│          ← 与针嘴配合控制出胶开关（圆头/尖头两种）
      └─┬─┘
        │
    ┌───┴───┐
    │  针嘴   │         ← 胶水最终出口
    └───────┘
```

### 6.2 各系统说明

| 系统 | 组成 | 功能 |
|:-----|:-----|:------|
| **气路部分** | 顶针、环、气缸、上下气接头 | 气路推动顶针上下运动，常开气管插上面接头 |
| **密封部分** | 气缸密封圈、顶针与缸体接触密封圈 | 防止漏气，需定期打黄油保养 |
| **出胶部分** | 胶管进胶、针嘴出胶、顶针控制开关 | 顶针抬起→胶水流出，顶针压下→胶水停止 |

### 6.3 工作原理

> **核心原理：顶针上下运动控制胶水开关**

```
气接通 → 气压推动活塞 → 顶针抬起 → 胶水流经针嘴 → 出胶
气断开 → 气压消失 → 顶针回落 → 顶针堵住针嘴 → 停胶
```

### 6.4 顶针类型

| 类型 | 特点 |
|:-----|:------|
| **圆头顶针** | 与圆孔配合密封 |
| **尖头顶针** | 与锥形孔配合密封 |

## 七、胶枪调试

### 7.1 三个核心参数

#### ① 行程调节 — 调出胶量

| 操作 | 效果 |
|:----|:------|
| 拧松行程螺母 | 顶针抬得更高→缝隙更大→出胶更多 |
| 拧紧行程螺母 | 顶针抬得更低→缝隙更小→出胶更少 |

> **注意**：避免冲击力过大导致收尾出胶异常

#### ② 回吸调节 — 防滴胶

| 现象 | 说明 |
|:----|:------|
| **无回吸** | 顶针关闭后针嘴挂着一滴胶→滴到产品上 |
| **有回吸** | 顶针关闭后往回抽一丁点→把针嘴上那滴吸回去 |

> 方法：气路反接，调节行程控制回吸量

#### ③ 速度调节 — 出胶响应

| 操作 | 效果 |
|:----|:------|
| 气路调节阀开至最大 | 快速响应，出胶干脆，不打胶异常 |

## 八、胶枪故障排查与维护

### 8.1 常见故障

| 故障 | 现象 | 原因 | 处理 |
|:----|:-----|:------|:------|
| **滴胶** | 顶针关闭后仍慢悠悠滴胶 | 顶针与针嘴未完全密封（杂质/干结胶水） | 拆下白色圈清洗 |
| **漏胶** | 胶水从非针嘴处渗出 | 密封圈老化、未锁紧 | 更换密封圈或锁紧部件 |
| **新胶枪不良** | 上机不正常 | 出厂密封圈未装好/装歪 | 拆开检查重新装配 |
| **不出胶** | 顶针不动作 | 气路不通/密封圈破损漏气 | 检查气管、电磁阀、密封圈 |

### 8.2 日常维护

| 项目 | 频率 | 方法 |
|:----|:----|:------|
| 密封圈保养 | 定期 | 打黄油润滑 |
| 针嘴清洗 | 每日 | 拆下清洗防止残胶堵塞 |
| 设备清洗 | 胶水粘死时 | 拆除清洗，尤其弹膜胶易干 |

> **胶水干固速度对比**：弹膜胶 > 白胶 ≈ AB胶

## 九、打胶工艺

### 9.1 打胶参数说明

| 参数 | 作用 | 适用场景 |
|:-----|:------|:---------|
| **提前开胶** | 移动前先出胶，保证起点铺满 | 直线起点、圆形第一个点 |
| **提前关胶** | 到达终点前先关胶，防止堆积 | 直线终点 |
| **开胶延时** | 胶枪打开后出一会儿胶再移动 | 防止起点胶量不足 |
| **关胶延时** | 关胶后再移动一小段才停 | 一般≥0.5秒，帽子胶需调长 |

### 9.2 三种打胶方式

#### ① 直线打胶

```
操作：胶枪沿直线从起点走到终点
参数：提前开胶 + 提前关胶 + 关胶延时
目标：胶线铺满且均匀，不拉丝
```

#### ② 圆形打胶

```
操作：胶枪沿圆周走一圈
参数：
  · 第一个点：用提前开胶
  · 最后一个点：用关胶延时
  · 中间点：不可用这两个功能（否则会停顿导致胶水堆积）
```

#### ③ 六工位打胶

| 参数 | 说明 |
|:-----|:------|
| 转盘转速 | 固定，转一圈0.8秒 |
| 出胶时间 | 不能太短（如0.2秒会打不满） |
| 调整依据 | 气压、针头粗细 |
| 示例 | 滑石胶大喇叭出胶时间约1.5秒 |

### 9.3 设备清洗

| 原因 | 后果 | 处理 |
|:-----|:------|:------|
| 胶水长时间使用粘死 | 拧动困难 | 拆除清洗 |
| 不清洗 | 影响出胶速度、造成滴胶 | 按期清洗 |
