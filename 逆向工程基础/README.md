## 这是什么

一套零基础的逆向工程入门笔记，用自动化设备行业的语言打比方，适合没有任何逆向背景的初学者。

**学习对象：** 陈绍禹（主人）的私人学习笔记。由猪猪侠（GG Bond）协助整理。

## 配套工具

| 工具 | 用途 | 平台 |
|:----|:-----|:-----|
| **Ghidra** | 静态反编译（看 exe 的代码逻辑） | Mac / Win / Linux |
| **Cheat Engine** | 内存搜索/修改（游戏逆向） | Windows（x86）|
| **WinDbg Preview** | 动态调试（运行中打断点看寄存器） | ARM64 Win11 ✅ |
| **CFF Explorer** | PE 结构查看 | Windows |
| **x64dbg** | 动态调试（x86 平台替代 WinDbg） | Windows（x86）❌ ARM64 不兼容 |

## 学习路线建议

```
① 第一章（汇编基础）→ 知道寄存器是啥、指令长啥样
     ↓
② 第二章（PE结构）→ 知道 exe 文件里有什么
     ↓
③ 第三章（反调试）→ 知道程序会怎么防你
     ↓
④ 第四章（Ghidra）→ 动手逆向第一个 CrackMe
     ↓
⑤ 番外篇（概念速查）→ 随时翻阅、巩固基础
     ↓
⑥ 第五章（Cheat Engine）→ 内存搜索实战
     ↓
⑦ 第六章（WinDbg）→ 动态调试（待完成）
```

**学习原则：** 不求一次全记住，反复翻阅、动手实操、卡住重来。
