# 设备点检表制作 — 需求与规格

## 需求（做什么）

参照 `设备点检表(1).xlsx` 的格式，为以下4台设备制作日常点检表（xlsx）：

1. **三轴点胶机** — 3轴联动运动 + 点胶系统
2. **视觉拔磁规机** — 视觉定位 + 磁规拔除机构
3. **视觉自动装磁规机** — 视觉定位 + 磁规自动装配机构
4. **六工位点胶机** — 6工位转盘 + 多工位点胶

## 规格（怎么做）

### 格式规格

- **模板**：严格参照 `设备点检表(1).xlsx` 的T铁点胶机sheet
  - 第1行：公司名 "奥普新(陆河)科技有限公司"
  - 第2行：英文公司名
  - 第3行："设备日常点检表"
  - 第4行：设备编号、设备型号、设备名称、年月
  - 第5行：表头（项次、点检保养内容、保养点检时间）
  - 第6行：周期 + 1~31日数字
  - 第7行：上午/下午交替
  - 第8~20行：每天点检项目（项次1~12/13）
  - 第21行：保养点检者（签名栏）
  - 第22行：检查确认者（每周）
  - 第23行：说明栏
  - 第24行：备注符号说明
- **保存**：每个设备一个独立xlsx，放在 `设备点检表/` 目录下

### 内容规格（每个设备的点检项目）

每个设备的点检项目必须针对该设备**特有的机械/电气部件**制定，不能照抄样本。点检项分为以下几类：

**通用类（每台设备都有的）：**
1. 清洁机身表面及工作台面，去除灰尘、污渍及加工残留
2. 检查压缩空气系统，确认气压稳定在 0.5MPa，无漏气
3. 检查电源、气管、数据线连接，无松动、破损
4. 检查各传感器外观状态，无遮挡、无损坏
5. 检查控制系统，参数正确，显示清晰，按键灵敏
6. 测试设备空载运行，各轴/各工位运动顺畅
7. 记录设备运行状态及异常情况

**设备专用类（针对每台设备特点）：**

#### 三轴点胶机
- 检查料筒密封性，无漏胶无漏气
- 检查胶管老化与破损情况
- 检查点胶阀/针头，确认无堵塞、无滴漏
- 检查X/Y/Z三轴运动机构（导轨、丝杆），无卡顿无异响
- 检查点胶量精度，定期校验

#### 视觉拔磁规机
- 检查视觉相机及镜头，无污染、对焦清晰
- 检查光源照明，亮度正常无闪烁
- 检查拔磁规机构（夹爪/气缸），动作准确、无晃动
- 检查磁规料道/供料机构，无卡料
- 检查废料排出通道，畅通无堵塞
- 检查视觉识别精度，定期校准

#### 视觉自动装磁规机
- 检查视觉相机及镜头，无污染、对焦清晰
- 检查光源照明，亮度正常无闪烁
- 检查磁规供料振动盘/料仓，运行正常、无卡料
- 检查装配机构（压装/推装），动作准确无卡滞
- 检查装配到位检测传感器，灵敏可靠
- 检查视觉定位精度，定期校准

#### 六工位点胶机
- 检查各工位点胶阀/针头，无堵塞无滴漏
- 检查转盘/分度盘，定位准确、无卡顿
- 检查各工位夹具，夹持牢固无磨损
- 检查料筒及胶管密封性
- 检查各工位同步性，确认无跑位
- 检查分割器运行状态，无异响无抖动

### 验收标准

1. [ ] 每个xlsx格式完整（标题、表头、日期列、签名栏、说明）
2. [ ] 点检项数每台 ≥ 12项
3. [ ] 点检项包含通用项 + 设备专用项
4. [ ] 专用项真正针对该设备特点（不能照搬样本）
5. [ ] 行高、列宽、合并单元格样式合理
6. [ ] 文件名规范：`xxx点检表.xlsx`
