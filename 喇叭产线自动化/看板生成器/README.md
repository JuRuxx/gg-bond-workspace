# 设备保养看板生成器

CSV 进 → 看板图 PNG 出，发微信群用的。

## 快速开始

### 1. 准备 CSV

```csv
优先级,项目,内容,负责人,交付日期,进度
P0,三轴点胶机,导轨润滑+丝杆精度校准,张三,2026-06-02,100
P0,视觉拔磁规机,相机镜头清洁+视觉定位校验,李四,2026-06-02,74
P0,六工位点胶机,分割器油位检查+换油,赵六,2026-06-03,5
P1,视觉自动装磁规机,振动盘检修+磁极识别校准,王五,2026-06-04,95
```

**字段说明：**

| 字段 | 必填 | 说明 |
|-----|------|------|
| 优先级 | 是 | P0(紧急) / P1(重要) / P2(常规) |
| 项目 | 是 | 设备名或任务名 |
| 内容 | 是 | 具体干什么 |
| 负责人 | 是 | 落实到人 |
| 交付日期 | 是 | 截止日期 |
| 进度 | 是 | 0~100 的数字，转进度条 |

### 2. 生成看板图

```bash
python3 plan_board.py                              # 默认 sample.csv
python3 plan_board.py 我的计划.csv                   # 用自己的数据
python3 plan_board.py --title "6月第一周保养计划"     # 自定义标题
python3 plan_board.py --sort priority               # 按优先级排序
python3 plan_board.py --sort progress --reverse     # 按进度倒序

# 组合
python3 plan_board.py 6月计划.csv --title "下周计划" --sort priority
```

### 3. 发群

生成 `看板图.png`，直接发微信群对齐。

## 参数说明

| 参数 | 值 | 默认 | 说明 |
|------|-----|------|------|
| 位置参数 | CSV 文件路径 | `sample.csv` | 数据文件 |
| `--title` | 字符串 | 自动周次 | 看板标题 |
| `--sort` | `due` / `priority` / `progress` | `due` | 排序字段 |
| `--reverse` | 无值 | 不启用 | 倒序排列 |

## 配色

莫兰迪柔和风，有活力不张扬：

| 元素 | 颜色 |
|------|------|
| P0 标签 | 暗红 |
| P1 标签 | 暖橙 |
| P2 标签 | 柔和蓝 |
| 进度 >90% | 绿色 |
| 进度 30-89% | 蓝色 |
| 进度 <30% | 灰色 |

## 依赖

```bash
pip3 install matplotlib
```

macOS 自带 Python3，按需装一下 matplotlib 就行。
