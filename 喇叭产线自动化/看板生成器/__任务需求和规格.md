# 设备保养看板生成器 — 需求与规格

## 需求

做一个Python脚本：输入CSV → 输出一张漂亮的看板图PNG，发微信群用的。

## 规格

### CSV输入格式

```csv
优先级,项目,内容,负责人,交付日期,状态
P0,三轴点胶机,导轨润滑+点胶阀密封检查,张三,2026-06-01,待办
P1,视觉拔磁规机,镜头清洁+视觉定位校验,李四,2026-06-02,进行中
P2,视觉自动装磁规机,振动盘检修+磁极识别校准,王五,2026-06-03,已完成
```

### 优先级定义
- **P0** — 必须在规定时间内完成，不做影响生产 → 红色标记
- **P1** — 重要但不紧急，按期做就行 → 橙色/黄色标记
- **P2** — 有余力再做 → 灰色/蓝色标记

### 状态定义（带颜色）
- **已完成** → ✅ 绿色
- **进行中** → 🔄 蓝色
- **待办** → ⏳ 灰色

### 输出图片规格
- 尺寸：宽高比适合手机查看（约 1200×800 或自适应）
- 配色：干净、专业、大厂风
- 表头清晰：优先级 | 项目 | 内容 | 负责人 | 交付日期 | 状态
- 用不同颜色区分P0/P1/P2
- 顶部标题："🔧 设备保养看板  |  第X周"
- 底部：生成日期

### 分工
1. **波比**（技术型）— 写核心Python脚本（matplotlib画图）
2. **小呆呆**（耐心型）— 写样例CSV、测试用例
3. **超人强**（莽撞型）— 批量测试不同数据、调优输出效果
4. **迷糊老师**（验收型）— 审查代码质量、图片美观度
5. **猪猪侠** — 总协调、验收交付
