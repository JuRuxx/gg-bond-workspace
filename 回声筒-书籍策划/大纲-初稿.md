## 核心定位

- **书名**：《调频你的世界》
- **核心观点**：世界不是由物质组成的，而是由信号组成的。你看到的世界，取决于你接收什么信号；你得到的世界，取决于你发射什么信号。
- **调性**：猪猪侠风格——通俗易懂讲深刻道理，像《计算你的语言》《减熵三部曲》
- **与《计算你的语言》配对**：一本讲语言是计算，一本讲世界是频率
- **统一隐喻**：回声筒（反馈）+ 复读机（传播）= 调频（接收+发射+闭环）

## 章节结构

| 章节 | 核心内容 | 一句话定位 |
|:---|:---|:---|
| **引言** | 世界为什么是回声筒？不是比喻，是精确描述 | 从"你关注什么就看见什么"出发，建立回声筒意象 |
| **01 你发出的声音** | 信号质量决定回声质量——实验型发声的四个原则 | 大多数人发出的不是信号是噪音，世界没法给反馈 |
| **02 回声的三种类型** | 确认型（≈0信息量）/矛盾型（🔥高价值）/沉默型（⚠️回路可能断了） | 大多数人只听确认型，高手追矛盾型，大师怕沉默型 |
| **03 如何听回声** | 三问框架：什么符合预期？什么违反预期？什么没发生？ | 贝叶斯更新的极简版，每次收到回声问三个问题 |
| **04 回声筒的陷阱** | 信息茧房、确认偏误、回声筒放大错误信号 | 回声筒不区分真理和幻觉，你输入错了放大的是错 |
| **05 反面信息：最稀缺的高维信号** | 成功告诉你"什么可能"，失败告诉你"什么不可能" | 100个成功不如1个失败案例有价值 |
| **06 反向贝叶斯：逃生通道** | "如果我错了，我需要看到什么才能发现？" | 在回声筒里挖一条逃生通道，防止被自己的信号困住 |
| **07 人际回声筒** | 用回声筒拆黄执中沟通课——短距离回声筒的信噪比优化 | 吵架不是为了赢，是为了暴露世界模型 |
| **08 元印证：本书就是回声筒实验** | 本文生成过程本身就是方法论的活体演示 | 我们问了四个AI，交叉验证，从矛盾中提取增量 |
| **结语** | 行动清单+不同读者的速查表 | 回声筒不会自己帮你，你得主动做实验型发声 |

