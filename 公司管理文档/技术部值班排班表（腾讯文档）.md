# 技术部值班排班表

**用途：** 腾讯文档在线表格，方便生产部知道今天找谁，也方便技术部管理排班和交接。

---

## 表头字段

| # | 字段 | 填写说明 | 示例 |
|---|------|---------|------|
| 1 | **日期** | YYYY-MM-DD | 2026-06-02 |
| 2 | **星期** | 自动或手动填 | 周二 |
| 3 | **值班工程师** | 当日产线第一响应人 | 李四 |
| 4 | **联系方式** | 手机号 / 微信号 | 138xxxxxxx 或 WeChat ID |
| 5 | **带班主管** | 当日可升级对接的主管 | 王主管 |
| 6 | **状态** | 正常 / 调班 / 请假 | 正常 |

---

## 填写规范

### 排班周期
- **每周五下班前**完成下一周的排班
- 排班表发企业微信群的排班人：技术部主管或指定排班人

### 排班原则
- 工作日每天指定一名**值班工程师**
- 值班工程师当日以产线支持为首要任务，非紧急项目工作可暂缓
- 如值班工程师请假或调班，需**提前一天**在表上更新，并通知技术部主管

### 联系方式填写
- 填写值班工程师的手机号 + 微信号
- 生产部在群里@不到人时，直接打这个电话

### 状态说明
| 状态 | 含义 |
|------|------|
| 正常 | 按计划值班 |
| 调班 | 已与XX同事调换，备注注明换班人 |
| 请假 | 当天休假，由带班主管临时调度 |

### P0紧急联系
- 下班后 / 节假日：仅响应P0
- 联系方式：**电话/微信语音联系技术部主管**（主管联系方式单独维护，不公开在表中）

---

## 腾讯文档设置建议

1. **新建** → 在线表格 → 命名《技术部值班排班表》
2. 表头按上述字段建列
3. **状态列** 设置 **下拉选项**（正常/调班/请假）
4. 设置**条件格式**：调班行标黄、请假行标红
5. **共享权限**：技术部全员可编辑，生产部设置"仅查看"
6. 建议置顶本周行 + 冻结首行表头

---

## 周视图建议（生产部快速查看）

可在表下方或另建一个 Sheet：

```
本周值班（6/1 - 6/7）
周一：张三 | 周二：李四 | 周三：王五 | 周四：赵六 | 周五：陈七
周末/节假日紧急联系：技术部主管 - 138xxxxxxx
```

这样生产部一眼扫过去就知道今天找谁。
