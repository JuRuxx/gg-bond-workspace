# 产线报修模板（生产部专用）

**报修请按以下格式发到微信群：**

---

## 📋 标准报修模板

```
【等级】P_
【设备】
【问题】
【机型】
【报修人】
【地点】
【时间】
```

---

## ✅ 填写示例

**P0（停产）：**
```
【等级】P0
【设备】三轴点胶机
【问题】主轴卡死不动，产线停线
【机型】SJ-3000
【报修人】张三
【地点】3楼A线
【时间】14:35
```

**P1（效率下降）：**
```
【等级】P1
【设备】视觉拔磁规机
【问题】视觉识别连续3次失败，节拍变慢
【机型】VPM-200
【报修人】李四
【地点】2楼B线
【时间】09:20
```

**P2（一般故障）：**
```
【等级】P2
【设备】自动上T铁点胶机
【问题】传感器误报，重启后正常
【机型】TDM-100
【报修人】王五
【地点】4楼C线
【时间】16:00
```

---

## ⚡ 特别提醒

- **P0停产** → 先微信或电话找当天负责人，修完再补填单子
- **P1-P3** → 直接群内留言
- **时间**写发现问题的时刻，不是发消息的时刻
