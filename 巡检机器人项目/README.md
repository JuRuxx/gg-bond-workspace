## ✨ 核心方案

| 维度 | 内容 |
|------|------|
| **预算** | ¥10,000 以内 |
| **主控** | Raspberry Pi 4B / Jetson Nano |
| **导航** | 🔄 待定（原方案：视觉导航；可选方案：磁条/磁钉导航） |
| **感知** | 摄像头 + 红外测温 + 超声波避障 |
| **通信** | WiFi + MQTT，机器人 → 云端推送 |
| **充电** | 自动回桩 + 磁吸触点充电 |
| **报警** | 蜂鸣器 + LED + 云端推送（钉钉/微信） |

## 📋 开发路线图

| 阶段 | 内容 | 工期 |
|------|------|------|
| ① 底盘运动 + 基础控制 | 电机驱动、里程计、避障 | 2-3周 |
| ② 感知 + 检测 | 摄像头图像识别、红外测温、定点采集 | 3-4周 |
| ③ 自动回桩 + 报警 | 充电回桩、全链路集成 | 3-4周 |

## 🧲 导航方案（补充于2026-05-27）

老板建议考虑**磁轨导航**替代纯视觉：

- **磁条导航**：地面贴磁条，成本低，适合固定路线
- **磁钉导航**：地面埋磁钉，耐用精度高，改路径麻烦
- **推荐**：如路线固定，可用"磁轨为主 + 视觉为辅"的混合方案

> 详见文档 `01-总体方案.md` 第0章
