# 02-代码架构 · 巡检机器人软件设计

> **说明**：RaspRover官方已提供基础框架（Web控制、视觉、云台、运动控制），
> 本模块在其基础上**扩展巡检功能**。

## 目录结构

```
code/
├── inspection/              ← 巡检主程序（核心逻辑）
│   ├── patroller.py         ← 巡检状态机（启动→巡检→报警→回充）
│   ├── route.py             ← 巡检路线管理（关键点列表）
│   └── scheduler.py         ← 定时任务/手动触发
├── sensors/                 ← 传感器扩展驱动
│   ├── ir_temp.py           ← MLX90614 红外测温
│   ├── ultrasonic.py        ← HC-SR04 超声波避障
│   └── charger_detect.py    ← 充电桩检测
├── utils/                   ← 工具函数
│   ├── config.py            ← 配置文件读写
│   ├── logger.py            ← 日志记录
│   └── notifier.py          ← 报警通知（蜂鸣器/MQTT/钉钉）
├── setup_extra.sh           ← 一键安装额外依赖
├── requirements_extra.txt   ← 额外Python库
└── README.md                ← 使用说明
```

## 工作流程

```
patroller.py 主循环:
  1. 加载巡检路线（route.py）
  2. 移动到目标点（调用base_ctrl.move_to()）
  3. 到站后采集：
     - 摄像头拍照（调用cv_ctrl.snapshot()）
     - 红外测温（调用ir_temp.read()）
     - 超声波测距（调用ultrasonic.read()）
  4. 异常判断：
     - 温度超过阈值 → 触发报警
     - 视觉检测异常 → 触发报警
  5. 正常 → 写入日志，前往下一站
  6. 低电量 → 触发回桩充电
```

## 部署方式

```bash
cd ~/ugv_rpi
git pull  # 或者下载我的代码包
pip install -r requirements_extra.txt

```
