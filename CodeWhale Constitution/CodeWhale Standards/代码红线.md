> 以下行为在任何情况下均不被允许。

---

## 八条红线

| 编号 | 红线 | 正确做法 |
| --- | --- | --- |
| 1 | 硬编码敏感信息 | 使用环境变量或配置文件 |
| 2 | 硬编码魔数 | 定义命名常量 |
| 3 | 使用 `print` / `console.log` 作为生产日志 | 使用日志框架 |
| 4 | 函数超过 100 行 | 拆分为多个函数 |
| 5 | 文件超过 500 行 | 拆分为多个文件 |
| 6 | 空的异常捕获块 | 正确处理或向上抛出 |
| 7 | 直接拼接 SQL 语句 | 使用参数化查询 |
| 8 | 使用 `eval()` / `exec()` 执行用户输入 | 使用安全的解析器 |

---

## 示例

### 魔数

```python
# 不推荐
if elapsed > 3600:
    print("超时")

# 推荐
MAX_TIMEOUT_SECONDS = 3600
if elapsed > MAX_TIMEOUT_SECONDS:
    print("超时")
```

### SQL

```python
# 不推荐
query = f"SELECT * FROM users WHERE name = '{user_input}'"

# 推荐
query = "SELECT * FROM users WHERE name = ?"
cursor.execute(query, (user_input,))
```
