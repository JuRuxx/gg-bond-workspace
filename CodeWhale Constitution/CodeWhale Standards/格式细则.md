## 异常处理

- 禁止空的异常捕获块
- 禁止使用异常控制正常流程
- 异常必须携带上下文信息

```python
# 不推荐
try:
    result = api.call()
except:
    pass

# 推荐
try:
    result = api.call()
except ConnectionError:
    log.warning("API 连接失败，跳过本次请求")
    result = None
```
