# SKILL 验收检查清单

> CodeWhale Constitution 附属文档 · 独立版本迭代 · 当前版本 2.1
> 每个 SKILL.md 创建或更新时逐项检查。

---

- [ ] 名称反映使用场景，而非函数名
- [ ] 触发场景描述了用户会在什么情况下使用
- [ ] 关键词至少 3 个
- [ ] Inputs 区分了必需和可选
- [ ] Tools 章节列出所有可调用的工具
- [ ] States 包含了完整的决策树或状态转换图
- [ ] 状态转换覆盖了缺失参数、多选场景、异常恢复
- [ ] Success 标准是可验证的具体条件（非模糊描述）
- [ ] Failure 覆盖了已知错误场景和恢复策略
- [ ] 不是工具说明书的复制
- [ ] 格式为 Markdown（非 YAML）
- [ ] 通过最终验收：删掉它以后，Agent 失去的是**决策**而非**动作**
