## 一、水面垃圾打捞船 ⭐ 最推荐

**与我们的方案相似度最高。**

### 工作原理

```
         ← 船前进方向
     ╱╲╱╲╱╲╱╲╱╲    ← 链条上装一排拨齿/捞斗
    ╱  ╱  ╱  ╱  ╲   ← 拨齿伸入水面以下
   ─────────────────   ← 水面
    🍼  🥤  📦        ← 漂浮垃圾
   
   拨齿把垃圾捞起来 → 链条带到船上 → 到顶部自然落入收集箱
```

### 核心结构
- **动力**：柴油机/电机带动链条
- **拨齿**：金属或高强度塑料齿，间隔排列在链条上
- **收集**：垃圾到顶部靠重力掉入收集舱
- **特点**：连续作业，效率极高，纯机械结构

### 与我们方案的对应关系

| 打捞船 | 我们的方案 |
|--------|-----------|
| 水面 | 地面 |
| 链条+拨齿捞垃圾 | 链条+拨齿捞垃圾 |
| 到顶部掉入收集舱 | 到顶部掉入收集袋 |
| 柴油机/电机驱动 | 手摇/推着走 |

**可借鉴的设计细节：**
- 拨齿的间距（太密容易堵，太疏漏垃圾）
- 拨齿的材质和硬度（要够硬钩起瓶子，又要有弹性保护链条）
- 链条速度和前进速度的配合

### 建议搜索方式
- **B站**："垃圾打捞船 工作原理"、"水面垃圾清理船 内部结构"
- **YouTube**："trash skimmer boat how it works"、"water garbage collector conveyor"
- **淘宝**："水面垃圾打捞网"（看结构）

## 二、农用捡石机（Rock Picker）⭐ 很接近

### 工作原理

```
   牵引车 → [捡石机]
              │
     ╱╲╱╲╱╲╱╲╱╲    ← 钢齿链条
    ╱  ╱  ╱  ╱  ╲   ← 齿插入土壤
   ─────────────────   ← 地面
    ●  ●    ●         ← 石头

   齿把石头耙起来 → 链条送到顶部 → 掉入收集斗
```

### 核心结构
- **动力**：拖拉机 PTO（取力器）或液压马达
- **拨齿**：高强度弹簧钢齿，间距可调
- **深度控制**：通过调节轮子高度控制齿入土深度
- **筛分**：部分机型带振动筛，把泥土筛掉只留石头

### 与我们方案的区别
- 捡石机的齿是**硬钢齿**，我们是**弹性拨齿**（因为垃圾比石头轻，要温柔些）
- 捡石机要入土（齿插进土里），我们只要扫地面

### 建议搜索方式
- **B站**："捡石机"、"除石机"
- **YouTube**："rock picker machine working"、"chain rock picker"
- **关键词**："finger rock picker"、"rake rock collector"

## 三、农用拾膜机（Plastic Film Collector）

### 工作原理

```
     ← 拖拉机前进
    ╱╲╱╲╱╲╱╲       ← 弹齿/钉齿
   ╱  ╱  ╱  ╱  ╲
  ────────────────    ← 地面
  ~~~~~~~~~~~~~~~     ← 地膜（塑料薄膜）

   齿钩住地膜 → 带起来 → 链条送到收集箱
```

### 与我们方案最有价值的地方
地膜是**软质的塑料薄膜**，跟包装袋/塑料袋很像。拾膜机能捡起软塌塌的薄膜，说明弹性拨齿对付塑料袋是可行的！

### 建议搜索方式
- **B站**："残膜回收机"、"拾膜机"
- **YouTube**："plastic mulch collector"、"film picker machine"

## 四、拨禾轮（Reel / Pick-up Reel）

### 工作原理

```
       ← 收割机前进
       ╱╲╱╲╱╲    ← 拨禾轮旋转
      ╱  ╱  ╱  ╲
     │  │  │  │  │  
     ─────────────   ← 作物
     
     拨禾轮把倒伏的作物拨起来 → 送到割台
```

### 与我们方案最有价值的地方
拨禾轮的**弹性拨齿**设计非常成熟——用弹簧钢片或工程塑料做成，碰到硬物会弯曲避开，过了自动弹回来。这个材质选择可以直接参考。

### 建议搜索方式
- **B站**："拨禾轮 结构"、"收割机拨禾轮原理"
- **YouTube**："combine reel working"、"pickup reel mechanism"

## 五、清淤抓斗/垃圾抓斗

### 工作原理

```
     ┌────┐
     │ 爪 │  ← 液压驱动，像八爪鱼触手一样合拢
     │ ╱╲ │
     │╱  ╲│
     └────┘
       ↓↓
    抓取垃圾 → 提升 → 松开

   但不是连续的，是一下一下的
```

### 与我们的关系
这个就是"八爪鱼"名字的来源——像爪子一样。但它是**间歇式作业**（抓一下、提起来、松开、再抓），而我们想做**连续式**的。

### 建议搜索方式
- **B站**："垃圾抓斗"、"液压抓斗"
- **YouTube**："hydraulic grapple garbage"

## 六、清扫车/扫路车

### 工作原理

```
     ← 扫路车前进
     ┌──────────┐
     │ [垃圾箱] │
     │    ↑     │
     │ ┌──┴──┐ │
     │ │传送带│ │  ← 把垃圾刮上去
     │ └──┬──┘ │
     │    ↑    │
     │  [滚刷]  │  ← 旋转扫入垃圾
     └────┬────┘
          ↓
       地面垃圾
```

### 与我们方案的比较
- 扫路车已经做得很成熟了，但它是**扫为主**（滚刷扫进去），我们是**拨/钩为主**（拨齿钩起来）
- 扫路车连泥土一起扫，我们要精准挑垃圾
- 扫路车要大功率风机+水喷淋，结构太重太贵

### 建议搜索方式
- **B站**："扫路车工作原理"、"道路清扫车内部结构"
- **YouTube**："street sweeper mechanism"、"road sweeper conveyor"
