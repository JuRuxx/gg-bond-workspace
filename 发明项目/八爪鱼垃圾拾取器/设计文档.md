# 🐙 八爪鱼垃圾拾取器 · 设计文档

> 发明人：陈绍禹 | 版本：v0.1 概念设计 | 创建：2026-05-25

---

## 一、问题定义

### 1.1 痛点

公园、山林、景区、路边的人工垃圾（饮料瓶、包装袋、易拉罐、纸屑等）清理效率低：
- **火钳**：一次只能夹一个，要弯腰，一拿一放，体力消耗大
- **吸尘器**：吸力小吸不动硬垃圾，吸力大成本暴涨且连泥土一起吸
- **扫帚**：只适合轻质垃圾，对瓶子罐子无效

### 1.2 设计目标

| 维度 | 目标 |
|------|------|
| **效率** | 连续作业，不弯腰，比火钳快3倍以上 |
| **成本** | 物料成本<1000元（零售价） |
| **重量** | 整机<8kg，可单人推/拉 |
| **适用** | 塑料瓶/玻璃瓶/易拉罐/包装袋/纸杯等常见人工垃圾 |
| **续航** | 电池版≥2小时连续运行 |
| **噪音** | ≤60dB（相当于正常说话） |
| **维护** | 无需专业技能，15分钟可清堵 |

## 二、技术方案

### 2.1 方案概述

**核心思路**：用橡胶吸盘替代人手/火钳的"抓取"动作，用链条/皮带实现"传送"和"释放"，从而实现连续不弯腰的垃圾拾取循环。

### 2.2 工作循环

```
┌─────────────────────────────────────────────────┐
│                 工作循环                          │
│                                                   │
│  ① 吸盘朝下                                      │
│     ↓ 推到垃圾上、按压排出空气                    │
│  ② 吸盘吸住垃圾 ↑ 负压保持                       │
│     ↓ 链条转动、垃圾被带着上升                    │
│  ③ 到达顶部                                      │
│     ↓ 手动/自动破真空                             │
│  ④ 垃圾掉入收集袋                                │
│     ↓                                            │
│  ⑤ 空吸盘转回底部 → 回到①                       │
└─────────────────────────────────────────────────┘
```

## 三、详细设计

### 3.1 吸盘子系统

#### 3.1.1 选型要求

| 参数 | 建议值 | 原因 |
|------|--------|------|
| 材质 | 丁腈橡胶(NBR)或硅胶 | 耐磨、弹性好 |
| 直径 | 50-80mm | 太小吸不住瓶子(瓶身直径~60mm)，太大灵活性差 |
| 形状 | 平口或浅波纹 | 平口对平面好，波纹对曲面更好 |
| 安装方式 | 螺纹接头(M6/M8)或快插 | 方便拆卸更换 |

#### 3.1.2 吸附原理

```
按压前：          按压中：         提起后：
┌────┐           ┌────┐           ┌────┐
│    │   ↓力     │    │           │  ↑ │  ← 吸盘被拉起
│    │   →→→     │ │  │  ←变形   │    │
│    │   瓶子    ││ │ │           │ │  │  ← 内部负压
└────┘           └──┴─┘           └──┬─┘
 空气              空气被排出      瓶子被吸住
```

**关键：** 吸盘需要足够的柔韧性来贴合瓶子弧面，但又需要有足够的刚性在提起来时不塌陷。

#### 3.1.3 安装方式

吸盘固定在链条的一个个**吸盘座**上：
- 吸盘座通过链条销轴连接在链条上
- 吸盘座可以自由旋转（重力自然使吸盘口朝下）
- 或用导轨强制吸盘方向

```
链条方向 →
    ○====○====○====○====○
    │    │    │    │    │
  吸盘 吸盘 吸盘 吸盘 吸盘
    ↓    ↓    ↓    ↓    ↓
   口朝下（靠重力自然下垂）
```

### 3.2 传动子系统

#### 3.2.1 传动方案对比

| 方案 | 优点 | 缺点 | 推荐 |
|------|------|------|:----:|
| 自行车链条+链轮 | 便宜(30元)、耐用、易改装 | 噪音稍大、需润滑 | ⭐ |
| 同步带+带轮 | 静音、无需润滑 | 较贵(50元+)、打滑风险 | |
| 双链条+横杆 | 稳定性好 | 复杂、重 | |

#### 3.2.2 链条回路设计

```
          ● ← 顶部链轮（从动）
         ╱ ╲
        ╱   ╲
      吸盘  吸盘  吸盘   ← 上行带垃圾
        │    │    │
        ╲   ╱
         ╲ ╱
          ● ← 底部链轮（主动，连接电机）
         ╱ ╲
        ╱   ╲
      吸盘  吸盘  吸盘   ← 下行空载回程
        │    │    │
        ╲   ╱
         ╲ ╱
          ● ← 张紧轮（调节链条松紧）
```

**关键尺寸（估算）：**
- 顶部到底部距离：约500-600mm
- 链条周长：约1200-1400mm（可装6-8个吸盘）
- 链轮齿数：15-20齿

#### 3.2.3 动力计算

| 参数 | 估算 |
|------|------|
| 链条总阻力 | ~5N（吸盘+链条自重+摩擦力） |
| 最大负载（6个吸盘全吸满） | ~30N（假设每个0.5kg） |
| 总阻力 | ~35N |
| 线速度 | 0.15m/s（正常慢走） |
| 所需功率 | P = F×v = 35×0.15 ≈ 5.25W |
| 考虑效率损失 | 选型10-20W减速电机 |
| 电机类型 | 12V直流减速电机，带减速箱，输出转速30-60rpm |

### 3.3 释放子系统

#### 3.3.1 方案A：手动拉线释放（推荐优先验证）

```
手柄位置 → [拉线/扳机]
               |
               ↓
         ┌─────┴─────┐
         │ 杠杆/拨叉  │ ← 在顶部位置
         └─────┬─────┘
               ↓ 顶开吸盘边缘
           ┌───┴───┐
           │ 吸盘   │ ← 边缘被撬起 → 空气进入 → 瓶子掉落
           └───────┘
```

**优点**：简单、零成本、操作者可控
**缺点**：需要手操作

#### 3.3.2 方案B：凸轮自动释放

```
链条上行方向 →
                
   吸盘上升 → 到达顶部 →
                
          [凸轮]
            ╱
           ╱  ← 凸轮曲面逐渐顶开吸盘边缘
     ┌────╯
     │吸盘│ ← 被顶开 → 破真空 → 掉落
     └────┘
           ╲
            ╲ ← 吸盘滑过凸轮，自动闭合
```

**优点**：全自动，无需额外操作
**缺点**：调试复杂，对不同尺寸吸盘通用性差

#### 3.3.3 方案选择

初期原型用 **方案A（手动拉线）**，验证可行后再优化为方案B。

### 3.4 车架子系统

#### 3.4.1 结构示意图（侧视图）

```
         ┌────────────┐
         │    收集袋   │ ← 帆布袋或网袋，快拆
         └─────┬──────┘
               │
    ┌──────────┴──────────┐
    │  链条提升机构        │ ← 钢管框架
    │  ○←←←←←←←←○ 顶部   │
    │  ↓ ↑  ↓ ↑  ↓ ↑     │
    │  ○→→→→→→→→○ 底部   │
    └──────────┬──────────┘
               │
    ═══════════╩═══════════ ← 地面
    ○                     ○ ← 轮子
```

#### 3.4.2 尺寸估算

| 维度 | 尺寸 | 说明 |
|------|------|------|
| 总长 | ~600mm | 前后轮距 |
| 总宽 | ~400mm | 吸盘宽度+框架 |
| 总高 | ~800mm | 从地面到顶部链轮 |
| 离地间隙 | ~50mm | 吸盘口离地高度 |
| 轮径 | ~200mm | 适合草地/土路 |
| 推把高度 | ~900mm | 自然站立推着走 |

#### 3.4.3 材料选择

| 部件 | 材料 | 原因 |
|------|------|------|
| 主框架 | 20×20mm 方钢管 | 强度够、便宜、好焊接 |
| 轮子 | 充气轮/实心橡胶轮 | 户外通过性 |
| 收集袋 | 帆布/尼龙 | 耐磨、透气 |
| 吸盘座 | 3D打印/金属片 | 需要定制形状 |

### 3.5 电气子系统

#### 3.5.1 电路框图

```
     [12V锂电池] ─┬─ [开关] ─ [减速电机]
                   │
                   └─ [LED指示灯]
```

**极简设计**：不需要控制器，直接开关控制电机启停。

#### 3.5.2 选型

| 器件 | 选型 | 价格 |
|------|------|:----:|
| 电池 | 12V 5Ah 锂电池包 | ~80元 |
| 电机 | JGA25-370 12V 50rpm 减速电机 | ~40元 |
| 开关 | 防水船型开关 | ~5元 |
| 充电器 | 12.6V 锂电池充电器 | ~15元 |

**续航估算**：
- 电机功耗约10-15W
- 5Ah × 12V = 60Wh
- 理论续航：60/12.5 ≈ 4.8小时
- 实际：约3-4小时

## 五、设计约束与注意事项

### 5.1 吸盘角度
- 吸盘口朝下是核心要求
- 链条转弯处吸盘会偏转，需确保在吸附区(底部直线段)吸盘垂直向下
- 可用导轨或挡板强制吸盘方向

### 5.2 垃圾掉落区
- 释放后垃圾自由落体进袋子
- 袋子口要足够大(≥250mm宽)
- 袋子固定要牢但可快拆

### 5.3 防卡设计
- 吸盘间距≥150mm，避免大垃圾卡在两个吸盘之间
- 链条两侧加护板，防止垃圾缠入链条
- 收集袋入口加挡板，防止垃圾反弹掉出

### 5.4 重心控制
- 电池和电机尽量靠下靠近轮轴
- 收集袋装垃圾后重心会后移，推车设计时考虑配重或调整轮位
