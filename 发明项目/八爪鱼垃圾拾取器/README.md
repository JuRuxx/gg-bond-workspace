## 痛点

公园/山林/景区的地面人工垃圾（饮料瓶、包装袋、易拉罐）清理，现有方案：
- **火钳**：一次一个，弯腰，一拿一放，效率低
- **吸尘器**：吸力小吸不动垃圾，吸力大成本暴涨
- **扫帚**：只能扫轻垃圾

## 方案

**八爪鱼吸盘 + 皮带传动 + 手动开关释放**

推着走，吸盘碰到垃圾自动吸住，皮带送到顶部释放掉入收集袋。全程不弯腰，连续作业。
