# 燃料路线三巨头：液氧煤油 vs 液氧甲烷 vs 液氧液氢

> 火箭发动机的燃料选择，本质上是一场"密度 vs 比冲"的永恒博弈。就像去便利店买东西——是选又大又便宜的老牌商品，还是前沿但有点贵的黑科技？煤油、甲烷、氢，三大家族各有各的算盘。

---

## 一、老三样扛把子：液氧/煤油（LOX/RP-1）

煤油是火箭燃料界的"老大哥"。从20世纪50年代第一枚洲际导弹开始，煤油就跟着上了天。你可能会问——都用了大半个世纪了，怎么还没被淘汰？因为它的**综合性价比实在太能打了**。

### 代表发动机

| 发动机 | 所属 | 火箭 | 核心特点 |
|--------|------|------|---------|
| **RD-180** | 俄罗斯（NPO Energomash） | Atlas V | 双燃烧室，真空推力423t，分级燃烧循环 |
| **Merlin 1D** | SpaceX | Falcon 9 | 真空95t，燃气发生器，已复用10+次 |
| **YF-100** | 中国航天六院 | 长征五/六/七/八 | 真空134t（增推版），富氧分级燃烧 |
| **YF-102** | 中国航天六院 | 天龙二号 | ~85t，3D打印+针栓，燃气发生器 |
| **天火-12（TH-12）** | 天兵科技 | 天龙三号 | 海平面~111t / 真空~138t，针栓+3D打印 |
| **焦点一号** | 星河动力 | 谷神星一号 | ~55t，燃气发生器 |

你可能不知道的是——RD-180的"精神祖先"RD-170（真空推力806t）至今仍是人类制造的最大推力液体火箭发动机，没有之一。

### 核心参数

| 参数 | 数值 |
|------|------|
| 密度 | ~820 kg/m³（常温，远高于甲烷和氢） |
| 真空比冲 | 320-348 s（视循环方式而定） |
| 推力范围 | 一般单机 30t-800t（RD-171MV 达到 806t） |
| 混合比（O/F） | 约2.2-2.7（质量比） |

### 优点 👍

1. **密度大，储箱小**：煤油密度是甲烷的1.6倍、液氢的12倍。同样容积，煤油装得最多，火箭箭体可以更短更轻。
2. **常温可储存**：煤油在室温下就是液体，不需要保温、不需要长期冷却——这对快速发射、长期在轨储存非常友好。
3. **成本极低**：煤油就是航空煤油RK-1，工业量产，一吨几千块钱。发动机工艺成熟，供应链稳定。
4. **技术最成熟**：从洲际导弹到登月飞船，煤油发动机验证了几千次飞行，设计经验高度积累。

### 缺点 👎

1. **积碳严重**：煤油燃烧不充分时会产生大量碳沉积（coking），堵塞喷注器和涡轮泵。Merlin 1D虽然实现了10+次复用，但每次飞行后都需要返厂清洁甚至更换部件。这是煤油路线绕不过去的坎。
2. **比冲垫底**：纯看能量效率，煤油的真空比冲落在320-348s之间，不如甲烷和氢。对追求极致性能的上面级来说，不够用。

## 二、新晋网红：液氧/甲烷（LOX/CH₄）

甲烷是"技术圈顶流"。但凡你是个航天公司，不提甲烷都不好意思说自己在做下一代发动机。为什么？因为SpaceX选了甲烷。

### 代表发动机

| 发动机 | 所属 | 火箭 | 核心特点 |
|--------|------|------|---------|
| **Raptor 3** | SpaceX | Starship/Super Heavy | 真空280t，全流量分级燃烧（FFSC），人类航天巅峰之作 |
| **BE-4** | Blue Origin | Vulcan/New Glenn | 真空245t（注*），富氧分级燃烧 |
| **天鹊-12（TQ-12）** | 蓝箭航天 | 朱雀二号/三号 | 真空82t，燃气发生器，国内甲烷先行者 |
| **天鹊-12A（TQ-12A）** | 蓝箭航天 | 朱雀三号 | 天鹊-12的增推版，推力提升中 |
| **CQ-50（Welkin/苍穹）** | 星河动力 | 智神星一号 | 海平面~101t / 真空~118t，燃气发生器 |

> `*` BE-4 的具体推力数据在公开资料中存在差异。Blue Origin 官方早期公布的 BE-4 海平面推力约为 2400 kN（约245t），但 ULA 在 Vulcan 火箭上的数据有所不同。

### 核心参数

| 参数 | 数值 |
|------|------|
| 密度 | ~425 kg/m³（沸点附近，-162°C） |
| 真空比冲 | 340-380 s |
| 推力范围 | 40t-280t（Raptor 3 达到 280t 级） |
| 沸点 | -162°C（比液氢的 -253°C 温和得多） |

### 优点 👍

1. **抗积碳**：甲烷燃烧后主要产生 CO₂ 和 H₂O，碳沉积微乎其微。这意味着——**发动机不需要频繁拆洗就能多次复用**。这对可回收火箭的运营成本是革命性的。
2. **比冲硬核**：真空比冲可达380s，介于煤油和氢之间。一级够用，上面级也能打得动。
3. **火星密码**：甲烷未来可以在火星上通过 Sabatier 反应（CO₂ + 4H₂ → CH₄ + 2H₂O）原位合成。虽然这种远期叙事对商业火箭不关键，但对 SpaceX 的"火星殖民"目标是战略层面的。
4. **冷却效果好**：甲烷的比热容高，在再生冷却中的吸热能力强，对发动机壁面保护更好。
5. **温度友好**：-162°C的液氧和-162°C的液甲烷可以共用一个储箱隔热方案（液氧是-183°C，相差不到20°C），简化了箭体设计。

### 缺点 👎

1. **密度低，储箱大**：甲烷密度只有煤油的约一半，意味着同样质量的燃料需要差不多2倍的储箱容积。火箭变长、结构增重、阻力变大。
2. **技术成熟度不足**：甲烷发动机大规模飞行的历史才几年。Raptor 虽然惊艳，但 Starship 还在试飞阶段；天鹊-12 飞过了但可靠性验证样本量小。
3. **比冲优势不够突出**：甲烷比冲虽然好于煤油，但差距并没有大到让所有煤油发动机立即淘汰的地步。

## 三、比冲天花板：液氧/液氢（LOX/LH₂）

氢是航天界的"贵族燃料"。它拥有最高比冲，但也最娇气。用一个不太恰当的比喻——煤油是干活的老黄牛，甲烷是刚毕业的985硕士，而液氢是养在深闺的千金大小姐。

### 代表发动机

| 发动机 | 所属 | 火箭 | 核心特点 |
|--------|------|------|---------|
| **RS-25（SSME）** | 美国（波音/洛克达因） | 航天飞机/SLS | 真空232t，可复用4次，史上最复杂的发动机之一 |
| **RL-10** | 美国（洛克达因） | 各类上面级（半人马座等） | 真空~11t，膨胀循环，服役近60年，人类液体火箭发动机的老寿星 |
| **LE-9** | 日本（三菱重工） | H3 | 真空150t，膨胀泄放循环，日本氢路线的代表作 |
| **LE-7A** | 日本（三菱重工） | H-IIA | 真空122t，分级燃烧 |
| **YF-77** | 中国航天六院 | 长征五号 | 真空70t，燃气发生器，中国最大氢氧发动机 |

你可能不知道的是——RL-10 从1962年首飞到现在，飞行次数超过500次，见证了从土星I到半人马座5的整个美国航天史，是最稳重的一款上面级发动机。

### 核心参数

| 参数 | 数值 |
|------|------|
| 密度 | ~70 kg/m³（-253°C，水的1/14！） |
| 真空比冲 | 430-465 s（碾压其他燃料） |
| 推力范围 | 1t-232t |
| 沸点 | -253°C（接近绝对零度了兄弟） |

### 优点 👍

1. **比冲最高**：真空比冲可达450s以上，比煤油高出近100s，比甲烷高出约70s。这个差距意味着——同样质量的燃料，氢能推上去更多东西。
2. **燃烧产物是水蒸气**：绝对零积碳，不像煤油那样燃烧室积碳堵塞。对环境也友好。
3. **冷氢可做冷却剂**：液氢在这个温度下是超级冷却剂，再生冷却效果极佳。这也是为什么RS-25能做到很高燃烧室压力的原因之一。

### 缺点 👎

1. **密度极低**：70kg/m³是什么概念？一个标准油桶装满才17公斤液氢。储箱体积巨大——航天飞机的外部橙色储箱就是装氢的，占了整枚火箭一半以上的体积。
2. **氢脆**：氢原子极小，会渗入金属晶格，与碳反应导致材料变脆。RS-25 每次飞行后都要更换大量管路和密封件，维修成本动辄数千万美元。你可能会问——为什么不换材料？因为到现在也没有完美的抗氢脆材料。
3. **极低温挑战**：-253°C的液氢意味着：
   - 要求世界上最顶级的隔热技术
   - 加注后无法长期保持（每天蒸发损失 1-3%）
   - 加注过程极其复杂，H3火箭多次因此推迟发射
4. **成本昂贵**：液氢制备成本远高于煤油和甲烷，液化过程消耗大量能源（约30%的能量用于冷却）。
5. **不适合一级复用**：低密度导致一级直径过大、结构质量占比过高，加上氢脆对寿命的限制，氢氧发动机做可回收一级极其困难。

## 四、横向对比：一句话看清楚

| 参数 | 液氧/煤油 | 液氧/甲烷 | 液氧/液氢 |
|------|:---------:|:---------:|:---------:|
| 密度（kg/m³） | ~820 | ~425 | ~70 |
| 真空比冲（s） | 320-348 | 340-380 | 430-465 |
| 沸点（°C） | ~200（沸点高，常温液体） | -162 | -253 |
| 燃烧温度（°C） | ~3500 | ~3200 | ~3000 |
| 积碳倾向 | 🔴 严重 | 🟢 极少 | 🟢 无 |
| 多次复用潜力 | 🟡 需处理积碳 | 🟢 本征优势 | 🔴 氢脆限制大 |
| 燃料成本 | 🟢 极低 | 🟡 中等 | 🔴 高 |
| 技术成熟度 | 🟢 最高 | 🟡 尚在发展 | 🟢 高但复杂 |
| 一级适用性 | 🟢 极好 | 🟢 很好 | 🟡 差 |

## 五、中国商业航天：为什么是煤油和甲烷当家？

如果你关注中国商业航天，会发现一个有意思的现象——国内民营火箭公司，几乎清一色选择煤油或甲烷路线。没有一家在做液氢一级。为什么？

**第一，氢的技术门槛太高了。** 液氢需要的极低温储箱、阀门、密封件，比煤油和甲烷复杂了不止一个数量级。国家队（航天六院）有YF-77，但站在国家肩膀上也不容易——YF-77的研制之路走得并不顺利。对缺资金、缺实验设施、缺年轻火箭设计师的民营公司来说，一上来就做氢氧，基本等于送人头。

**第二，市场和商业模式决定路线。** 中国商业航天的第一战场是中低轨卫星星座组网（逻辑类似Starlink）。这意味着需要什么？——**频繁发射、低成本、可回收复用**。煤油和甲烷在这三个维度上都有自己的杀手锏。液氢的复杂度和运营成本，与商业航天的底层逻辑有一定偏离。

**第三，液氢的竞争对手不是液氧煤油，而是物流效率。** 如果一颗卫星的成本是200万，射一次套餐800万起，谁会在乎上面级比冲多50s带来的微量运力提升？钱更多要考虑的是——能不能多飞一次、能不能快速发射、能不能降单价。

不过话说回来——**上面级仍然是液氢的主场**。RL-10 沿用60年不是白给的。如果将来中国商业航天做重型火箭的上面级，氢氧方案会重新成为高优先级选项。

## 六、国际对比：各有各的道理

### 🇺🇸 美国（SpaceX）→ 甲烷

SpaceX选甲烷，逻辑非常清晰：**火星殖民**。你要在火星上造燃料，只能造甲烷（C+H₂O→CH₄+O₂）。再加上Raptor的FFSC循环把甲烷的潜力压榨到了极致。而且Falcon 9已经是成熟产品，Merlin 1D还在用煤油，Raptor是面向下一代。

### 🇷🇺 俄罗斯 → 煤油

俄罗斯几乎把煤油发动机练到了登峰造极——RD-180、RD-171MV、NK-33，每一台都是煤油发动机里的暴力美学。为什么俄罗斯不碰甲烷和氢？
- 冷战时期积累的煤油发动机技术太深厚了，换个燃料等于放弃护城河
- 俄罗斯地理上极寒地区多，液氢的极低温在发射场操作更困难
- 没有强烈的"火星叙事"需求——安加拉系列用煤油就够了

### 🇯🇵 日本 → 氢

日本是唯一坚持一级也上氢的国家。H-IIA的LE-7A、H3的LE-9都是氢氧一级发动机。日本为什么这么选？
- 国土小、发射场有限，用高比冲氢氧可以用更轻的火箭做到更高的运力
- 日本在液化天然气（LNG）领域世界领先，液氢技术继承了很多LNG的积累
- 日本不做煤油路线，因为没有煤油发动机的历史积累，不如押注氢氧一条路走到黑

### 🇨🇳 中国 → 煤油为主、甲烷追赶、氢做上面级

中国的路线比较务实——国家队用YF-100（煤油分级燃烧）走遍天下，配合YF-77做上面级；民营企业在煤油（天兵、星河）和甲烷（蓝箭、星际）两条线同时下注，灵活靠岸。

## 结语

燃料选择没有标准答案，取决于火箭定位和商业模式。煤油是稳定的基本盘，甲烷是未来的主流方向，氢是比冲的巅峰但也是一级复用的代价。与其问"哪种更好"，不如问"哪条路更匹配你要打的那场仗"。
