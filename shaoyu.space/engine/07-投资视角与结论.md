# 投资视角：发动机路线决定胜负

> 前面五篇讲了燃料路线、循环方式、全球解剖、可回收影响和中国格局。
> 这一篇，我们把所有线索串起来，回答唯一的问题：**在投资天兵科技时，发动机是怎么判断的依据？**

---

## 一、发动机在商业航天投资中的权重

发动机占火箭总成本约**30-60%**，但占投资决策的关注度应该 **超过70%**。

为什么？因为：

1. **发动机决定了火箭能否复用**——而复用决定了发射成本能否降到1/10
2. **发动机开发周期最长**——4-6年起步，是火箭开发进度的"关键路径"
3. **发动机是"换不了"的**——火箭的燃料、循环方式定了就改不了，这个选择影响未来5-10年
4. **发动机团队决定了技术天花板**——流体力学+燃烧室设计+材料科学的复合能力

> **一句话结论**：看商业火箭公司，先看它用什么发动机，再说别的。

## 二、燃料路线的终局判断

### 中短期（3-5年）：煤油仍是主流

| 优势 | 说明 |
|:---|:------|
| ✅ 技术成熟 | RD-180/Merlin 1D/YF-100都验证了数十年 |
| ✅ 供应链完善 | 中国有YF-100/YF-102产线，自主可控 |
| ✅ 推重比够用 | 天火-12达111吨级，7台并联够天龙三号用 |
| ✅ 成本低 | RP-1比甲烷便宜，比氢便宜一个数量级 |

**天兵选煤油的判断**：务实、正确。在2020-2025年这个窗口期，拿成熟的煤油快速出产品，是商业航天唯一能活下来的路线。

### 长期（5-10年）：甲烷会逐步取代煤油

| 优势 | 说明 |
|:---|:------|
| 🔥 复用清洁 | 甲烷不积碳，理论上适合无限次复用 |
| 🚀 比冲更高 | 真空比冲~370s vs 煤油~350s |
| 🌎 火星叙事 | 甲烷可在火星原位制造，NASA已证实 |
| ✅ 已验证 | Raptor已飞、BE-4已飞、天鹊已飞 |

**关键信号**：如果SpaceX的Starship在2026-2027年进入商业运营，甲烷路线的"确定性"会大幅提高，全球商业航天可能加速向甲烷切换。

### 氢的定位：上面级专用

液氢的密度太低、储箱太大，不适合一级复用。但**上面级用氢是永远的正确答案**——RL-10（1960年代设计的发动机）至今仍在用，说明氢在上面级的地位不可撼动。

### 路线判断结论

```
天兵（煤油）：3-5年内确定性高，5年后需关注是否储备甲烷能力
蓝箭（甲烷）：布局3-5年后的趋势，短期技术风险更高
国家队（氢）：上面级王者，但一级路线不考虑
```

## 三、循环方式的终局判断

| 循环 | 定位 | 投资判断 |
|:----|:----|:--------|
| **燃气发生器** | 现在 | 商业航天当前最优解。Merlin 1D证明了"不完美但够用"的威力 |
| **分级燃烧** | 未来3-5年 | 国产YF-100已验证，民营做分级燃烧是下一个门槛 |
| **全流量分级（FFSC）** | 未来5-10年 | Raptor已验证但极复杂，不适合民营初创 |
| **膨胀循环** | 上面级 | 上面级的正确选择，投资人关注谁有好的上面级发动机 |

**关键结论**：天兵的天火-12用燃气发生器+针栓喷注器，当前阶段这个选择是正确的。投资人应该关注的是——天兵有没有在研发下一代的**分级燃烧版本发动机**。如果有，说明团队有长期视野；如果没有，要小心3-5年后技术上被蓝箭反超。

## 四、天兵 vs 蓝箭：发动机对比

### 全面横向对比

| 维度 | 天兵科技（天火-12） | 蓝箭航天（天鹊-12A） | 差距分析 |
|:----|:-----------------:|:-----------------:|:--------|
| **推力** | 111/138吨 | 67/82吨 | 天兵赢了50%+ |
| **燃料路线** | 煤油（成熟） | 甲烷（前沿） | 短期天兵✅，长期蓝箭✅ |
| **节流范围** | 40%~110% | 未公开（预计60-100%） | 天兵占优 |
| **已飞次数** | 0次（热试车多次） | 2次（朱雀二号已飞） | 蓝箭实际飞行经验更好 |
| **3D打印** | ✅ 已应用 | ✅ 已应用 | 持平 |
| **可回收设计** | ✅ 针栓+大节流 | ✅ 甲烷+设计复用 | 持平，路径不同 |
| **供应链** | 航天六院（YF-102基础） | 自研 | 天兵供应链更成熟 |
| **累计投入** | ~30-50亿元 | ~40亿元 | 蓝箭略多 |
| **成本** | 煤油更便宜 | 甲烷燃料贵一点 | 天兵占优 |

**综合判断**：天火-12目前参数更好看，但天鹊-12有实际飞行经验。如果说天火-12赢了"参数阶段"，那天鹊-12赢了"实战阶段"。两家的下一轮迭代（天火-12实际飞行 vs 天鹊-12A升级版）将决定谁是发动机赛道的老大。

## 六、投资建议：发动机视角

### 结论一：天兵当前的发动机选择是最优的短期方案

煤油+燃气发生器+针栓+3D打印这个组合，在2020-2026年的时间窗口里，是把"商业航天能活下来"和"技术上够做可回收"这两个目标平衡得最好的方案。

### 结论二：但天兵必须在3-5年内布局甲烷

如果天兵在2028年前后还不出甲烷发动机的规划，就是重大战略失误——这不是烧煤油好不好的问题，而是市场会**用脚投票**：投资人会担心天兵5年后技术落后。

### 结论三：关键观测信号（milestones）

| 信号 | 时间窗口 | 判断 |
|:----|:-------:|:----|
| 天火-12实际飞行 | 2026-2027 | 最核心信号，飞了再说 |
| 天火-12实现回收 | 2027-2029 | 火箭能否真正复用的事实验证 |
| 天兵是否公布甲烷发动机 | 2027-2028 | 是否有长期视野 |
| 天龙三号并联7台天火-12 | 2026-2027 | 验证天火-12的并联可靠性 |
| 天鹊-12A升级版（推力提升+节流） | 2026-2027 | 蓝箭的反击 |

### 最终判断

> **在发动机这个维度，天兵当前是投资组合里"高确定性"的那一块，蓝箭是"高赔率"的那一块。**
>
> 如果你更看重**3年内的确定性**（天龙三号成功、订单兑现、团队执行），天火-12的参数已经够看了。
>
> 如果你更看重**5-10年的终局**（全球商业航天的最终赢家），别忽视蓝箭的甲烷路线——最终是对的。
>
> **最佳策略：两家都看，天兵重仓近端确定性，蓝箭作为"看涨期权"轻仓持有。**
