## 1. 产品定位

这是一个**个人名片式作品展示站**。核心使命：用最少的点击告诉访客"你是谁、你做了什么"，然后引导他们进入具体作品。不花哨、不堆砌，每个像素都有它的意义。

**目标访客画像：**
- 技术同行 / 投资人 / 猎头 —— 快速判断作品质量
- 随机访客 —— 被暗黑风格勾住，顺手点进去看看

## 2. 页面结构

```
┌─ Header ──────────────────────────┐
│  标志(logo)       [空/预留]        │
├─ Hero / 简介区 ────────────────────┤
│  一句话+副线（态度到位）            │
├─ 作品卡片区 ───────────────────────┤
│  ┌─ 卡片1 ──┐  ┌─ 卡片2 ──┐       │
│  │ 小说     │  │ 投资报告 │        │
│  └──────────┘  └──────────┘       │
├─ Footer ──────────────────────────┤
│  © + 链接                          │
└────────────────────────────────────┘
```

### 2.1 Header

- **左侧**：一个极简logo/标识，可以是名字缩写的文字标
- **右侧**：留白，或者一个小状态指示器（"在线"之类的彩蛋）
- 固定不固定？**不固定**。页面就一屏多，没必要fixed header占空间

### 2.2 简介区（Hero）

- **主标题（一句话）**：_「写故事的人，也是造火箭的人。」_
  干净利落，反差感拉满 —— 科幻小说 × 航天投资，一个人的两个极端。

- **副线**：_「关于加班管理局的暗黑世界，和走进天兵科技的硬核分析。」_
  点名两个作品，但不啰嗦。

- **动效**：主标题打字机效果或逐字淡入，给第一印象加点仪式感

## 3. 视觉风格

### 配色

| 用途 | 色值 |
|------|------|
| 背景 | `#0a0a0f`（深空黑，带一丝蓝底） |
| 卡片背景 | `#14141e`（比背景亮一档） |
| 主文字 | `#e8e8f0`（暖白偏冷） |
| 强调色 | `#00d4aa`（青绿色，作为光效和交互反馈） |
| 次要强调 | `#7c3aed`（紫，用于标签和装饰线） |

### 字体

- 中文：**Noto Sans SC** 或 **Inter**（无衬线，科技感干净）
- 英文/数字：**JetBrains Mono**（等宽，代码感）
- Hero主标题：字重 600，字号 2rem+
- 正文：字重 400，字号 1rem

### 质感细节

- **微妙的渐变**：背景右上角到左下角的深蓝→黑渐变，不抢眼但避免死黑
- **卡片边框发光**：1px `#00d4aa` 低透明度边框，hover时亮度提升并增大发光半径
- **磨砂玻璃效果**（可选）：卡片用 `backdrop-filter: blur(4px)`，增强层次感
- **粒子 / 网格背景**：非常 subtle 的科技网格线（不干扰阅读，只增加纹理）

## 4. 交互细节

| 交互 | 行为 |
|------|------|
| **卡片区整体 hover** | 轻微上浮（`translateY(-4px)`）+ 阴影加深，持续 0.3s ease |
| **卡片按钮 hover** | 按钮填充色从透明变为 `#00d4aa`，文字变为 `#0a0a0f`（反色），右侧箭头略微右移 |
| **点击卡片** | 点击卡片任一处（除按钮外）也跳转到对应链接。整卡可点击，降低操作成本 |
| **Hero 动效** | 页面加载后主标题逐词淡入（每个词 0.2s 间隔），保留一次，不循环 |
| **滚动** | 平滑滚动（`scroll-behavior: smooth`），但不需要复杂的视差 |
