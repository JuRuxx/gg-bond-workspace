一

凌晨两点五十五分，陆小闲蹲在一栋写字楼的安全通道里，思考人生。

不是那种"我从哪里来、要到哪里去"的人生——是更急迫的那种："我为什么要答应做这个工作？"

十二个小时前，他还是一个面了十七次试、身上只有二十七块五毛钱的失业青年。现在他蹲在冰凉的水泥台阶上，手机屏幕发出微弱的光，照亮一张A4纸——上面是苏棠给他写的"案件简报"：

> \1：LV1-20260520-003 > \1：南山区科苑路XX号·6楼·某在线教育公司 > \1：夜班保安老赵 > \1：凌晨十二点后，6楼会传出键盘敲击声。声音密集、急促、没有节奏。电梯经过该楼层时会自动停下开门，但门外什么也没有。老赵昨天巡楼时路过茶水间，看到微波炉的门一开一合，像有只看不见的手在热东西。 > \1：该公司近期无员工加班至凌晨的记录。三个月前，有一名员工——王磊，34岁，后端工程师——在该楼层加班至凌晨四点后猝死。

陆小闲看完最后一行字，抬头看了看面前的安全门。门缝里透出走廊的日光灯——惨白、嗡嗡响、典型的深夜写字楼氛围。

"……第一个案件就是有人猝死过的地方？这是实习难度？"

他拿出手机想给苏棠发消息，想起来苏棠说"你自己处理，我就在楼下咖啡厅，有事打我电话"——但同时也说了"除非你打不过，否则别打扰我喝热美式"。

"热美式比我命重要是吧。"

他把A4纸折好塞进口袋，深吸一口气，推开了安全门。

走廊很安静。日光灯一排亮着，照得地面白惨惨的。左右两边都是玻璃门办公区——左边某教育公司的logo亮着，右边是另一家做电商的公司，黑着灯。

他朝左边走去。

越靠近那家公司的玻璃门，温度就越低——不是空调的那种凉，是一种往骨头缝里钻的冷。陆小闲搓了搓手臂，发现胳膊上的汗毛全竖起来了。

"……好吧，有点吓人了。"

他站在玻璃门前，往里看。办公室里黑漆漆的，只有几台显示器的电源灯一明一灭。工位空无一人，椅子整整齐齐地推在桌下——一看就是那种下班前被行政提醒过"注意办公桌整洁"的地方。

然后他听到了。

从办公室深处传来的——一个极快的、机械的键盘敲击声。嗒嗒嗒嗒嗒嗒嗒嗒。没有停顿，没有节奏的变化，像有人在用打字机输入一本永远打不完的书。

陆小闲的心跳突然加速了。

他的手按在玻璃门上——

然后他想起来一个很重要的问题。

摸鱼空间，到底怎么用？

苏棠只说"想象你摸鱼的时候那种感觉，然后把那种感觉往前推"。但谁能在凌晨三点、面对一扇可能有鬼的玻璃门的时候，进入摸鱼状态啊？

"……也行，硬着头皮上。"

他推开了门。

二

门缝里灌出一股冷气，像是有人把一整台空调的冷风都聚焦在了这个门口。

键盘声在他推门的瞬间停了。

整个办公室彻底安静下来。安静到他能听到头顶日光灯的电流声——嗡——和自己的心跳声——砰、砰、砰。

"你好？"他试探性地说了一声。

没有回应。

他拿出手机打开手电筒，照向办公室深处。手电的光圈在一个个工位上扫过——显示器、键盘、文件夹、马克杯、一盆已经枯死的绿萝……

光圈停在角落的一个工位上。

那个工位的显示器是亮着的。

屏幕上是满屏的代码，光标在最后一行的末尾一跳一跳。像是有人刚刚才打完，然后起身离开了。

陆小闲咽了一口唾沫，朝那个工位走去。

三米、两米、一米——

他站到了那个工位面前。显示器上是一个没写完的函数：

``` function fixTheBug() { // TODO: 我真的好累 return "I want to go home"; } ```

陆小闲盯着那行注释看了两秒。

然后他身后的黑暗里，传来一声极轻的——

叹息。

不是毛骨悚然的那种，不是鬼片里拖长音的呻吟。就是一个很普通的、带着疲惫的叹息——像你加班到凌晨、终于改完最后一个bug、靠在椅背上长出一口气。那种声音。

陆小闲僵硬地转过身。

办公室里什么都没有。

但电脑屏幕上的代码变了。

新的一行出现在末尾：

``` // 你能看到我吗？ ```

三

陆小闲盯着那行字看了五秒钟，后背一阵凉意。

"……能看到。"

他的声音在空旷的办公室里显得又小又虚。

光标又跳了一下。新的一行缓缓浮现：

``` // 我已经很久没跟人说过话了 // 他们都看不到我 // 只有你 // 你是新来的吧 ```

"你是……王磊？"

光标停了两秒。然后迅速敲出：

``` // 你还知道我名字啊 // 看来他们没把我忘干净 ```

陆小闲深吸一口气。他想起了苏棠说过的话——大部分Lv1案件的本质是"一个累到极致的人的投射"。他决定把苏棠那套"先别管鬼不鬼，先看看这个人怎么了"的思路用起来。

"你不是故意吓人的，对吧？"

光标又停了。这次停顿比较久。

``` // 我没有想吓人 // 我只是……还在写代码 // 那天晚上有一个bug没改完 // 改到凌晨四点也没改出来 // 我说休息一下 // 然后就没有然后了 ```

陆小闲喉咙发紧。

"那个bug……现在还在跑？"

``` // 在 // 一直在跑 // 每天晚上十二点之后它就会自己跑起来 // 我也跟着起来继续写 // 因为我没写完 // 不写完总觉得差一点什么 ```

陆小闲看着那行字，突然觉得这比他看过的任何恐怖片都让人难受。

一个死了三个月的人。每天晚上还要起来加班。就因为有一个bug没改完。

他张了张嘴想说点什么——但他被一个想法击中了。

"你在工作能量上'卡住'了。你的WL场在你离开之后还在运转——因为你的执念太强了，强到身体不在了，能量还在原地加班。"

这是苏棠昨天说的原话，现在突然在他脑子里响起来。

他低头看了看自己的手。

然后他问自己：摸鱼空间能做什么？苏棠说它能收集"被浪费的时间"。那这里呢——一个卡在同一个bug里三个月的人，算不算"时间被浪费了"？

只能试试了。

他闭上眼睛，努力回想苏棠说的"摸鱼的感觉"。

什么是摸鱼的感觉？

是早上闹钟响了按掉再睡十分钟。是开会的时候假装记笔记实际在画小人。是用"我去倒杯水"从工位上逃出来在茶水间刷五分钟手机。是蹲在写字楼门口啃煎饼果子的那十七次间隙——

就是那些不属于任何人的、偷来的、属于自己的时间。

陆小闲睁开了眼睛。

他看到了一样东西。

在他的手掌和那个显示器之间，漂浮着一些微小的、灰白色的光点——它们从显示器上方飘出来，像烟雾一样，缓慢地扩散到空气中。那不是他昨天看到的暗红色能量线，而是另一种东西：细碎、苍白、带着一种精疲力竭的灰。

他想也没想，伸出手，做了一个"抓"的动作。

那些灰白色光点像被吸引了一样——朝他的掌心汇聚。

一阵冰凉的触感从掌心蔓延到手腕，再沿着手臂往上爬。凉，但不是刺骨的凉，更像把手伸进溪水里——透着一种"终于停下来了"的寒意。

电脑屏幕上的代码突然开始滚动。

不是正常的滚动——是倒着滚。那些该死的、没写完的函数一行一行地消失，像是有人在一帧一帧地回放录像。文件关闭、编辑器退出、系统关机——屏幕暗了。

然后，黑暗中，响起了一个男人的声音。

不是通过扬声器——是空气里，真真切切的、带着疲惫的中年嗓音：

"……谢谢。"

陆小闲愣住了。

"终于可以……休息了。"

声音消失在空气中。像一阵风从敞开的窗户吹进来，然后窗户关上了。

他站在原地，掌心还残留着那股凉意，嘴张着不知道该说什么。

头顶的日光灯恢复了正常的亮度。办公室里的温度回升了——那种往骨头缝里钻的冷消失了。

他低头看了看自己的手。掌心什么也没有，但那种"凉一下又暖回来"的感觉还在。

所以这个就是……摸鱼空间？

四

陆小闲走出写字楼大门的时候，凌晨三点的街上一个人也没有。

冷风一吹，他才发现自己后背全湿了。

他坐在门口的台阶上，掏出手机想给苏棠发消息——发现苏棠已经发了一条过来：

> 苏棠：处理完了？ > 苏棠：我在咖啡厅看到6楼的灯闪了一下，然后又正常了。 > 苏棠：你活着吧？

> 陆小闲：活着。 > 陆小闲：但累死了。 > 陆小闲：我用了摸鱼空间。

> 苏棠：效果怎么样？

> 陆小闲：好像……把那一段"卡住的时间"释放了。 > 陆小闲：那个叫王磊的程序员，他不用再加班了。

苏棠那边沉默了一分钟。然后回：

> 苏棠：第一次用就成功了，比我想象中快。 > 苏棠：放在桌上的炒面是给你的，凉了自己热。 > 苏棠：还有，别忘了还时间。

陆小闲看着最后一行字，心里一沉。

还时间。

摸鱼空间的代价。

他使用了多长时间的摸鱼值？刚才那些灰白色的光点——如果那是王磊卡在"凌晨四点改bug"的那段时间——三个月，将近一百个夜晚，断断续续的能量残留。他不确定自己能换算成具体多少小时，只觉得自己像是用手掌接住了一场下了很久很久的雨。

他低头看了看掌心。

"……应该不会太多吧？"

他决定先不想了。台阶旁边的花坛上放着一盒炒面，还微微冒着热气。

他打开盖子。

是鸡蛋炒面。加了一根肠。

他低头吃起来。凌晨三点的风冷飕飕的，但炒面是热的。

吃着吃着他突然想：那个代码鬼被我超度了，我下个月房租还没着落，屁股底下这座写字楼我三个月前还来面试过一家做跨境电商的，挂了。人生真的很神奇——三个月前还在被人面试，三个月后在同一个区帮鬼改bug。

他嚼着炒面，觉得这工作虽然又穷又危险又欠债，但至少说出去挺唬人的。

虽然他也说不出去。

五

回到星辉大厦的员工宿舍已经快凌晨四点了。

员工宿舍就在同栋楼的十五层——改造成了两室一厅的上下铺，条件简陋，但胜在免费。陆小闲洗了个澡躺在床上，盯着天花板发呆。

脑子里还在转今天晚上的事。

王磊。那行"我真的好累"的注释。那个疲惫的叹息。"终于可以休息了。"

他拿起手机，犹豫了一下，搜索了"王磊 后端工程师 猝死"。

新闻链接跳出来——是一个本地资讯号发的消息，阅读量不到两千。写得很克制："某某科技公司一名34岁员工在加班后突发不适，送医院抢救无效死亡。"评论区只有十几条，一半在说"加班害人"，一半在问"赔了多少钱"。

他盯着那行新闻看了很久，忍不住又搜了一下王磊的名字。没有连篇累牍的报道，没有纪念页面，只有零星几条转发。倒是在某个程序员论坛里翻到一条旧帖——发帖时间是三个月前的一个凌晨：

> 标题：今晚又要通宵了 > 内容：项目deadline下周二，后端接口还有三个没写完。组长说这周辛苦一下，下周给大家调休。我已经连续加班十天了，今天又从早上十点干到凌晨三点。刚才站起来倒水的时候腿是麻的，差点摔一跤。 > 底下有一条回复：注意身体啊兄弟。 > 楼主回：没事，改完这个bug就睡。

帖子底下再没有其他回复。

发帖人的ID是一串乱码，但IP归属地——跟王磊公司所在的写字楼一致。

陆小闲盯着那行"改完这个bug就睡"看了很久。那句写在网页上的话，跟刚才电脑上那行"我真的好累"的注释，像是同一个人的两行遗言——一句话留在人间，一句话留在了死后。

他关掉手机，翻了个身。

这就是苏棠说的"大部分Lv1案件"吗？不是什么超自然恐怖——是一个累到极致的人在死后的凌晨还要继续写代码。

他盯着墙上的裂缝发呆。

然后手机震了一下。

一条短信。

没有归属地，号码是一串他认不出的组合——像是从境外发来的虚拟号。

他打开短信。

只有一行字：

> 不要相信管理局。

陆小闲的心跳漏了一拍。

他立刻回复：你是谁？

发送失败——号码不存在。

他翻来覆去看了好几遍那六个字，又看了看自己刚才使用摸鱼空间的那只手——掌心的凉意早已消散，但那句"你用了多少就要还多少"的话还在脑子里打转。

墙上的裂缝在月光下像一条细小的暗红色线条。他眨了眨眼，那条线又不见了。

窗外，城市的夜景被薄薄的云层覆盖着。远处的高楼还亮着零零星星的灯光——大概又是谁在加班。

陆小闲把手机放在枕头底下，闭上眼睛。

但睡不着。

不要相信管理局？

那他能相信谁？
